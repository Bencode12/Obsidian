# 5. Elastic Force (Hooke's Law)

## 🔬 Elastic Force

### What is Elastic Force?

**Elastic Force** is the restoring force exerted by a stretched or compressed spring/elastic material.

### Examples of Elastic Objects

- Springs
- Rubber bands
- Bungee cords
- Trampoline springs
- Bow and arrow
- Suspension systems

---

## 📐 Hooke's Law

### Statement

**The extension of a spring is directly proportional to the applied force, provided the elastic limit is not exceeded.**

### Formula

```
F = kx
```

Where:
- **F** = Applied force (N)
- **k** = Spring constant (N/m)
- **x** = Extension (m) = new length - original length

### Spring Constant (k)

- Measures **stiffness** of spring
- **High k** = Stiff spring (hard to stretch)
- **Low k** = Soft spring (easy to stretch)
- Units: N/m (Newtons per meter)

---

## 📊 Force-Extension Graph

```
Force (N)
    ↑
    │     /
    │    /  (Linear region - obeys Hooke's Law)
    │   /
    │  /
    │ /
    │/___________→ Extension (m)
    0
```

**Key Points:**
- **Slope** = Spring constant (k)
- **Straight line** through origin = obeys Hooke's Law
- **Curved section** = elastic limit exceeded

### Elastic Limit

- Maximum extension before permanent deformation
- Beyond this point, spring doesn't return to original length
- Graph curves (no longer linear)

## 🧮 Worked Examples

### Example 1: Finding Force

**Problem:** A spring with k = 200 N/m is stretched by 0.1 m. Calculate the force.

**Solution:**
```
F = kx
F = 200 × 0.1
F = 20 N
```

### Example 2: Finding Extension

**Problem:** A 50 N force is applied to a spring (k = 250 N/m). Find extension.

**Solution:**
```
F = kx
50 = 250 × x
x = 50/250 = 0.2 m
```

### Example 3: Finding Spring Constant

**Problem:** A spring extends 5 cm when a 10 N force is applied. Find k.

**Solution:**
```
x = 5 cm = 0.05 m
F = kx
10 = k × 0.05
k = 10/0.05 = 200 N/m
```

### Example 4: Comparing Springs

**Problem:** Spring A (k = 100 N/m) and Spring B (k = 300 N/m) both have 30 N applied. Which extends more?

**Solution:**
```
Spring A: x = F/k = 30/100 = 0.3 m
Spring B: x = F/k = 30/300 = 0.1 m

Spring A extends more (softer spring)
```

## 🔋 Elastic Potential Energy

### Energy Stored in Springs

When you stretch/compress a spring, you store **elastic potential energy (EPE)**.

### Formula

```
EPE = ½kx²
```

Where:
- **EPE** = Elastic potential energy (Joules, J)
- **k** = Spring constant (N/m)
- **x** = Extension (m)

### Example

**Problem:** A spring (k = 400 N/m) is stretched by 0.2 m. Calculate stored energy.

**Solution:**
```
EPE = ½kx²
EPE = ½ × 400 × (0.2)²
EPE = 200 × 0.04
EPE = 8 J
```

## 🌍 Applications

### 1. Vehicle Suspension

- Springs absorb bumps
- Compress when hitting bump
- Release energy smoothly
- Provides comfortable ride

### 2. Weighing Scales (Spring Balance)

- Object stretches spring
- Extension proportional to weight
- Calibrated scale shows mass/weight

### 3. Trampoline

- Springs stretch when you land
- Store elastic potential energy
- Release energy to bounce you up

### 4. Archery Bow

- Bow acts like spring
- Archer does work pulling string
- Energy stored in bent bow
- Released as kinetic energy of arrow

### 5. Click Pen

- Spring compressed when clicking
- Stores energy
- Releases to extend pen tip

## ⚖️ Springs in Series and Parallel

### Springs in Series

```
|--Spring1--Spring2--|
```

- Same force in both springs
- Total extension = x₁ + x₂
- **Effective k is smaller** (softer)

```
1/k_total = 1/k₁ + 1/k₂
```

### Springs in Parallel

```
|--Spring1--|
|           |
|--Spring2--|
```

- Force shared between springs
- Same extension in both
- **Effective k is larger** (stiffer)

```
k_total = k₁ + k₂
```

## ✏️ Practice Problems

### Problem 1
A spring extends 0.08 m when a 16 N force is applied. Find k.

**Answer:** 200 N/m

### Problem 2
Calculate the force needed to stretch a spring (k = 150 N/m) by 0.12 m.

**Answer:** 18 N

### Problem 3
A spring stores 12 J of energy when compressed by 0.3 m. Find k.

**Solution:**
```
EPE = ½kx²
12 = ½ × k × (0.3)²
12 = 0.5 × k × 0.09
k = 12 / 0.045 = 267 N/m
```

### Problem 4
**Explain** why a trampoline with stiffer springs (higher k) gives a higher bounce.

**Answer:** Stiffer springs store more elastic potential energy for the same extension, which is converted to more kinetic energy.

## 📝 Summary

- Hooke's Law: F = kx
- k = spring constant (stiffness)
- Extension ∝ Force (linear relationship)
- Elastic limit: maximum before permanent stretch
- Elastic potential energy: EPE = ½kx²
- Applications: suspension, scales, trampolines, bows
