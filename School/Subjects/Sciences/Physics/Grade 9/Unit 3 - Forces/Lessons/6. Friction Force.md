# 6. Friction Force

## 📋 Learning Objectives

- [ ] Define friction and explain its causes
- [ ] Distinguish between static and kinetic friction
- [ ] Calculate friction force using f = μN
- [ ] Understand when friction is helpful or harmful

---

## 🔬 What is Friction?

### Definition

**Friction** is a force that opposes motion between two surfaces in contact.

### Direction

- Always **opposite** to direction of motion (or intended motion)
- Acts **parallel** to the surfaces

```
Motion → → → 
Friction ← ← ←
```

### Causes of Friction

1. **Surface roughness** - microscopic bumps and valleys
2. **Molecular attraction** - adhesive forces between surfaces
3. **Deformation** - surfaces press into each other

---

## 📊 Types of Friction

### 1. Static Friction

**Definition:** Friction between stationary surfaces

- Prevents motion from starting
- **Variable force** - matches applied force up to maximum
- **Maximum static friction:** f_s(max) = μ_s N

**Example:** Pushing heavy box that won't move

```
Applied force: 10 N → Friction: 10 N ← (no motion)
Applied force: 20 N → Friction: 20 N ← (no motion)
Applied force: 30 N → Friction: 25 N ← (box starts moving!)
```

### 2. Kinetic (Dynamic) Friction

**Definition:** Friction between moving surfaces

- Opposes ongoing motion
- **Constant** (doesn't depend on speed)
- Usually **less than** maximum static friction
- **Kinetic friction:** f_k = μ_k N

**Example:** Box sliding across floor

---

## 🧮 Calculating Friction

### Friction Formula

```
f = μN
```

Where:
- **f** = Friction force (N)
- **μ** = Coefficient of friction (no units)
- **N** = Normal force (N) - force pressing surfaces together

### Coefficient of Friction (μ)

- Depends on **materials** in contact
- **Higher μ** = More friction (rough surfaces)
- **Lower μ** = Less friction (smooth surfaces)
- Typically between 0 and 1

### Typical Values

| Surfaces | μ_s (static) | μ_k (kinetic) |
|----------|--------------|---------------|
| Rubber on dry concrete | 1.0 | 0.8 |
| Rubber on wet concrete | 0.7 | 0.5 |
| Wood on wood | 0.5 | 0.3 |
| Steel on steel | 0.7 | 0.5 |
| Ice on ice | 0.1 | 0.03 |
| Teflon on Teflon | 0.04 | 0.04 |

**Note:** μ_k is usually less than μ_s

## 🧮 Worked Examples

### Example 1: Basic Friction

**Problem:** A 10 kg box slides across a horizontal floor (μ_k = 0.3). Calculate friction force.

**Solution:**
```
N = mg = 10 × 9.8 = 98 N
f_k = μ_k N = 0.3 × 98 = 29.4 N
```

### Example 2: Pushing at Constant Speed

**Problem:** What force is needed to push a 20 kg box across a floor (μ_k = 0.4) at constant speed?

**Solution:**
```
For constant speed: F_applied = f_k
N = mg = 20 × 9.8 = 196 N
f_k = μ_k N = 0.4 × 196 = 78.4 N

Therefore: F_applied = 78.4 N
```

### Example 3: Static Friction

**Problem:** A 50 kg box sits on a floor (μ_s = 0.5). What minimum force starts it moving?

**Solution:**
```
f_s(max) = μ_s N
N = mg = 50 × 9.8 = 490 N
f_s(max) = 0.5 × 490 = 245 N

Minimum force needed: 245 N
```

## 🌍 Friction: Helpful vs Harmful

### When Friction is Helpful

| Situation | Why Helpful |
|-----------|-------------|
| Walking | Prevents slipping |
| Car tires | Provides traction |
| Brakes | Slows/stops vehicle |
| Writing | Pencil grips paper |
| Holding objects | Prevents dropping |
| Climbing | Grip on surfaces |

### When Friction is Harmful

| Situation | Why Harmful |
|-----------|-------------|
| Engine parts | Causes wear, heat |
| Door hinges | Makes squeaking, hard to open |
| Bicycle chain | Reduces efficiency |
| Air resistance | Slows vehicles |
| Machine bearings | Wastes energy |

---

## ⚙️ Reducing Friction

### Methods

1. **Lubrication** (oil, grease)
   - Fills gaps between surfaces
   - Creates smooth layer

2. **Ball bearings**
   - Rolling friction < sliding friction
   - Used in wheels, fans

3. **Smoothing surfaces**
   - Polishing
   - Sanding

4. **Streamlining**
   - Reduces air resistance
   - Aerodynamic shapes

5. **Using wheels**
   - Rolling friction much less than sliding

## ✏️ Practice Problems

### Problem 1
A 15 kg box slides on a floor with μ_k = 0.25. Calculate friction force.

**Answer:** 36.75 N

### Problem 2
What force is needed to push a 30 kg box at constant speed if μ_k = 0.35?

**Answer:** 102.9 N

### Problem 3
**Explain** why it's harder to start pushing a heavy box than to keep it moving.

**Answer:** Static friction (prevents starting) is greater than kinetic friction (opposes motion).

### Problem 4
**List** three ways to reduce friction in a bicycle.

**Answers:** Lubricate chain, use ball bearings in wheels, inflate tires properly.

---

## 📝 Summary

- Friction opposes motion
- Static friction > Kinetic friction
- Formula: f = μN
- μ depends on materials
- Helpful: walking, driving, gripping
- Harmful: wear, heat, energy loss
- Reduce with: lubrication, bearings, smoothing
