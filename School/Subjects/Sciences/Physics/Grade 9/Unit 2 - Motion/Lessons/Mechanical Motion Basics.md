# Mechanical Motion - Introduction

## 📋 Learning Objectives

- [ ] Define motion and reference frames
- [ ] Distinguish between distance and displacement
- [ ] Understand speed vs velocity
- [ ] Interpret motion graphs

---

## 🔬 What is Motion?

**Motion** is the change in position of an object over time.

### Reference Frame

- A **reference point** is needed to describe motion
- Motion is **relative** - depends on observer's viewpoint

**Example:** A person on a moving train:
- Relative to train: Not moving
- Relative to ground: Moving

---

## 📊 Distance vs Displacement

| Distance | Displacement |
|----------|--------------|
| Total path length | Straight line from start to end |
| Scalar (magnitude only) | Vector (magnitude + direction) |
| Always positive | Can be positive, negative, or zero |
| **d** or **s** | **Δx** or **s** |

### Example

Walking 3m East, then 4m West:
- **Distance** = 3m + 4m = 7m
- **Displacement** = 1m West

---

## ⚡ Speed vs Velocity

### Speed
**Rate of change of distance**

```
Speed = Distance / Time
v = d/t
```

- Scalar quantity
- Units: m/s, km/h

### Velocity
**Rate of change of displacement**

```
Velocity = Displacement / Time
v = Δx/t
```

- Vector quantity (has direction)
- Units: m/s (with direction)

---

## 📈 Motion Graphs

### Distance-Time Graph

```
Distance
    ↑
    │    /
    │   /  (constant speed)
    │  /
    │ /
    │/
    └────────→ Time
```

- **Slope** = Speed
- **Steeper slope** = Faster speed
- **Horizontal line** = Stationary (not moving)

### Displacement-Time Graph

- **Slope** = Velocity
- **Positive slope** = Moving forward
- **Negative slope** = Moving backward
- **Horizontal** = Stationary

---

## 🧮 Formulas

### Average Speed
```
Average Speed = Total Distance / Total Time
```

### Average Velocity
```
Average Velocity = Total Displacement / Total Time
```

### Converting Units
```
1 m/s = 3.6 km/h
1 km/h = 1/3.6 m/s
```

---

## ✏️ Practice Problems

### Problem 1
A car travels 150 km in 2 hours. Calculate average speed.

**Solution:**
```
v = d/t = 150 km / 2 h = 75 km/h
```

### Problem 2
A runner completes one lap of a 400m track in 50 seconds.
a) Distance = ?
b) Displacement = ?
c) Average speed = ?
d) Average velocity = ?

**Answers:**
a) 400m
b) 0m (back to start)
c) 8 m/s
d) 0 m/s

---

## 📝 Summary

- Motion requires a reference frame
- Distance (scalar) ≠ Displacement (vector)
- Speed (scalar) ≠ Velocity (vector)
- Slope of distance-time graph = speed
- Slope of displacement-time graph = velocity
