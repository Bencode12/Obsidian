## Motivation & Historical Context

### Why Measure Theory Matters

Measure theory provides the foundation for modern probability, harmonic analysis, and quantum mechanics. It solves fundamental limitations of Riemann integration.

**Key Problems It Solves:**
- **Pathological functions**: Dirichlet function (1 on rationals, 0 on irrationals) is Lebesgue integrable but not Riemann
- **Limit interchanges**: Powerful convergence theorems with minimal hypotheses
- **Probability foundations**: Probability is just measure with total mass 1

**Historical Crisis**: Around 1900, mathematicians realized Riemann integration was inadequate for Fourier analysis and probability.

## Core Concepts

### Concept 1: Sigma-Algebras

Collections of "measurable" sets closed under countable operations.

**Teaching analogy**: Not all subsets are measurable (Vitali set)—sigma-algebra specifies which ones we can measure.

### Concept 2: Lebesgue Measure

Extends length from intervals to complicated sets via outer measure.

**Key insight**: Measure is defined "from the outside in"—approximate arbitrary sets by unions of intervals.

### Concept 3: Lebesgue Integration

Integrate by slicing horizontally (by values) rather than vertically (by domain).

**Visual contrast**: 
- Riemann: Partition x-axis, sum f(x_i)·Δx
- Lebesgue: Partition y-axis, sum y·measure({x : f(x) ≈ y})

## Key Definitions

**Measurable Set**: Set E where outer measure satisfies Carathéodory criterion.

**Measurable Function**: Preimage of every interval is measurable.

**Lebesgue Integral**: Defined via simple function approximation.

**L^p Space**: Functions with finite p-norm.

## Main Theorems

### Monotone Convergence Theorem

If $f_n \uparrow f$ (increasing sequence), then $\int f_n \uparrow \int f$.

**Why powerful**: No domination needed, just monotonicity.

### Dominated Convergence Theorem

If $f_n \to f$ and $|f_n| \leq g$ integrable, then $\int f_n \to \int f$.

**Most used theorem**: Justifies swapping limits and integrals in practice.

### Fatou's Lemma

$\int \liminf f_n \leq \liminf \int f_n$.

**Use case**: Provides inequality when equality might fail.

## Teaching Tips

### Explaining Measurability

**Analogy**: Measurable sets are "well-behaved" enough to assign size. Non-measurable sets are pathological constructions using Axiom of Choice.

**Reassurance**: All sets encountered in practice (intervals, open/closed, Borel) are measurable.

### Visualizing Lebesgue Integration

**Riemann**: Vertical slices (partition domain)  
**Lebesgue**: Horizontal slices (partition range)

Draw both approaches for same function to show difference.

### Common Misconceptions

|Misconception|Correction|
|---|---|
|"Lebesgue integral is always bigger"|Equals Riemann when both exist|
|"All functions are measurable"|Non-measurable functions exist (but are exotic)|
|"L^2 convergence means pointwise"|No—only convergence in norm|
|"Measure zero means empty"|Rationals have measure zero but aren't empty|
