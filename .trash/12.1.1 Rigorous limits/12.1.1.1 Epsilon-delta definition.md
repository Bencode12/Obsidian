## Definition

Let $f: D \to \mathbb{R}$ where $D \subseteq \mathbb{R}$ and let $a$ be a limit point of $D$. We say that:

$$\lim_{x \to a} f(x) = L$$

if for every $\epsilon > 0$, there exists $\delta > 0$ such that for all $x \in D$:

$$0 < |x - a| < \delta \implies |f(x) - L| < \epsilon$$

> [!warning] Critical Detail
> The condition $0 < |x - a|$ means $x \neq a$. The limit describes behavior **near** $a$, not **at** $a$.

---

## Understanding the Definition

### Logical Structure

The definition has the form: $\forall \epsilon > 0, \exists \delta > 0$ such that $P(\epsilon, \delta)$

|Step|Who|Action|
|---|---|---|
|1|Challenger|Picks any $\epsilon > 0$ (no matter how small)|
|2|Defender|Responds with $\delta > 0$ (depends on $\epsilon$ and $a$)|
|3|Guarantee|All $x$ within $\delta$ of $a$ have $f(x)$ within $\epsilon$ of $L$|

### Geometric Interpretation

- Draw horizontal lines at $y = L \pm \epsilon$ (the "epsilon band")
- Find vertical lines at $x = a \pm \delta$ (the "delta interval")
- The function must stay within the epsilon band whenever $x$ is in the delta interval

---

## Proof Strategy Template

To prove $\lim_{x \to a} f(x) = L$:

1. **Start**: "Let $\epsilon > 0$ be given"
2. **Choose**: Find/define $\delta$ (often in terms of $\epsilon$)
3. **Assume**: "Suppose $0 < |x - a| < \delta$"
4. **Show**: Prove that $|f(x) - L| < \epsilon$ using algebraic manipulation
5. **Conclude**: "Therefore, $\lim_{x \to a} f(x) = L$"

## Worked Examples

### Linear Function

**Prove**: $\lim_{x \to 2} (3x + 1) = 7$

Let $\epsilon > 0$ be given. Choose $\delta = \epsilon/3$.

Suppose $0 < |x - 2| < \delta$. Then:
$$|(3x + 1) - 7| = |3x - 6| = 3|x - 2| < 3\delta = 3 \cdot \frac{\epsilon}{3} = \epsilon$$

Therefore, $\lim_{x \to 2} (3x + 1) = 7$. âˆŽ

### Quadratic Function

**Prove**: $\lim_{x \to 1} x^2 = 1$

Let $\epsilon > 0$ be given. We need to bound $|x^2 - 1| = |x - 1||x + 1|$.

First, restrict $\delta \leq 1$. Then $|x - 1| < 1$ implies $0 < x < 2$, so $|x + 1| < 3$.

Choose $\delta = \min\{1, \epsilon/3\}$.

Suppose $0 < |x - 1| < \delta$. Then:
$$|x^2 - 1| = |x - 1||x + 1| < \delta \cdot 3 \leq \frac{\epsilon}{3} \cdot 3 = \epsilon$$

Therefore, $\lim_{x \to 1} x^2 = 1$. âˆŽ

### Rational Function

**Prove**: $\lim_{x \to 2} \frac{1}{x} = \frac{1}{2}$

Need to bound $\left|\frac{1}{x} - \frac{1}{2}\right| = \frac{|x - 2|}{2|x|}$ from above.

Restrict $\delta \leq 1$, so $1 < x < 3$, giving $\frac{1}{|x|} < 1$.

Choose $\delta = \min\{1, \epsilon\}$.

> [!example] Key Insight
> For rational functions, control the denominator by restricting $\delta$ to keep $x$ away from zeros.

## Common Techniques

### Bounding Strategy

When proving limits, you often need to:

1. Factor or manipulate $|f(x) - L|$ into a product
2. Bound one factor by restricting $\delta$
3. Make the other factor small by choosing $\delta$ appropriately

### Minimum Delta

Often use $\delta = \min\{\delta_1, \delta_2\}$ to satisfy multiple constraints simultaneously.

### Triangle Inequality

Useful for breaking complex expressions:

$$|a + b| \leq |a| + |b|$$

## Negation of the Definition

To prove a limit does **NOT** exist, show:

$$\exists \epsilon > 0 \text{ such that } \forall \delta > 0, \exists x \text{ with } 0 < |x - a| < \delta \text{ but } |f(x) - L| \geq \epsilon$$

> [!example] Classic Non-Existence
> Show $\lim_{x \to 0} \sin(1/x)$ does not exist
> 
> Choose $\epsilon = 1/2$. For any $\delta > 0$, we can find $x_1, x_2$ with $0 < |x_i| < \delta$ but $\sin(1/x_1) = 1$ and $\sin(1/x_2) = -1$, so they can't both be within $\epsilon$ of any single limit $L$.

## Practice Problems

Prove the following limits using epsilon-delta definitions:

- $\lim_{x \to 3} (2x - 5) = 1$
- $\lim_{x \to 0} x^3 = 0$
- $\lim_{x \to 4} \sqrt{x} = 2$ (hint: multiply by conjugate)
- Show that $\lim_{x \to 0} \sin(1/x)$ does not exist
- Prove that if $\lim_{x \to a} f(x) = L_1$ and $\lim_{x \to a} f(x) = L_2$, then $L_1 = L_2$ (uniqueness)
