## Overview

**Hamiltonian mechanics** reformulates classical mechanics using symplectic geometry. Instead of Newton's second-order equations, we get first-order equations on phase space.

---

## Hamiltonian Vector Fields

### Definition

Given a smooth function $H: M \to \mathbb{R}$ (the **Hamiltonian**) on a symplectic manifold $(M, \omega)$, the **Hamiltonian vector field** $X_H$ is defined by:

$$\iota_{X_H}\omega = dH$$

or equivalently:
$$\omega(X_H, \cdot) = dH$$

where $\iota$ denotes interior product.

### Existence and Uniqueness

Since $\omega$ is nondegenerate, this equation uniquely determines $X_H$.

In coordinates, if $\omega = \sum dq^i \wedge dp_i$:
$$X_H = \sum_{i=1}^n \left(\frac{\partial H}{\partial p_i}\frac{\partial}{\partial q^i} - \frac{\partial H}{\partial q^i}\frac{\partial}{\partial p_i}\right)$$

---

## Hamilton's Equations

The flow of $X_H$ satisfies **Hamilton's equations**:

$$\dot{q}^i = \frac{\partial H}{\partial p_i}, \quad \dot{p}_i = -\frac{\partial H}{\partial q^i}$$

for $i = 1, ..., n$.

### Matrix Form

Let $z = (q^1, ..., q^n, p_1, ..., p_n)^T$. Then:

$$\dot{z} = J\nabla H$$

where $J = \begin{pmatrix} 0 & I_n \\ -I_n & 0 \end{pmatrix}$ and $\nabla H$ is the gradient.

```desmos-graph
y = x^2
y' = 2x
```

> [!example] Physical Interpretation
> - $q^i$: generalized coordinates (positions)  
> - $p_i$: conjugate momenta  
> - $H$: total energy (kinetic + potential)  
> - Hamilton's equations: time evolution of the system

## Examples

### Example 1: Harmonic Oscillator

Hamiltonian:
$$H(q,p) = \frac{p^2}{2m} + \frac{1}{2}kq^2$$

Hamilton's equations:
$$\dot{q} = \frac{p}{m}, \quad \dot{p} = -kq$$

Solution: Simple harmonic motion with frequency $\omega = \sqrt{k/m}$.

Phase portrait: Ellipses in $(q,p)$ plane.

### Example 2: Pendulum

Hamiltonian:
$$H(\theta, p_\theta) = \frac{p_\theta^2}{2ml^2} - mgl\cos\theta$$

Hamilton's equations:
$$\dot{\theta} = \frac{p_\theta}{ml^2}, \quad \dot{p}_\theta = -mgl\sin\theta$$

This gives the nonlinear pendulum equation: $\ddot{\theta} + \frac{g}{l}\sin\theta = 0$.

### Example 3: Kepler Problem

For a particle in a central potential $V(r) = -k/r$:

$$H = \frac{|\mathbf{p}|^2}{2m} - \frac{k}{r}$$

Using spherical coordinates and conservation of angular momentum, this reduces to an effective 1D problem.

Solutions are conic sections (ellipses, parabolas, hyperbolas).

## Conservation Laws

### Energy Conservation

If $H$ doesn't explicitly depend on time:

$$\frac{dH}{dt} = \{H, H\} = 0$$

Energy is conserved along trajectories!

### Poisson Brackets

The **Poisson bracket** of two functions $F, G: M \to \mathbb{R}$ is:

$$\{F, G\} = \omega(X_F, X_G) = X_G(F) = -X_F(G)$$

In canonical coordinates:
$$\{F, G\} = \sum_{i=1}^n \left(\frac{\partial F}{\partial q^i}\frac{\partial G}{\partial p_i} - \frac{\partial F}{\partial p_i}\frac{\partial G}{\partial q^i}\right)$$

### Properties

1. **Antisymmetry**: $\{F, G\} = -\{G, F\}$
2. **Jacobi identity**: $\{F, \{G, H\}\} + \{G, \{H, F\}\} + \{H, \{F, G\}\} = 0$
3. **Leibniz rule**: $\{FG, H\} = F\{G, H\} + G\{F, H\}$

### Time Evolution

For any observable $F$:
$$\frac{dF}{dt} = \{F, H\} + \frac{\partial F}{\partial t}$$

If $\{F, H\} = 0$, then $F$ is a **conserved quantity** (constant of motion).

## Noether's Theorem

### Statement

Every continuous symmetry of the Hamiltonian corresponds to a conserved quantity.

**Precisely**: If $\phi_t$ is a 1-parameter group of symplectomorphisms preserving $H$, then there exists a conserved quantity $F$ such that $X_F$ generates the symmetry.

### Examples

|Symmetry|Conserved Quantity|
|---|---|
|Time translation|Energy ($H$)|
|Spatial translation|Momentum|
|Rotation|Angular momentum|

## Liouville's Theorem

### Statement

The flow $\phi_t$ of a Hamiltonian vector field preserves the symplectic form:

$$\phi_t^*\omega = \omega$$

Therefore, it also preserves:
- The volume form $\omega^n$
- Phase space volume

### Proof Sketch

$$\frac{d}{dt}\phi_t^*\omega = \phi_t^*(\mathcal{L}_{X_H}\omega) = \phi_t^*(d\iota_{X_H}\omega + \iota_{X_H}d\omega) = \phi_t^*(d(dH) + 0) = 0$$

### Physical Meaning

Phase space volume is conserved under time evolution. This is fundamental to statistical mechanics!

## Integrable Systems

### Definition

A Hamiltonian system with $n$ degrees of freedom is **completely integrable** if there exist $n$ independent conserved quantities $F_1, ..., F_n$ in involution:

$$\{F_i, F_j\} = 0 \quad \text{for all } i,j$$

### Arnold-Liouville Theorem

For a completely integrable system, the level sets $\{F_1 = c_1, ..., F_n = c_n\}$ are Lagrangian tori, and the flow is linear on these tori.

### Action-Angle Variables

On each torus, we can find coordinates $(I_1, ..., I_n, \theta_1, ..., \theta_n)$ such that:
- $H = H(I_1, ..., I_n)$ (depends only on actions)
- $\dot{I}_i = 0$, $\dot{\theta}_i = \omega_i(I)$ (angles evolve linearly)

**Examples**: Harmonic oscillator, Kepler problem, rigid body.

---

## Canonical Transformations

A transformation $(q,p) \to (Q,P)$ is **canonical** if it preserves the symplectic form:

$$\sum dq^i \wedge dp_i = \sum dQ^i \wedge dP_i$$

Equivalently, it preserves Poisson brackets.

### Generating Functions

Canonical transformations can be generated by functions. For example, type 2 generating function $F_2(q,P,t)$:
$$p_i = \frac{\partial F_2}{\partial q^i}, \quad Q^i = \frac{\partial F_2}{\partial P_i}$$

## Applications

### Celestial Mechanics

Planetary motion, stability of solar system, KAM theory.

### Statistical Mechanics

Liouville's theorem → microcanonical ensemble → thermodynamics.

### Quantum Mechanics

Canonical quantization: Replace Poisson brackets with commutators:
$$\{F, G\} \to \frac{1}{i\hbar}[\hat{F}, \hat{G}]$$

### Plasma Physics

Vlasov equation, charged particle dynamics.

## Common Mistakes

> [!warning] Confusing Hamiltonian with Lagrangian
> Lagrangian: $L(q, \dot{q})$ on tangent bundle $TQ$  
> Hamiltonian: $H(q, p)$ on cotangent bundle $T^*Q$  
> They're related by Legendre transform, but live on different spaces!

> [!warning] Forgetting Sign Conventions
> Different authors use different sign conventions for Poisson brackets and Hamilton's equations. Always check!

> [!warning] Assuming All Systems Are Integrable
> Most Hamiltonian systems are NOT integrable. Chaos is generic!

> [!warning] Mixing Up Total and Partial Time Derivatives
> $\frac{dF}{dt} = \{F, H\} + \frac{\partial F}{\partial t}$ includes explicit time dependence!
