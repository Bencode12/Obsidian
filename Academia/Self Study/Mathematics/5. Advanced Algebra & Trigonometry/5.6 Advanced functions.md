## Types of Advanced Functions

### Composite Function

Combining two functions — the output of one becomes the input of the other:

$$(f \circ g)(x) = f(g(x))$$

> [!example] $f(x) = x^2$, $g(x) = x + 1$ $(f \circ g)(x) = (x+1)^2$

---

### Inverse Function

Reverses the effect of a function: $f(f^{-1}(x)) = x$

**Steps to find $f^{-1}(x)$:**

1. Replace $f(x)$ with $y$
2. Swap $x$ and $y$
3. Solve for $y$

> [!example] $f(x) = 2x + 3$ Swap: $x = 2y + 3 \implies y = \dfrac{x-3}{2}$ So $f^{-1}(x) = \dfrac{x-3}{2}$

---

### Piecewise Function

Defined by different rules on different intervals:

$$f(x) = \begin{cases} x^2 & x \geq 0 \ -x & x < 0 \end{cases}$$

---

### Exponential and Logarithmic Functions

$$f(x) = a^x \qquad f(x) = \log_a x$$

These are inverses of each other. See [[3.10_Exponential_and_Logarithmic_Functions]].

---

### Trigonometric Functions

Periodic — repeat after a fixed interval. Inverse trig functions: $\sin^{-1}$, $\cos^{-1}$, $\tan^{-1}$.

---

## Key Properties

|Property|Description|
|---|---|
|**Domain and range**|What values $x$ can take; what values $f(x)$ can produce|
|**Monotonicity**|Whether the function is increasing or decreasing on an interval|
|**Periodicity**|Whether $f(x+T) = f(x)$ for some period $T$|
|**Symmetry**|Even: $f(-x) = f(x)$; Odd: $f(-x) = -f(x)$|
