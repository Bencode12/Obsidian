## **Lesson 10.2.4 — Moment Generating Functions (MGFs)**

Moment generating functions provide a **compact way to encode all moments** (mean, variance, skewness, etc.) of a random variable. They are extremely useful for **deriving distributions, sums of independent variables, and proving the Central Limit Theorem**.

---

### **1. Definition**

For a random variable (X), the **moment generating function** (M_X(t)) is defined as:

- **Discrete case:**  
    [  
    M_X(t) = E[e^{tX}] = \sum_{x} e^{tx} P(X=x)  
    ]
    
- **Continuous case:**  
    [  
    M_X(t) = E[e^{tX}] = \int_{-\infty}^{\infty} e^{tx} f_X(x) dx  
    ]
    
- (t) is a real number (or in some cases complex number) for which the expectation exists.
    

---

### **2. Properties**

1. **Moments from derivatives:**
    

[  
E[X^n] = M_X^{(n)}(0) = \left. \frac{d^n}{dt^n} M_X(t) \right|_{t=0}  
]

- First derivative at 0 → mean: (M_X'(0) = E[X])
    
- Second derivative at 0 → second moment: (M_X''(0) = E[X^2])
    

2. **Uniqueness:**
    

- If two random variables have the **same MGF**, they have the **same distribution** (when MGF exists in an interval around 0).
    

3. **Sum of independent variables:**
    

- If (X) and (Y) are independent:  
    [  
    M_{X+Y}(t) = M_X(t) M_Y(t)  
    ]
    
- Extends to any finite sum of independent random variables.
    

---

### **3. Common MGFs**

1. **Bernoulli (X \sim \text{Bern}(p))**
    

[  
M_X(t) = (1-p) + p e^t  
]

2. **Binomial (X \sim \text{Bin}(n,p))**
    

[  
M_X(t) = [(1-p) + p e^t]^n  
]

3. **Poisson (X \sim \text{Pois}(\lambda))**
    

[  
M_X(t) = \exp(\lambda (e^t - 1))  
]

4. **Normal (X \sim N(\mu, \sigma^2))**
    

[  
M_X(t) = \exp\Big(\mu t + \frac{\sigma^2 t^2}{2}\Big)  
]

5. **Exponential (X \sim \text{Exp}(\lambda))**
    

[  
M_X(t) = \frac{\lambda}{\lambda - t}, \quad t < \lambda  
]

---

### **4. Using MGFs**

1. **Compute moments:**
    

- (E[X] = M_X'(0)), (E[X^2] = M_X''(0)), etc.
    

2. **Find distribution of sums:**
    

- If (X_1, X_2, \dots, X_n) are independent:  
    [  
    M_{S_n}(t) = \prod_{i=1}^n M_{X_i}(t)  
    ]
    

3. **Prove convergence to normal:**
    

- CLT can be elegantly derived using MGFs.
    

---

### **5. Examples**

1. **Mean and variance of Poisson(λ) using MGF:**
    

[  
M_X(t) = e^{\lambda(e^t - 1)}  
]  
[  
E[X] = M'_X(0) = \lambda, \quad E[X^2] = M''_X(0) = \lambda + \lambda^2  
]  
(\text{Var}(X) = E[X^2] - (E[X])^2 = \lambda)

2. **Sum of two independent Poisson(λ1) and Poisson(λ2):**
    

[  
M_{X+Y}(t) = e^{\lambda_1(e^t-1)} e^{\lambda_2(e^t-1)} = e^{(\lambda_1+\lambda_2)(e^t-1)}  
]

- Sum is Poisson(λ1+λ2)
