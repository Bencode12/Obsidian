## Transformations of Single Variables

### Method 1: CDF Technique

For $Y = g(X)$ where $g$ is monotone:

1. Find CDF of $Y$:
   $$F_Y(y) = P(Y \leq y) = P(g(X) \leq y)$$

2. Differentiate to get PDF:
   $$f_Y(y) = \frac{d}{dy}F_Y(y)$$

### Method 2: Change of Variables Formula

If $Y = g(X)$ and $g$ is differentiable and monotone:

$$f_Y(y) = f_X(x)\left|\frac{dx}{dy}\right| = f_X(g^{-1}(y))\left|\frac{d}{dy}g^{-1}(y)\right|$$

where $x = g^{-1}(y)$.

### Example: Linear Transformation

If $Y = aX + b$ with $a \neq 0$:

$$f_Y(y) = \frac{1}{|a|}f_X\left(\frac{y-b}{a}\right)$$

**Special case**: If $X \sim N(\mu, \sigma^2)$, then $aX+b \sim N(a\mu+b, a^2\sigma^2)$.

## Common Transformations

### Square Transformation

If $Y = X^2$ and $X$ has symmetric distribution about 0:

$$f_Y(y) = \frac{1}{2\sqrt{y}}[f_X(\sqrt{y}) + f_X(-\sqrt{y})], \quad y > 0$$

**Example**: If $X \sim N(0,1)$, then $Y = X^2 \sim \chi^2_1$ (chi-squared with 1 df).

### Log Transformation

If $Y = \log(X)$ and $X > 0$:

$$f_Y(y) = e^y f_X(e^y)$$

**Example**: If $Y \sim N(\mu, \sigma^2)$, then $X = e^Y$ is **lognormal**.

### Reciprocal Transformation

If $Y = 1/X$:

$$f_Y(y) = \frac{1}{y^2}f_X\left(\frac{1}{y}\right)$$

## Functions of Several Variables

### Joint Transformations

For $(U,V) = g(X,Y)$ where $g$ is one-to-one:

$$f_{U,V}(u,v) = f_{X,Y}(x,y)|J|$$

where $J$ is the **Jacobian determinant**:

$$J = \det\begin{pmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \\ \frac{\partial y}{\partial u} & \frac{\partial y}{\partial v} \end{pmatrix}$$

### Steps

1. Solve for $x = h_1(u,v)$ and $y = h_2(u,v)$
2. Compute Jacobian $J$
3. Substitute into joint PDF
4. Multiply by $|J|$

## Important Examples

### Sum of Independent Variables

If $Z = X + Y$ and $X, Y$ independent:

$$f_Z(z) = \int_{-\infty}^\infty f_X(x)f_Y(z-x)\,dx$$

This is the **convolution** of $f_X$ and $f_Y$!

**Example**: Sum of independent normals is normal.

### Product of Independent Variables

If $Z = XY$ and $X, Y$ independent:

$$f_Z(z) = \int_{-\infty}^\infty \frac{1}{|x|}f_X(x)f_Y\left(\frac{z}{x}\right)\,dx$$

### Ratio of Independent Variables

If $Z = X/Y$ and $X, Y$ independent:

$$f_Z(z) = \int_{-\infty}^\infty |y|f_X(zy)f_Y(y)\,dy$$

**Example**: If $X, Y \sim N(0,1)$ independent, then $X/Y \sim \text{Cauchy}(0,1)$.

## Order Statistics

### Definition

Arrange sample in ascending order: $X_{(1)} \leq X_{(2)} \leq \cdots \leq X_{(n)}$

These are **order statistics**.

### Distribution of Maximum

For i.i.d. sample with CDF $F$:

$$F_{X_{(n)}}(x) = [F(x)]^n$$

$$f_{X_{(n)}}(x) = n[F(x)]^{n-1}f(x)$$

### Distribution of Minimum

$$F_{X_{(1)}}(x) = 1 - [1-F(x)]^n$$

$$f_{X_{(1)}}(x) = n[1-F(x)]^{n-1}f(x)$$

### Joint Distribution

For $i < j$, joint PDF of $X_{(i)}$ and $X_{(j)}$:

$$f_{X_{(i)},X_{(j)}}(u,v) = \frac{n!}{(i-1)!(j-i-1)!(n-j)!}[F(u)]^{i-1}[F(v)-F(u)]^{j-i-1}[1-F(v)]^{n-j}f(u)f(v)$$

for $u < v$.

## Moment Generating Function Method

### Theorem

If $Y = g(X)$, find MGF of $Y$:

$$M_Y(t) = E[e^{tY}] = E[e^{tg(X)}]$$

Then identify distribution from MGF.

### Example: Sum of Independent Variables

If $X$ and $Y$ independent:

$$M_{X+Y}(t) = M_X(t)M_Y(t)$$

Much easier than convolution!

## Simulation Methods

### Inverse Transform Method

To generate random variable with CDF $F$:

1. Generate $U \sim U[0,1]$
2. Return $X = F^{-1}(U)$

Works when inverse CDF is available.

### Acceptance-Rejection Method

To sample from $f$ using proposal $g$:

1. Generate $Y \sim g$
2. Generate $U \sim U[0,1]$
3. Accept $Y$ if $U \leq \frac{f(Y)}{Mg(Y)}$, else reject and repeat

Requires $f(x) \leq Mg(x)$ for all $x$.

## Common Mistakes

> [!warning] Forgetting Absolute Value in Jacobian
> Always use $|J|$, not just $J$. Determinant can be negative!

> [!warning] Ignoring Support Changes
> Transformation may change the support. Check domain carefully!

> [!warning] Assuming Linearity
> $E[g(X)] \neq g(E[X])$ in general. Use LOTUS instead.
