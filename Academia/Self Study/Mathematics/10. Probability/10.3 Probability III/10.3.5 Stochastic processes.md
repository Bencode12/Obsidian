## **Lesson 10.3.5 — Stochastic Processes**

A **stochastic process** is a family of random variables ({X_t : t \in T}) indexed by **time or space**, describing the evolution of a **system under uncertainty** over time. This is central to probability theory, finance, physics, and engineering.

---

### **1. Definition**

- Let ((\Omega, \mathcal{F}, P)) be a probability space.
    
- A stochastic process is a collection of random variables:
    

[  
X = { X_t : t \in T }  
]

- (t \in T) can be **discrete** ((t = 0,1,2,\dots)) or **continuous** ((t \in [0,\infty)))
    
- Each (X_t) maps outcomes (\omega \in \Omega) to (\mathbb{R}) (or (\mathbb{R}^n))
    
- **Notation:**  
    [  
    X_t(\omega) = \text{value of the process at time } t \text{ for outcome } \omega  
    ]
    

---

### **2. Classification of Stochastic Processes**

1. **Discrete-Time vs Continuous-Time:**
    

- Discrete: (t = 0,1,2,\dots) (e.g., random walk)
    
- Continuous: (t \in [0,\infty)) (e.g., Brownian motion)
    

2. **State Space:**
    

- Discrete: (X_t \in {0,1,2,\dots})
    
- Continuous: (X_t \in \mathbb{R})
    

3. **Independent Increments:**
    

- A process has independent increments if non-overlapping increments are independent:  
    [  
    X_{t_2}-X_{t_1} \perp X_{t_4}-X_{t_3}  
    ]
    

4. **Stationary Increments:**
    

- Distribution of increments depends only on the **length of time interval**, not start time.
    

---

### **3. Examples of Stochastic Processes**

1. **Random Walk (Discrete-Time):**  
    [  
    S_0 = 0, \quad S_n = S_{n-1} + X_n, \quad X_n = \pm 1 \text{ with probability } 0.5  
    ]
    
2. **Poisson Process (Continuous-Time):**
    

- Counts **number of events in time interval [0,t]**
    
- Properties:
    
    - (N(0) = 0)
        
    - Independent increments
        
    - (P(N(t+s)-N(s) = k) = e^{-\lambda t} (\lambda t)^k / k!)
        

3. **Brownian Motion (Wiener Process):**
    

- (B_0 = 0), continuous paths, independent increments, (B_t - B_s \sim N(0, t-s))
    
- Foundation for **stochastic calculus, option pricing, and SDEs**
    

4. **Martingale Processes:**
    

- (X_n) is a martingale: (E[X_{n+1} \mid \mathcal{F}_n] = X_n)
    

---

### **4. Key Concepts**

1. **Filtration:**  
    ({\mathcal{F}_t}) represents all information available up to time (t)
    
2. **Adapted Process:**  
    (X_t) is measurable with respect to (\mathcal{F}_t), i.e., no “future” information is used
    
3. **Markov Property:**
    

- Future depends only on present, not past:  
    [  
    P(X_{t+1} \mid X_t, X_{t-1}, \dots) = P(X_{t+1} \mid X_t)  
    ]
    

4. **Stationarity:**
    

- Distribution of (X_{t+s}) is the same as (X_t) (for stationary process)
    

---

### **5. Applications**

1. **Finance:** Stock prices, derivative pricing (Black-Scholes uses Brownian motion)
    
2. **Queueing theory:** Arrival and service processes modeled as Poisson or Markov processes
    
3. **Physics:** Particle diffusion (Brownian motion), stochastic differential equations
    
4. **Engineering:** Signal processing, control systems with noise
