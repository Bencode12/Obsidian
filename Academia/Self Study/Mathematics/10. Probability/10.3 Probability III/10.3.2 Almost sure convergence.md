## **Lesson 10.3.2 — Almost Sure Convergence**

Almost sure convergence is one of the **strongest forms of convergence** for sequences of random variables. It is foundational in probability theory, measure-theoretic probability, and stochastic processes.

---

### **1. Definition**

Let (X_1, X_2, \dots) be a sequence of random variables defined on a probability space ((\Omega, \mathcal{F}, P)). We say:

[  
X_n \xrightarrow{\text{a.s.}} X \quad \text{(almost surely)}  
]

if:

[  
P\Big({\omega \in \Omega : \lim_{n \to \infty} X_n(\omega) = X(\omega)}\Big) = 1  
]

- Intuition: **for almost every outcome**, the sequence (X_n(\omega)) converges to (X(\omega)).
    
- “Almost surely” means **except on a set of probability zero**.
    

---

### **2. Properties**

1. **Strong form:**
    

- Almost sure convergence implies convergence in probability:  
    [  
    X_n \xrightarrow{\text{a.s.}} X \implies X_n \xrightarrow{P} X  
    ]
    
- Converse is **not always true**.
    

2. **Operations on limits:**
    

- If (X_n \xrightarrow{\text{a.s.}} X) and (Y_n \xrightarrow{\text{a.s.}} Y), then:  
    [  
    X_n + Y_n \xrightarrow{\text{a.s.}} X + Y  
    ]  
    [  
    X_n Y_n \xrightarrow{\text{a.s.}} XY  
    ]  
    [  
    g(X_n) \xrightarrow{\text{a.s.}} g(X) \quad \text{for continuous } g  
    ]
    

3. **Relation with expectations:**
    

- (X_n \xrightarrow{\text{a.s.}} X) does **not** imply (E[X_n] \to E[X]) unless dominated convergence conditions hold.
    

---

### **3. Examples**

1. **Coin Toss Law of Large Numbers:**
    

- Let (X_i = 1) for heads, 0 for tails, independent and fair.
    
- Sample mean:  
    [  
    \bar{X}_n = \frac{1}{n} \sum_{i=1}^n X_i  
    ]
    
- Then: (\bar{X}_n \xrightarrow{\text{a.s.}} 0.5) (Strong Law of Large Numbers)
    

2. **Deterministic sequence:**
    

- (X_n(\omega) = \frac{1}{n}) for all (\omega)
    
- Then (X_n \xrightarrow{\text{a.s.}} 0) (converges for all outcomes, probability 1)
    

3. **Counterexample for convergence in probability but not a.s.:**
    

- Construct (X_n(\omega) = 1) if (\omega \in [0,1/n]), 0 otherwise, on (\Omega = [0,1]) with uniform measure.
    
- (X_n \xrightarrow{P} 0) but **no single (\omega) converges to 0 for all n**, so almost sure convergence fails.
    

---

### **4. Techniques to Prove Almost Sure Convergence**

1. **Borel-Cantelli Lemma:**
    

- Let (A_n) be events with (\sum P(A_n) < \infty)
    
- Then (P(A_n \text{ infinitely often}) = 0)
    
- Useful to show (\lim X_n = X) almost surely
    

2. **Monotone sequences:**
    

- If (X_n \uparrow X) almost surely, limit is well-defined
    

3. **Strong Law of Large Numbers (SLLN):**
    

- Provides almost sure convergence of sample mean for i.i.d. variables with finite expectation
    

---

### **5. Exercises**

1. Let (X_n \sim \text{Uniform}[0,1/n]). Show that (X_n \xrightarrow{\text{a.s.}} 0).
    
2. Toss a fair coin infinitely many times. Let (X_n = 1) if heads on n-th toss, else 0. Show (\frac{1}{n}\sum_{i=1}^{n} X_i \xrightarrow{\text{a.s.}} 0.5).
    
3. Construct a sequence (X_n) that converges in probability to 0 but not almost surely.
    
4. Apply Borel-Cantelli lemma: Let (P(A_n) = 1/n^2), show (A_n) occurs only finitely often almost surely.
