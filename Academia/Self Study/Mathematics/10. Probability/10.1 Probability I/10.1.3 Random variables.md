## **Lesson 10.1.3 — Random Variables**

Random variables are the bridge between **outcomes of an experiment** and **quantitative analysis**. They allow us to assign numbers to outcomes and study their distribution, expectation, and variability.

---

### **1. Definition**

A **random variable** (X) on a probability space ((\Omega, \mathcal{F}, P)) is a **measurable function**:

[  
X: \Omega \to \mathbb{R} \quad (\text{or } \mathbb{C})  
]

- “Measurable” ensures that for any real number (x):
    

[  
{\omega \in \Omega : X(\omega) \le x} \in \mathcal{F}  
]

- This guarantees we can **assign probabilities to events like (X \le x)**.
    

---

### **2. Types of Random Variables**

1. **Discrete Random Variable**
    

- Takes **countable values** ({x_1, x_2, \dots})
    
- Described by a **Probability Mass Function (PMF)**:
    

[  
p_X(x_i) = P(X = x_i)  
]

- Properties:  
    (\sum_i p_X(x_i) = 1), (p_X(x_i) \ge 0)
    

**Examples:**

- Number of heads in 3 coin tosses
    
- Roll of a die
    

---

2. **Continuous Random Variable**
    

- Takes values in a **continuous range** ([a,b]) or (\mathbb{R})
    
- Described by a **Probability Density Function (PDF)** (f_X(x)):
    

[  
P(a \le X \le b) = \int_a^b f_X(x) , dx  
]

- Properties:  
    (f_X(x) \ge 0), (\int_{-\infty}^{\infty} f_X(x) dx = 1)
    

**Example:** Uniform random variable on [0,1], (f_X(x) = 1) for (0 \le x \le 1)

---

### **3. Cumulative Distribution Function (CDF)**

For **any random variable** (X):

[  
F_X(x) = P(X \le x)  
]

- Non-decreasing, right-continuous, limits:  
    (\lim_{x \to -\infty} F_X(x) = 0), (\lim_{x \to \infty} F_X(x) = 1)
    
- For discrete (X): step function
    
- For continuous (X): (F_X(x) = \int_{-\infty}^x f_X(t) dt)
    

---

### **4. Expectation and Moments**

1. **Expected value (mean)**
    

- Discrete: (E[X] = \sum_i x_i p_X(x_i))
    
- Continuous: (E[X] = \int_{-\infty}^{\infty} x f_X(x) dx)
    

2. **Variance**  
    [  
    \text{Var}(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2  
    ]
    
3. **Higher moments**
    

- (E[X^n]) measures **spread, skewness, kurtosis**, etc.
    

---

### **5. Important Discrete Distributions**

- **Bernoulli**: (X \in {0,1}), (P(X=1)=p)
    
- **Binomial**: Sum of (n) independent Bernoulli trials
    
- **Geometric**: Number of trials until first success
    
- **Poisson**: Counts of rare events, parameter (\lambda)
    

---

### **6. Important Continuous Distributions**

- **Uniform [a,b]**: constant density
    
- **Normal (Gaussian)**: (f_X(x) = \frac{1}{\sqrt{2\pi \sigma^2}} e^{-(x-\mu)^2/(2\sigma^2)})
    
- **Exponential**: Time until event, (f_X(x) = \lambda e^{-\lambda x}, x \ge 0)
