## **Lesson 10.1.2 — Probability Measures (Axioms of Probability)**

Once we have a **sample space** (\Omega) and a **σ-algebra** (\mathcal{F}), a **probability measure** formalizes the notion of “how likely events are” using **Kolmogorov’s axioms**.

---

### **1. Probability Measure Definition**

A **probability measure** (P: \mathcal{F} \to [0,1]) is a function that assigns a number to each event (A \in \mathcal{F}) such that:

1. **Non-negativity**  
    [  
    P(A) \ge 0 \quad \forall A \in \mathcal{F}  
    ]
    
2. **Normalization**  
    [  
    P(\Omega) = 1  
    ]
    
3. **Countable Additivity (σ-additivity)**  
    For any countable collection of **disjoint events** (A_1, A_2, A_3, \dots \in \mathcal{F}):
    

[  
P\Big(\bigcup_{i=1}^\infty A_i\Big) = \sum_{i=1}^\infty P(A_i)  
]

> These are known as **Kolmogorov’s axioms** and form the foundation of probability theory.

---

### **2. Consequences of the Axioms**

From the axioms, we can derive key properties:

1. **Probability of empty set:**  
    [  
    P(\emptyset) = 0  
    ]
    
2. **Complement rule:**  
    [  
    P(A^c) = 1 - P(A)  
    ]
    
3. **Monotonicity:** If (A \subseteq B), then (P(A) \le P(B))
    
4. **Inclusion-Exclusion (two events):**  
    [  
    P(A \cup B) = P(A) + P(B) - P(A \cap B)  
    ]
    
5. **Finite additivity:** Derived from countable additivity for finite collections of disjoint events.
    

---

### **3. Examples of Probability Measures**

#### **Discrete Uniform Probability**

- Sample space: (\Omega = {1,2,3,4,5,6}) (die roll)
    
- σ-algebra: power set (2^\Omega)
    
- Assign: (P({i}) = \frac{1}{6}, i = 1,\dots,6)
    
- Then for any event (A \subseteq \Omega):  
    [  
    P(A) = \sum_{i \in A} P({i})  
    ]
    

#### **Continuous Probability Measure (Uniform on [0,1])**

- Sample space: (\Omega = [0,1])
    
- σ-algebra: Borel σ-algebra
    
- Probability measure: (P([a,b]) = b-a) for (0 \le a \le b \le 1)
    
- Extended to all Borel sets via measure theory.
    

---

### **4. Conditional Probability and Multiplication Rule**

If (P(B) > 0), define **conditional probability**:

[  
P(A \mid B) = \frac{P(A \cap B)}{P(B)}  
]

- Leads to multiplication rule: (P(A \cap B) = P(A \mid B) P(B))
    
- Can be extended to **multiple events** using the chain rule.
    

---

### **5. Independence**

Two events (A) and (B) are **independent** if:

[  
P(A \cap B) = P(A) P(B)  
]

- Independence generalizes to random variables.
    
- Independence implies (P(A \mid B) = P(A)) and (P(B \mid A) = P(B))
