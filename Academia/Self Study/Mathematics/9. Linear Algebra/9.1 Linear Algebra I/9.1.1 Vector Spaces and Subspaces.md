## Definition

A **vector space** (or linear space) over a field $F$ (typically $\mathbb{R}$ or $\mathbb{C}$) is a set $V$ equipped with two operations:

1. **Vector addition**: $+ : V \times V \to V$
2. **Scalar multiplication**: $\cdot : F \times V \to V$

satisfying the following **axioms** for all $u,v,w \in V$ and $a,b \in F$:

### Addition Axioms

|Axiom|Statement|
|---|---|
|Associativity|$(u + v) + w = u + (v + w)$|
|Commutativity|$u + v = v + u$|
|Zero vector|$\exists 0 \in V$ such that $v + 0 = v$|
|Additive inverse|$\forall v \in V, \exists -v \in V$ such that $v + (-v) = 0$|

### Scalar Multiplication Axioms

|Axiom|Statement|
|---|---|
|Distributivity (vectors)|$a(u + v) = au + av$|
|Distributivity (scalars)|$(a + b)v = av + bv$|
|Compatibility|$a(bv) = (ab)v$|
|Identity|$1v = v$ where $1 \in F$|

> [!example] Why Vector Spaces Matter
> Vector spaces are the foundation of linear algebra. They generalize the concept of vectors beyond geometry to:  
> - Functions and sequences  
> - Polynomials  
> - Matrices  
> - Solutions to differential equations  
> - Quantum states in physics

## Examples

### Example 1: Euclidean Space $\mathbb{R}^n$

The prototypical example! Vectors are $n$-tuples:

$$\mathbf{v} = (v_1, v_2, ..., v_n)$$

Operations:
- Addition: $(u_1,...,u_n) + (v_1,...,v_n) = (u_1+v_1,...,u_n+v_n)$
- Scalar multiplication: $c(v_1,...,v_n) = (cv_1,...,cv_n)$

```desmos-graph
[0, 0] -> [3, 2]
[0, 0] -> [-1, 3]
[0, 0] -> [2, -2]
```

### Example 2: Function Space

Let $V = C([a,b])$ be the set of continuous real-valued functions on $[a,b]$.

Operations:
- $(f + g)(x) = f(x) + g(x)$
- $(cf)(x) = c \cdot f(x)$

This is an **infinite-dimensional** vector space!

### Example 3: Polynomial Space

$P_n = \{a_0 + a_1x + ... + a_nx^n : a_i \in \mathbb{R}\}$

Polynomials of degree ≤ n form a vector space of dimension $n+1$.

### Example 4: Matrix Space

$M_{m \times n}(\mathbb{R})$ = all $m \times n$ matrices with real entries

Dimension: $mn$

### Example 5: Solution Space

The set of solutions to a homogeneous linear system $A\mathbf{x} = 0$ forms a vector space (the **null space**).

## Subspaces

### Definition

A subset $W \subseteq V$ is a **subspace** if:

1. $0 \in W$ (contains zero vector)
2. Closed under addition: $u,v \in W \implies u+v \in W$
3. Closed under scalar multiplication: $v \in W, c \in F \implies cv \in W$

> [!tip] Subspace Test
> To verify $W$ is a subspace, check these three conditions. If any fails, $W$ is NOT a subspace!

### Examples of Subspaces

**In $\mathbb{R}^3$**:
- Lines through the origin
- Planes through the origin
- The trivial subspaces: $\{0\}$ and $\mathbb{R}^3$ itself

**Not subspaces**:
- A line NOT through the origin
- The first quadrant $\{(x,y) : x,y \geq 0\}$ (not closed under scalar multiplication by negative numbers)

## Key Properties

### Consequences of Axioms

From the vector space axioms, we can prove:

1. **Uniqueness of zero**: The zero vector is unique
2. **Uniqueness of inverses**: Each vector has a unique additive inverse
3. **Zero scalar**: $0v = 0$ for all $v \in V$
4. **Negative scalar**: $(-1)v = -v$
5. **Cancellation**: If $u + w = v + w$, then $u = v$

### Proof Example

**Claim**: $0v = 0$ for all $v \in V$.

**Proof**:
$$\begin{align*}
0v &= (0 + 0)v && \text{(additive identity in } F\text{)} \\
&= 0v + 0v && \text{(distributivity)} \\
\end{align*}$$

Adding $-(0v)$ to both sides:
$$0 = 0v$$

∎

## Operations on Subspaces

### Intersection

If $W_1, W_2$ are subspaces of $V$, then $W_1 \cap W_2$ is also a subspace.

### Sum

The **sum** of subspaces:
$$W_1 + W_2 = \{w_1 + w_2 : w_1 \in W_1, w_2 \in W_2\}$$

This is the smallest subspace containing both $W_1$ and $W_2$.

### Direct Sum

If $W_1 \cap W_2 = \{0\}$, we write:
$$V = W_1 \oplus W_2$$

Every vector in $V$ has a **unique** decomposition as $w_1 + w_2$.

## Common Mistakes

> [!warning] Forgetting the Zero Vector
> A subset must contain the zero vector to be a subspace! Check this first.

> [!warning] Confusing Subset with Subspace
> Not every subset is a subspace. Must verify closure under addition and scalar multiplication.

> [!warning] Assuming Union is a Subspace
> $W_1 \cup W_2$ is generally NOT a subspace (unless one contains the other). Use sum instead: $W_1 + W_2$.

> [!warning] Finite vs. Infinite Dimensions
> Some vector spaces are infinite-dimensional (functions, sequences). Don't assume everything behaves like $\mathbb{R}^n$!

- Show that the set of all $2 \times 2$ symmetric matrices is a vector space
- Is the set of polynomials of degree EXACTLY 3 a vector space? Why or why not?
- Verify that solutions to $y'' + y = 0$ form a vector space

Identify subspaces:

- Which of these are subspaces of $\mathbb{R}^3$?
  - $\{(x,y,z) : x + y + z = 0\}$
  - $\{(x,y,z) : x = 1\}$
  - $\{(x,y,z) : xy = 0\}$
- Show that the intersection of any collection of subspaces is a subspace
- Find a counterexample showing union of subspaces need not be a subspace

Conceptual:

- Why do we need all eight axioms? Can some be derived from others?
- Explain why the zero vector must be unique
- What's the difference between a vector space and a subspace?
