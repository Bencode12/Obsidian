## **Lesson 9.1.3 — Linear Transformations**

Linear transformations are the **morphisms** of linear algebra: structure-preserving maps between vector spaces. Everything that follows—matrices, eigenvalues, diagonalization, spectral theory—rests on this concept.

---

## **9.1.3.1 Definition**

A function  
[  
T: V \rightarrow W  
]  
between vector spaces over the same field is a **linear transformation** if, for all (u,v \in V) and scalars (a):

1. **Additivity:**  
    [  
    T(u+v) = T(u) + T(v)  
    ]
    
2. **Homogeneity:**  
    [  
    T(av) = aT(v)  
    ]
    

Equivalently:  
[  
T(a u + b v) = aT(u) + bT(v)  
]

---

## **9.1.3.2 Immediate Consequences**

For any linear transformation (T):

- (T(0) = 0)
    
- (T(-v) = -T(v))
    
- (T\left(\sum a_i v_i\right) = \sum a_i T(v_i))
    

Linearity implies **complete predictability** once the map is known on a basis.

---

## **9.1.3.3 Examples**

1. **Matrix transformation**  
    [  
    T(x) = Ax  
    ]  
    where (A) is an (m \times n) matrix
    
2. **Differentiation operator**  
    [  
    T(f) = f'  
    \quad \text{on } C^1([a,b])  
    ]
    
3. **Integral operator**  
    [  
    T(f)(x) = \int_a^x f(t),dt  
    ]
    
4. **Projection**  
    [  
    T(x,y,z) = (x,y,0)  
    ]
    
5. **Rotation in (\mathbb{R}^2)**  
    [  
    T(x,y) = (x\cos\theta - y\sin\theta, x\sin\theta + y\cos\theta)  
    ]
    

---

## **9.1.3.4 Kernel and Image**

### **Kernel (Null Space)**

[  
\ker(T) = { v \in V \mid T(v) = 0 }  
]

- Measures **loss of information**
    
- Always a subspace of (V)
    

### **Image (Range)**

[  
\text{Im}(T) = { T(v) \mid v \in V }  
]

- Measures **what is reachable**
    
- Always a subspace of (W)
    

---

## **9.1.3.5 Injectivity and Surjectivity**

- **Injective (one-to-one):**  
    [  
    T(u)=T(v) \Rightarrow u=v  
    \quad \Leftrightarrow \quad \ker(T)={0}  
    ]
    
- **Surjective (onto):**  
    [  
    \text{Im}(T)=W  
    ]
    
- **Isomorphism:** injective + surjective  
    (vector spaces are structurally identical)
    

---

## **9.1.3.6 Rank–Nullity Theorem**

For finite-dimensional (V):

[  
\dim(V) = \dim(\ker T) + \dim(\text{Im } T)  
]

- **Nullity** = dimension of kernel
    
- **Rank** = dimension of image
    

This theorem governs **solvability of linear systems** and underlies modern linear algebra.

---

## **9.1.3.7 Matrix Representation**

Given bases (B) for (V) and (C) for (W), there exists a unique matrix ([T]_{C \leftarrow B}) such that:

[  
[T(v)]_C = [T]_{C \leftarrow B} [v]_B  
]

- Changing bases changes the matrix, **not** the transformation.
    
- Linear transformations are **coordinate-free** objects.
