## **Lesson 9.1.2 — Linear Independence, Span, and Basis**

This lesson introduces the **structural core of linear algebra**: how vectors generate spaces, when information is redundant, and how dimension is defined.

---

## **9.1.2.1 Linear Combinations**

Given vectors (v_1, v_2, \dots, v_k \in V), a **linear combination** is:

[  
a_1 v_1 + a_2 v_2 + \cdots + a_k v_k  
\quad \text{where } a_i \in F  
]

- Linear combinations describe **everything you can build** from given vectors.
    

---

## **9.1.2.2 Span**

The **span** of vectors ({v_1, \dots, v_k}) is the set of all linear combinations:

[  
\text{span}{v_1,\dots,v_k} = \left{ \sum_{i=1}^k a_i v_i \mid a_i \in F \right}  
]

- Span is always a **subspace**.
    
- If the span equals the whole space, the vectors **generate** the space.
    

**Example:**  
[  
\text{span}{(1,0),(0,1)} = \mathbb{R}^2  
]

---

## **9.1.2.3 Linear Independence**

Vectors (v_1,\dots,v_k) are **linearly independent** if:

[  
a_1 v_1 + \cdots + a_k v_k = 0  
\quad \Rightarrow \quad  
a_1 = \cdots = a_k = 0  
]

- No vector can be written as a combination of the others.
    
- If **any nontrivial solution exists**, the set is **dependent**.
    

**Geometric meaning:**

- Independent → new direction of information
    
- Dependent → redundancy
    

---

## **9.1.2.4 Basis**

A **basis** of a vector space (V) is a set of vectors that is:

1. **Linearly independent**
    
2. **Spans (V)**
    

Every vector in (V) can be written **uniquely** as a linear combination of basis vectors.

**Example:**

- Standard basis of (\mathbb{R}^n):  
    [  
    {e_1=(1,0,\dots,0), \dots, e_n=(0,\dots,1)}  
    ]
    

---

## **9.1.2.5 Dimension**

The **dimension** of a vector space is the **number of vectors in any basis**.

- All bases of a finite-dimensional space have the same size.
    
- Dimension measures **degrees of freedom**.
    

**Examples:**

- (\dim(\mathbb{R}^3) = 3)
    
- (\dim(P_2) = 3)
    
- (\dim({0}) = 0)
    

---

## **9.1.2.6 Key Theorems**

1. Any linearly independent set can be extended to a basis.
    
2. Any spanning set can be reduced to a basis.
    
3. In a finite-dimensional space:
    
    - Independent set size ≤ dimension
        
    - Spanning set size ≥ dimension
        

These results underpin **rank, invertibility, and matrix theory**.
