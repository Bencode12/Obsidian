### **2.8.1 What is a table?**

A **table** organizes data in rows and columns for easy reading.

Example:

|Day|Sales (units)|
|---|---|
|Mon|10|
|Tue|15|
|Wed|12|

- Rows → items/observations
- Columns → variables/measurements

### **2.8.2 What is a graph?**

A **graph** visually represents data from a table.

- **Bar graph** → compares quantities
- **Line graph** → shows trends over time
- **Pie chart** → shows parts of a whole

### **2.8.3 Coordinate plane**

- X-axis → horizontal
- Y-axis → vertical
- Each point: (x, y)

Example:

- (3, 5) → move 3 units right, 5 units up

### **2.8.4 Plotting points**

1. Read the x-coordinate → move right/left
2. Read the y-coordinate → move up/down
3. Mark the point

### **2.8.5 Reading graphs**

- Identify axes and scale
- Read values accurately
- Interpret trends: increase, decrease, constant
