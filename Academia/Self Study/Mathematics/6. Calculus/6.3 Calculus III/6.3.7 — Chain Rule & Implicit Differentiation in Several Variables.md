## Chain Rule (Two Variables)

If $z = f(x,y)$, and $x$ and $y$ depend on $t$:

$$x = x(t), \quad y = y(t)$$

Then derivative of $z$ with respect to $t$ is:

$$\frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt} = f_x\frac{dx}{dt} + f_y\frac{dy}{dt}$$

- Captures **total rate of change** along the path
- Each term represents contribution from one variable

### Example

If $z = x^2y$ where $x = t^2$ and $y = e^t$:

1. Compute partials: $\frac{\partial z}{\partial x} = 2xy$, $\frac{\partial z}{\partial y} = x^2$
2. Compute derivatives: $\frac{dx}{dt} = 2t$, $\frac{dy}{dt} = e^t$
3. Apply chain rule: $\frac{dz}{dt} = 2xy(2t) + x^2(e^t) = 4t^3e^t + t^4e^t$

## Chain Rule (Multiple Variables)

If $z = f(x,y)$ with $x = x(s,t)$, $y = y(s,t)$:

$$\frac{\partial z}{\partial s} = f_x\frac{\partial x}{\partial s} + f_y\frac{\partial y}{\partial s}, \quad \frac{\partial z}{\partial t} = f_x\frac{\partial x}{\partial t} + f_y\frac{\partial y}{\partial t}$$

### Tree Diagram Method

```
      z
     / \
    x   y
   / \ / \
  s  t s  t
```

To find $\frac{\partial z}{\partial s}$: follow all paths from $z$ to $s$ and sum products.

> [!tip] General Formula
> 
> For $w = f(x_1, \ldots, x_n)$ where each $x_i = x_i(t_1, \ldots, t_m)$:
> $$\frac{\partial w}{\partial t_j} = \sum_{i=1}^n \frac{\partial f}{\partial x_i}\frac{\partial x_i}{\partial t_j}$$

## Implicit Differentiation

For equation $F(x,y) = 0$ defining $y$ implicitly as function of $x$:

$$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

### Derivation

Differentiate $F(x, y(x)) = 0$ with respect to $x$:

$$F_x + F_y\frac{dy}{dx} = 0 \Rightarrow \frac{dy}{dx} = -\frac{F_x}{F_y}$$

### Example

Find $\frac{dy}{dx}$ for $x^2 + y^2 = 1$:

1. Write as $F(x,y) = x^2 + y^2 - 1 = 0$
2. Partials: $F_x = 2x$, $F_y = 2y$
3. Result: $\frac{dy}{dx} = -\frac{2x}{2y} = -\frac{x}{y}$

## Implicit Functions of Several Variables

For $F(x,y,z) = 0$ defining $z$ as function of $x,y$:

$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

### Example

Find $\frac{\partial z}{\partial x}$ for $x^2 + y^2 + z^2 = 1$:

1. $F(x,y,z) = x^2 + y^2 + z^2 - 1 = 0$
2. Partials: $F_x = 2x$, $F_z = 2z$
3. Result: $\frac{\partial z}{\partial x} = -\frac{2x}{2z} = -\frac{x}{z}$

## Practice Problems

1. If $z = x^2 + y^2$ where $x = \cos t$ and $y = \sin t$, find $\frac{dz}{dt}$.
2. Use the chain rule to find $\frac{\partial w}{\partial u}$ if $w = xy + z^2$, $x = u + v$, $y = u - v$, $z = uv$.
3. Find $\frac{dy}{dx}$ for $x^3 + y^3 = 6xy$ using implicit differentiation.
4. If $T(x,y) = x^2e^y$ where $x = r\cos\theta$ and $y = r\sin\theta$, find $\frac{\partial T}{\partial r}$.
5. Verify that the chain rule gives the same result as direct substitution for $z = xy$ with $x = t+1$, $y = t^2$.
6. Find $\frac{\partial z}{\partial x}$ and $\frac{\partial z}{\partial y}$ for $xe^z + ye^z = z$.
7. Use implicit differentiation to find the slope of the tangent line to $x^2y + y^3 = 4$ at $(1,1)$.
8. If $w = f(x,y,z)$ and $x = g(r,s)$, $y = h(r,s)$, $z = k(r,s)$, write the formula for $\frac{\partial w}{\partial s}$.
9. Show that for $F(x,y) = 0$, we have $\frac{dy}{dx} \cdot \frac{dx}{dy} = 1$ (when both exist).
10. Find all first partial derivatives of $z$ defined implicitly by $\sin(xyz) = x + 2y + 3z$.
