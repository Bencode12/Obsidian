## **7.1.5.1 Definition of Limit**

For a function $f(x,y)$, we say:

lim‚Ā°(x,y)‚Üí(a,b)f(x,y)=L$\lim_{(x,y)\to(a,b)} f(x,y)$ = L(x,y)‚Üí(a,b)limf(x,y)=L

if, for **all paths** approaching (a,b)(a,b)(a,b), the function values approach $L$.

- In other words, approaching from **any direction** must yield the same result.

## **7.1.5.3 Definition of Continuity**

A function $f(x,y)$ is **continuous at (a,b)(a,b)(a,b)** if:

lim‚Ā°(x,y)‚Üí(a,b)f(x,y)=f(a,b)$\lim_{(x,y)\to(a,b)} f(x,y)$ = f(a,b)(x,y)‚Üí(a,b)limf(x,y)=f(a,b)

- Requires the limit to exist and equal the function value
- Continuous functions include **polynomials, exponentials, sine, cosine** in any number of variables.
