## Chain Rule (Two Variables)

If $z = f(x,y)$, and $x$ and $y$ depend on $t$:

$$x = x(t), \quad y = y(t)$$

Then derivative of $z$ with respect to $t$ is:

$$\frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt} = f_x\frac{dx}{dt} + f_y\frac{dy}{dt}$$

- Captures **total rate of change** along the path.

> [!tip] Tree Diagram Method Draw a tree:
> - Top: $z$
> - Middle: $x$ and $y$ (both depend on $t$)
> - Multiply along each path and add
> - Result: $\frac{dz}{dt} = \frac{\partial z}{\partial x}\frac{dx}{dt} + \frac{\partial z}{\partial y}\frac{dy}{dt}$

## Chain Rule (Multiple Variables)

If $z = f(x,y)$ with $x = x(s,t)$, $y = y(s,t)$:

$$\frac{\partial z}{\partial s} = f_x\frac{\partial x}{\partial s} + f_y\frac{\partial y}{\partial s}, \quad \frac{\partial z}{\partial t} = f_x\frac{\partial x}{\partial t} + f_y\frac{\partial y}{\partial t}$$

> [!info] General Chain Rule Pattern For any number of intermediate variables:
> $$\frac{dz}{dt} = \sum_{i} \frac{\partial z}{\partial x_i}\frac{dx_i}{dt}$$
>
> Each path from $z$ to $t$ contributes one term!

## Implicit Differentiation

If $F(x,y) = 0$, then $y$ is a function of $x$ implicitly.

$$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

|Concept|Meaning|
|---|---|
|$F_x$|$\frac{\partial F}{\partial x}$|
|$F_y$|$\frac{\partial F}{\partial y}$|

> [!warning] Don't Forget the Negative Sign! The formula has a negative sign:
> $$\frac{dy}{dx} = -\frac{F_x}{F_y}$$
>
> **Example:** For $x^2 + y^2 = 1$:
> - $F(x,y) = x^2 + y^2 - 1$
> - $F_x = 2x$, $F_y = 2y$
> - $\frac{dy}{dx} = -\frac{2x}{2y} = -\frac{x}{y}$ ✓

---

## Total Differential

For $z = f(x,y)$:

$$dz = f_x dx + f_y dy$$

|Property|Description|
|---|---|
|**Purpose**|Linear approximation near a point|
|**Connection**|Directly related to chain rule|

## Practice Problems

1. If $z = x^2y + y^3$, $x = t^2$, $y = 2t$, find $\frac{dz}{dt}$.
2. Use the chain rule to find $\frac{\partial z}{\partial s}$ if $z = e^{xy}$, $x = s + t$, $y = st$.
3. Find $\frac{dy}{dx}$ for $x^3 + y^3 = 6xy$ using implicit differentiation.
4. Find the total differential of $f(x,y) = x^2y + \cos(xy)$.
