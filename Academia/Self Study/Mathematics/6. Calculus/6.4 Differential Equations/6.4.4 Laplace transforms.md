## Definition

**Laplace Transform** of function $f(t)$ (defined for $t \geq 0$):

$$\mathcal{L}\{f(t)\} = F(s) = \int_0^\infty e^{-st}f(t)\,dt$$

Transforms function from time domain to frequency domain.

### Existence

Transform exists if $f(t)$ is piecewise continuous and of exponential order:
$$|f(t)| \leq Me^{at} \text{ for } t \geq T$$

Then converges for $s > a$.

## Basic Transforms

|Function $f(t)$|Transform $F(s)$|Region|
|---|---|---|
|$1$|$\frac{1}{s}$|$s > 0$|
|$t^n$|$\frac{n!}{s^{n+1}}$|$s > 0$|
|$e^{at}$|$\frac{1}{s-a}$|$s > a$|
|$\sin(bt)$|$\frac{b}{s^2+b^2}$|$s > 0$|
|$\cos(bt)$|$\frac{s}{s^2+b^2}$|$s > 0$|
|$t^n e^{at}$|$\frac{n!}{(s-a)^{n+1}}$|$s > a$|
|$e^{at}\sin(bt)$|$\frac{b}{(s-a)^2+b^2}$|$s > a$|
|$e^{at}\cos(bt)$|$\frac{s-a}{(s-a)^2+b^2}$|$s > a$|

## Properties

### Linearity

$$\mathcal{L}\{af(t) + bg(t)\} = aF(s) + bG(s)$$

### First Derivative

$$\mathcal{L}\{f'(t)\} = sF(s) - f(0)$$

### Second Derivative

$$\mathcal{L}\{f''(t)\} = s^2F(s) - sf(0) - f'(0)$$

### $n$-th Derivative

$$\mathcal{L}\{f^{(n)}(t)\} = s^nF(s) - s^{n-1}f(0) - \cdots - f^{(n-1)}(0)$$

### First Shifting Theorem

$$\mathcal{L}\{e^{at}f(t)\} = F(s-a)$$

### Second Shifting Theorem (Time Delay)

$$\mathcal{L}\{u(t-a)f(t-a)\} = e^{-as}F(s)$$

where $u(t)$ is Heaviside step function.

### Multiplication by $t$

$$\mathcal{L}\{tf(t)\} = -F'(s)$$

### Convolution

$$\mathcal{L}\{(f*g)(t)\} = F(s)G(s)$$

where $(f*g)(t) = \int_0^t f(\tau)g(t-\tau)\,d\tau$.

## Inverse Laplace Transform

### Notation

$$f(t) = \mathcal{L}^{-1}\{F(s)\}$$

### Method: Partial Fractions

Decompose $F(s)$ into simpler fractions, then use table.

### Example 1: Simple Poles

Find $\mathcal{L}^{-1}\left\{\frac{3s+2}{s^2+3s+2}\right\}$:

Factor denominator: $s^2+3s+2 = (s+1)(s+2)$

Partial fractions: $\frac{3s+2}{(s+1)(s+2)} = \frac{A}{s+1} + \frac{B}{s+2}$

Solve: $A = -1$, $B = 4$

Inverse: $\mathcal{L}^{-1}\left\{-\frac{1}{s+1} + \frac{4}{s+2}\right\} = -e^{-t} + 4e^{-2t}$

### Example 2: Complex Poles

Find $\mathcal{L}^{-1}\left\{\frac{s}{s^2+4s+13}\right\}$:

Complete square: $s^2+4s+13 = (s+2)^2+9$

Rewrite: $\frac{s}{(s+2)^2+9} = \frac{s+2-2}{(s+2)^2+9} = \frac{s+2}{(s+2)^2+9} - \frac{2}{(s+2)^2+9}$

Inverse: $e^{-2t}\cos(3t) - \frac{2}{3}e^{-2t}\sin(3t)$

## Solving ODEs with Laplace Transforms

### General Procedure

1. Take Laplace transform of both sides
2. Use derivative formulas (incorporate initial conditions)
3. Solve algebraically for $Y(s)$
4. Find inverse transform $y(t) = \mathcal{L}^{-1}\{Y(s)\}$

### Example

Solve $y'' + 3y' + 2y = 0$ with $y(0) = 1$, $y'(0) = 0$:

Take Laplace: $[s^2Y - sy(0) - y'(0)] + 3[sY - y(0)] + 2Y = 0$

Substitute ICs: $(s^2Y - s) + 3(sY - 1) + 2Y = 0$

Simplify: $(s^2 + 3s + 2)Y = s + 3$

Solve: $Y = \frac{s+3}{s^2+3s+2} = \frac{s+3}{(s+1)(s+2)}$

Partial fractions: $Y = \frac{2}{s+1} - \frac{1}{s+2}$

Inverse: $y(t) = 2e^{-t} - e^{-2t}$

## Step Functions and Piecewise Functions

### Heaviside Function

$$u(t-a) = \begin{cases} 0 & t < a \\ 1 & t \geq a \end{cases}$$

Transform: $\mathcal{L}\{u(t-a)\} = \frac{e^{-as}}{s}$

### Representing Piecewise Functions

Function that "turns on" at $t=a$:
$$f(t) = u(t-a)g(t)$$

Function that changes at $t=a$:
$$f(t) = g_1(t) + u(t-a)[g_2(t) - g_1(t)]$$

### Example

Find $\mathcal{L}\{f(t)\}$ where $f(t) = \begin{cases} t & 0 \leq t < 2 \\ 0 & t \geq 2 \end{cases}$:

Write: $f(t) = t - u(t-2)t = t - u(t-2)[(t-2)+2]$

Transform: $F(s) = \frac{1}{s^2} - e^{-2s}\left[\frac{1}{s^2} + \frac{2}{s}\right]$

## Impulse Functions

### Dirac Delta Function

$\delta(t-a)$: "Infinite spike" at $t=a$ with unit area.

Property: $\int_{-\infty}^\infty \delta(t-a)f(t)\,dt = f(a)$

Transform: $\mathcal{L}\{\delta(t-a)\} = e^{-as}$

### Application: Impulsive Forcing

Equation with impulse at $t=a$:
$$y'' + py' + qy = \delta(t-a)$$

Models sudden impact or shock.

## Convolution

### Definition

$$(f*g)(t) = \int_0^t f(\tau)g(t-\tau)\,d\tau$$

### Convolution Theorem

$$\mathcal{L}\{f*g\} = F(s)G(s)$$

Therefore: $\mathcal{L}^{-1}\{F(s)G(s)\} = f*g$

### Example

Find $\mathcal{L}^{-1}\left\{\frac{1}{s(s+1)}\right\}$:

Recognize: $\frac{1}{s(s+1)} = \frac{1}{s} \cdot \frac{1}{s+1}$

So: $\mathcal{L}^{-1}\left\{\frac{1}{s(s+1)}\right\} = 1 * e^{-t} = \int_0^t e^{-\tau}\,d\tau = 1 - e^{-t}$

## Applications

### Electrical Circuits

RLC circuit: $L\frac{d^2q}{dt^2} + R\frac{dq}{dt} + \frac{q}{C} = E(t)$

Laplace transform converts to algebraic equation.

### Mechanical Vibrations

Mass-spring-damper: $mx'' + cx' + kx = F(t)$

Handle discontinuous forcing functions easily.

### Control Systems

Transfer function: $G(s) = \frac{Y(s)}{U(s)}$

Analyze system stability and response.

## Common Mistakes

> [!warning] Forgetting Initial Conditions
> Derivative formulas include $f(0)$, $f'(0)$, etc. Don't omit them!

> [!warning] Incorrect Partial Fraction Decomposition
> Check your decomposition by combining fractions back together.

> [!warning] Confusing Time and Frequency Shifts
> First shifting: multiply by $e^{at}$ in time. Second shifting: delay in time.
