## Standard Form

**Second-order linear ODE**:
$$a(x)y'' + b(x)y' + c(x)y = f(x)$$

If $a, b, c$ are constants: **constant coefficient equation**.

If $f(x) = 0$: **homogeneous equation**.

---

## Homogeneous Equations with Constant Coefficients

### Standard Form

$$ay'' + by' + cy = 0$$

### Characteristic Equation

Assume solution $y = e^{rx}$:
$$ar^2 + br + c = 0$$

Solve quadratic for $r$.

### Three Cases

|Case|Roots|General Solution|
|---|---|---|
|Distinct real|$r_1 \neq r_2$|$y = C_1e^{r_1x} + C_2e^{r_2x}$|
|Repeated real|$r_1 = r_2 = r$|$y = (C_1 + C_2x)e^{rx}$|
|Complex conjugate|$r = \alpha \pm i\beta$|$y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$|

```desmos-graph
y = e^(-0.5*x)*cos(3*x)
y = e^(-0.5*x)*sin(3*x)
```

> [!example] Visual Interpretation
> Damped oscillations from complex roots with negative real part.

### Example 1: Distinct Real Roots

Solve $y'' - 3y' + 2y = 0$:

Characteristic: $r^2 - 3r + 2 = 0$

Factor: $(r-1)(r-2) = 0$

Roots: $r_1 = 1$, $r_2 = 2$

Solution: $y = C_1e^x + C_2e^{2x}$

### Example 2: Complex Roots

Solve $y'' + 4y = 0$:

Characteristic: $r^2 + 4 = 0$

Roots: $r = \pm 2i$

Solution: $y = C_1\cos(2x) + C_2\sin(2x)$

## Nonhomogeneous Equations

### General Solution Structure

$$y = y_h + y_p$$

where:
- $y_h$: General solution to homogeneous equation
- $y_p$: Particular solution to nonhomogeneous equation

### Method of Undetermined Coefficients

Guess form of $y_p$ based on $f(x)$:

|Form of $f(x)$|Trial $y_p$|
|---|---|
|$P_n(x)$ (polynomial)|$A_nx^n + \cdots + A_0$|
|$e^{ax}$|$Ae^{ax}$|
|$\cos(bx)$ or $\sin(bx)$|$A\cos(bx) + B\sin(bx)$|
|$e^{ax}\cos(bx)$|$e^{ax}(A\cos(bx) + B\sin(bx))$|

If trial solution overlaps with $y_h$, multiply by $x$ (or $x^2$ if needed).

### Example

Solve $y'' - y = e^x$:

Homogeneous: $y_h = C_1e^x + C_2e^{-x}$

For $f(x) = e^x$, normally try $Ae^x$, but this overlaps with $y_h$.

Modified trial: $y_p = Axe^x$

Compute: $y_p' = Ae^x + Axe^x$, $y_p'' = 2Ae^x + Axe^x$

Substitute: $(2Ae^x + Axe^x) - Axe^x = e^x$

Simplify: $2Ae^x = e^x$, so $A = 1/2$

Solution: $y = C_1e^x + C_2e^{-x} + \frac{1}{2}xe^x$

## Variation of Parameters

### General Method

For $y'' + p(x)y' + q(x)y = f(x)$:

If $y_1, y_2$ are solutions to homogeneous equation, try:
$$y_p = u_1(x)y_1 + u_2(x)y_2$$

where $u_1, u_2$ satisfy:
$$u_1'y_1 + u_2'y_2 = 0$$
$$u_1'y_1' + u_2'y_2' = f(x)$$

### Wronskian

$$W(y_1, y_2) = \begin{vmatrix} y_1 & y_2 \\ y_1' & y_2' \end{vmatrix} = y_1y_2' - y_2y_1'$$

Solutions:
$$u_1' = -\frac{y_2f}{W}, \quad u_2' = \frac{y_1f}{W}$$

Integrate to find $u_1, u_2$.

## Cauchy-Euler Equations

### Standard Form

$$ax^2y'' + bxy' + cy = 0$$

Variable coefficients, but special structure.

### Solution Method

Try $y = x^r$:
$$ar(r-1) + br + c = 0$$

Quadratic in $r$. Same three cases as constant coefficient.

### Solutions

|Case|Roots|General Solution|
|---|---|---|
|Distinct real|$r_1 \neq r_2$|$y = C_1x^{r_1} + C_2x^{r_2}$|
|Repeated|$r_1 = r_2 = r$|$y = (C_1 + C_2\ln x)x^r$|
|Complex|$r = \alpha \pm i\beta$|$y = x^\alpha[C_1\cos(\beta\ln x) + C_2\sin(\beta\ln x)]$|

## Initial and Boundary Value Problems

### Initial Value Problem (IVP)

$$y'' + py' + qy = f, \quad y(x_0) = y_0, \quad y'(x_0) = y_0'$$

Unique solution exists for linear equations.

### Boundary Value Problem (BVP)

$$y'' + py' + qy = f, \quad y(a) = \alpha, \quad y(b) = \beta$$

May have no solution, unique solution, or infinitely many.

### Example: IVP

Solve $y'' - 4y = 0$ with $y(0) = 1$, $y'(0) = 2$:

General: $y = C_1e^{2x} + C_2e^{-2x}$

Derivative: $y' = 2C_1e^{2x} - 2C_2e^{-2x}$

Initial conditions:
- $y(0) = C_1 + C_2 = 1$
- $y'(0) = 2C_1 - 2C_2 = 2$

Solve: $C_1 = 1$, $C_2 = 0$

Solution: $y = e^{2x}$

## Applications

### Simple Harmonic Oscillator

$$m\frac{d^2x}{dt^2} + kx = 0$$

Solution: $x(t) = A\cos(\omega t) + B\sin(\omega t)$ where $\omega = \sqrt{k/m}$.

### Damped Oscillator

$$mx'' + cx' + kx = 0$$

Behavior depends on damping ratio $\zeta = \frac{c}{2\sqrt{mk}}$:
- $\zeta < 1$: Underdamped (oscillates)
- $\zeta = 1$: Critically damped
- $\zeta > 1$: Overdamped

### Forced Oscillations

$$mx'' + cx' + kx = F_0\cos(\omega t)$$

Exhibits resonance when $\omega \approx \sqrt{k/m}$.

## Common Mistakes

> [!warning] Forgetting to Check for Overlap
> When using undetermined coefficients, check if trial solution is in $y_h$. If yes, multiply by $x$.

> [!warning] Confusing IVP and BVP
> IVP always has unique solution. BVP may not!

> [!warning] Sign Errors in Characteristic Equation
> Double-check: $ay'' + by' + cy = 0$ gives $ar^2 + br + c = 0$.
