## Review: Martingale Definition

### Discrete Time

Process $\{M_n\}$ is **martingale** if:
1. $\mathbb{E}[|M_n|] < \infty$ (integrable)
2. $\mathbb{E}[M_{n+1} | \mathcal{F}_n] = M_n$ (fair game)

### Continuous Time

Process $\{M_t\}_{t \geq 0}$ is **martingale** if:
1. $\mathbb{E}[|M_t|] < \infty$ for all $t$
2. $\mathbb{E}[M_t | \mathcal{F}_s] = M_s$ for all $s < t$

Interpretation: Best prediction of future value is current value.

## Martingales from Itô Integrals

### Fundamental Result

If $H_t$ is adapted with $\mathbb{E}\int_0^T H_t^2\,dt < \infty$, then:

$$M_t = \int_0^t H_s\,dW_s$$

is a martingale.

### Proof Sketch

For $s < t$:
$$\mathbb{E}[M_t | \mathcal{F}_s] = M_s + \mathbb{E}\left[\int_s^t H_u\,dW_u \bigg| \mathcal{F}_s\right] = M_s + 0 = M_s$$

Future increment has conditional expectation zero.

## Examples of Martingales

### Example 1: Brownian Motion

$W_t$ is a martingale:
$$\mathbb{E}[W_t | \mathcal{F}_s] = W_s + \mathbb{E}[W_t - W_s | \mathcal{F}_s] = W_s + 0 = W_s$$

Independent increments with mean zero.

### Example 2: Exponential Martingale

$$M_t = \exp\left(W_t - \frac{t}{2}\right)$$

is a martingale.

Verify using Itô's formula:
$dM_t = M_t\,dW_t$ (no drift term!)

### Example 3: Quadratic Variation Martingale

$$M_t = W_t^2 - t$$

is a martingale.

From Itô: $d(W_t^2) = 2W_t\,dW_t + dt$, so $d(W_t^2 - t) = 2W_t\,dW_t$.

## Local Martingales

### Definition

Process $M_t$ is **local martingale** if exists stopping times $\tau_n \uparrow \infty$ such that $M_{t \wedge \tau_n}$ is martingale for each $n$.

Every martingale is local martingale, but not conversely!

### Example: Strict Local Martingale

$$M_t = \exp\left(W_t - \frac{t}{2}\right)$$

is true martingale, but some transformations yield strict local martingales.

### Importance

In finance, asset prices should be true martingales under risk-neutral measure, not just local.

## Martingale Representation Theorem

### Theorem

Let $M_t$ be martingale with respect to Brownian filtration $\mathcal{F}_t^W$. Then exists adapted process $H_t$ such that:

$$M_t = M_0 + \int_0^t H_s\,dW_s$$

### Significance

- Every martingale can be represented as Itô integral
- Foundation of hedging in mathematical finance
- Shows Brownian filtration is "rich enough"

## Girsanov's Theorem

### Motivation

Change probability measure to eliminate drift from SDE.

### Theorem

Let $W_t$ be Brownian motion under $\mathbb{P}$. Define new measure $\mathbb{Q}$ via Radon-Nikodym derivative:

$$\frac{d\mathbb{Q}}{d\mathbb{P}} = \exp\left(-\int_0^T \theta_s\,dW_s - \frac{1}{2}\int_0^T \theta_s^2\,ds\right)$$

Then under $\mathbb{Q}$:
$$\tilde{W}_t = W_t + \int_0^t \theta_s\,ds$$

is Brownian motion.

### Application: Risk-Neutral Pricing

Under physical measure $\mathbb{P}$:
$$dS_t = \mu S_t\,dt + \sigma S_t\,dW_t^{\mathbb{P}}$$

Choose $\theta = \frac{\mu - r}{\sigma}$ to get risk-neutral measure $\mathbb{Q}$:
$$dS_t = rS_t\,dt + \sigma S_t\,dW_t^{\mathbb{Q}}$$

Drift becomes risk-free rate!

## Optional Stopping Theorem

### Theorem

If $M_t$ is martingale and $\tau$ is bounded stopping time, then:
$$\mathbb{E}[M_\tau] = \mathbb{E}[M_0]$$

### Conditions

Need either:
1. $\tau$ bounded, or
2. $M_t$ uniformly integrable and $\mathbb{E}[\tau] < \infty$, or
3. $|M_{t \wedge \tau}| \leq K$ for some constant $K$

### Example: Gambler's Ruin

Random walk starting at $a$, stop at 0 or $N$. Probability of hitting $N$ before 0 is $a/N$.

## Doob's Inequalities

### Maximal Inequality

For nonnegative submartingale $X_t$:
$$\mathbb{P}\left(\sup_{0 \leq s \leq t} X_s \geq \lambda\right) \leq \frac{\mathbb{E}[X_t]}{\lambda}$$

### $L^p$ Inequality

For martingale $M_t$ and $p > 1$:
$$\mathbb{E}\left[\sup_{0 \leq s \leq t} |M_s|^p\right] \leq \left(\frac{p}{p-1}\right)^p \mathbb{E}[|M_t|^p]$$

Control maximum of martingale by terminal value.

## Applications

### Finance: No-Arbitrage Pricing

Under risk-neutral measure $\mathbb{Q}$, discounted asset prices are martingales:
$$e^{-rt}S_t \text{ is } \mathbb{Q}\text{-martingale}$$

Option price: $V_0 = e^{-rT}\mathbb{E}^{\mathbb{Q}}[V_T]$

### Statistics: Sequential Testing

Use martingale properties to design sequential hypothesis tests.

### Physics: Fair Games

Martingales model fair games, efficient markets, equilibrium systems.

## Common Mistakes

> [!warning] Assuming All Local Martingales Are Martingales
> Check uniform integrability or other conditions. Not automatic!

> [!warning] Misapplying Optional Stopping
> Must verify conditions (bounded stopping time, UI, etc.). Can fail otherwise.

> [!warning] Confusing Physical and Risk-Neutral Measures
> Drift changes under measure change. Volatility stays same.
