## Motivation

Decide optimal time to take action based on evolving information:
- When to exercise American option?
- When to sell an asset?
- When to stop a clinical trial?

**Optimal stopping** problems!

---

## Problem Formulation

### Setup

Given:
- Stochastic process $X_t$ (state variable)
- Reward function $g(x)$ (payoff if stop at state $x$)
- Discount rate $r \geq 0$

### Objective

Find stopping time $\tau^*$ maximizing expected discounted reward:

$$V(x) = \sup_{\tau} \mathbb{E}_x[e^{-r\tau}g(X_\tau)]$$

where supremum over all stopping times $\tau$.

### Value Function

$V(x)$: Maximum expected reward starting from state $x$.

$\tau^*$: Optimal stopping rule.

---

## Free Boundary Problems

### Continuation and Stopping Regions

**Continuation region** $\mathcal{C}$: Keep waiting
$$\mathcal{C} = \{x : V(x) > g(x)\}$$

**Stopping region** $\mathcal{S}$: Stop now
$$\mathcal{S} = \{x : V(x) = g(x)\}$$

**Free boundary**: Boundary between $\mathcal{C}$ and $\mathcal{S}$.

### Variational Inequality

Value function satisfies:

$$\max\left\{\mathcal{L}V - rV, \, g - V\right\} = 0$$

where $\mathcal{L}$ is infinitesimal generator of $X_t$.

Equivalently:
1. $\mathcal{L}V - rV \leq 0$ everywhere
2. $V \geq g$ everywhere
3. $(\mathcal{L}V - rV)(V - g) = 0$ (complementary slackness)

---

## Smooth Fit Principle

### Condition

At optimal boundary $x^*$:
$$V(x^*) = g(x^*) \quad \text{(value matching)}$$
$$V'(x^*) = g'(x^*) \quad \text{(smooth fit)}$$

Sometimes also:
$$V''(x^*) = g''(x^*) \quad \text{(super contact)}$$

### Intuition

Value function should join payoff smoothly at boundary. Kinks would suggest suboptimal boundary.

### Verification

Must verify smooth fit holds (not always true!).

## Perpetual American Options

### Call Option

Payoff: $g(S) = (S - K)^+$

Stock follows GBM: $dS_t = rS_t\,dt + \sigma S_t\,dW_t$

### Solution Method

Guess solution form in continuation region $(0, S^*)$:
$$V(S) = AS^\beta$$

where $\beta$ solves characteristic equation:
$$\frac{1}{2}\sigma^2\beta(\beta-1) + r\beta - r = 0$$

Roots: $\beta_1 > 1$ and $\beta_2 < 0$.

For call: Use $\beta_1$ (solution must be bounded as $S \to 0$).

### Boundary Conditions

At optimal exercise boundary $S^*$:
1. Value matching: $A(S^*)^{\beta_1} = S^* - K$
2. Smooth fit: $A\beta_1(S^*)^{\beta_1-1} = 1$

Solve:
$$S^* = \frac{\beta_1}{\beta_1 - 1}K$$
$$A = \frac{S^* - K}{(S^*)^{\beta_1}}$$

### Result

Optimal strategy: Exercise when $S_t \geq S^*$.

Value function:
$$V(S) = \begin{cases} A S^{\beta_1} & S < S^* \\ S - K & S \geq S^* \end{cases}$$

---

## Perpetual American Put

### Solution

Similar method with boundary at lower end.

Optimal exercise boundary:
$$S^* = \frac{\beta_2}{\beta_2 - 1}K$$

where $\beta_2 < 0$ is negative root.

Value function:
$$V(S) = \begin{cases} K - S & S \leq S^* \\ B S^{\beta_2} & S > S^* \end{cases}$$

### Key Difference from Call

- **Put**: Always optimal to exercise if $S$ low enough
- **Call**: Never optimal to exercise early if no dividends! (But perpetual case different)

## Finite Horizon Problems

### American Put with Maturity $T$

More complex! Free boundary $S^*(t)$ depends on time.

PDE in continuation region:
$$\frac{\partial V}{\partial t} + \frac{1}{2}\sigma^2S^2\frac{\partial^2 V}{\partial S^2} + rS\frac{\partial V}{\partial S} - rV = 0$$

Boundary conditions:
- $V(S, T) = (K - S)^+$ (terminal payoff)
- $V(S^*(t), t) = K - S^*(t)$ (value matching)
- $\frac{\partial V}{\partial S}(S^*(t), t) = -1$ (smooth fit)

### No Closed-Form Solution

Must use:
- Numerical PDE methods (finite difference)
- Monte Carlo with regression (Longstaff-Schwartz)
- Integral equation methods

---

## Dynamic Programming Approach

### Bellman Equation

Discrete time version:
$$V(x) = \max\left\{g(x), \, e^{-r\Delta t}\mathbb{E}[V(X_{t+\Delta t}) | X_t = x]\right\}$$

Either stop now or continue one step.

### Continuous Time Limit

Leads to variational inequality mentioned earlier.

---

## Methods of Solution

### 1. Analytical Methods

- Guess solution form, verify with smooth fit
- Works for perpetual options, simple processes
- Limited applicability

### 2. PDE Methods

Solve free boundary problem numerically:
- Finite difference schemes
- Penalty methods
- Front-fixing transformations

### 3. Monte Carlo Methods

**Longstaff-Schwartz algorithm**:
1. Simulate many paths
2. At each timestep, regress continuation value on basis functions
3. Compare immediate exercise vs estimated continuation value
4. Work backward from maturity

Most flexible, handles high dimensions.

### 4. Variational Methods

Formulate as optimization problem:
$$V = \inf\{w : w \geq g, \, \mathcal{L}w - rw \leq 0\}$$

Use numerical optimization.

## Applications

### American Options

Most common application. Early exercise feature valuable for:
- Deep ITM puts
- Stocks with dividends
- Currency options

### Real Options

Investment decisions under uncertainty:
- When to invest in project?
- When to expand operations?
- When to abandon unprofitable venture?

### Sequential Hypothesis Testing

Statistical decision making:
- When to stop collecting data?
- Balance sampling cost vs decision accuracy

### Quickest Detection

Detect change in system dynamics:
- Quality control
- Financial regime changes
- Security breach detection

---

## Multiple Stopping Problems

### Swing Options

Right to exercise multiple times:
$$V = \sup_{\tau_1 < \cdots < \tau_n} \mathbb{E}\left[\sum_{i=1}^n e^{-r\tau_i}g(X_{\tau_i})\right]$$

Applications:
- Energy contracts (multiple deliveries)
- Employee stock options (multiple grants)

### Solution Approach

Dynamic programming with state variable = number of exercises remaining.

---

## Common Mistakes

> [!warning] Forgetting Discount Factor
> Must include $e^{-r\tau}$ in objective. Future rewards worth less!

> [!warning] Assuming Smooth Fit Always Holds
> Smooth fit can fail for discontinuous payoffs or certain processes. Verify!

> [!warning] Confusing American and European Options
> American allows early exercise. European only at maturity. American $\geq$ European.
