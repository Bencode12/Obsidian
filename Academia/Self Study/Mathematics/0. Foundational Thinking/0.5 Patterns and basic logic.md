## What is a Pattern?

A pattern is a **repeating or predictable rule** in numbers. Patterns are the foundation of algebra.

|Sequence|Rule|
|---|---|
|$2, 4, 6, 8, \ldots$|Add $2$ each time|
|$1, 4, 9, 16, \ldots$|Square numbers ($n^2$)|
|$5, 10, 20, 40, \ldots$|Multiply by $2$ each time|

---

## Finding the Rule

To identify a pattern, ask:

- What is **changing** between terms?
- Is it adding, subtracting, multiplying, dividing — or something else?

> [!tip] Check the gap between consecutive terms first Subtract any term from the next. If the difference is constant → **add/subtract rule**. If the ratio is constant → **multiply/divide rule**.

---

## Growing vs Repeating Patterns

|Type|Example|Behaviour|
|---|---|---|
|**Repeating**|$1, 2, 1, 2, 1, 2, \ldots$|Cycles through a fixed block|
|**Growing**|$3, 6, 9, 12, \ldots$|Moves forward by a rule — see [[Sequences]]|

## Introduction to Logic

Logic means **reasoning correctly** — drawing valid conclusions from true statements.

|Form|Meaning|
|---|---|
|If $A$ then $B$|When $A$ is true, $B$ must be true|
|If not $A$ then not $B$|When $A$ is false, $B$ is also false|

> [!example] _If a number is even, it is divisible by 2._ $8$ is even → $8$ is divisible by $2$ ✓

---

## True or False

Every mathematical statement is either **true** or **false** — never "maybe."

| Statement               | Value |
| ----------------------- | ----- |
| $4$ is even             | True  |
| $7$ is divisible by $2$ | False |
| $3 + 3 = 6$             | True  |
| $5 > 9$                 | False |
