### **0.5.1 What is a pattern?**

A pattern is a **repeating or predictable rule** in numbers.

Examples:

- 2, 4, 6, 8 → add 2 each time
- 1, 4, 9, 16 → square numbers

Patterns are the foundation of algebra.

### **0.5.2 Finding the rule**

To identify a pattern, ask:

- What is changing?
- Is it adding, subtracting, multiplying, or something else?

Example:

- 5, 10, 20, 40  
    Rule: multiply by 2

### **0.5.3 Growing vs repeating patterns**

- **Repeating pattern**: 1, 2, 1, 2, 1, 2
- **Growing pattern**: 3, 6, 9, 12

Growing patterns move forward with a rule.

### **0.5.4 Introduction to logic**

Logic means reasoning correctly.

Basic logical ideas:

- If A happens, then B happens
- If A does not happen, B does not happen

Example:

- If a number is even, it is divisible by 2

### **0.5.5 True or false**

Statements in math are either **true** or **false**, never “maybe.”

Examples:

- 4 is even → true
- 7 is divisible by 2 → false
