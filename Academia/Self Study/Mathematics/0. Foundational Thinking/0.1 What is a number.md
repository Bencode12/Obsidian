## The Idea of Quantity

A number represents **how much** or **how many**. Numbers are symbolic labels for quantities.

|Number|Meaning|
|---|---|
|1|a single item|
|2|a pair|
|10|ten items|

---

## The Three Families of Basic Numbers

|Family|Set|Used for|
|---|---|---|
|**Natural numbers**|$1, 2, 3, 4, \ldots$|Counting|
|**Whole numbers**|$0, 1, 2, 3, \ldots$|Counting including zero|
|**Integers**|$\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots$|Counting with negatives|

Each family **contains** the one above it — integers include whole numbers, whole numbers include natural numbers.

## Place Value

Every digit in a number has a **position** that changes its value. In $4{,}382$:

|Digit|Position|Value|
|---|---|---|
|4|Thousands|$4000$|
|3|Hundreds|$300$|
|8|Tens|$80$|
|2|Ones|$2$|

$$4{,}382 = 4000 + 300 + 80 + 2$$

This positional structure is the foundation of our number system — the same digit means something different depending on where it sits.
