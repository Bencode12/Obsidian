### **1.1 The idea of “quantity”**

A number represents _how much_ or _how many_.

Examples:

- 1 means a single item.
- 2 means a pair.
- 10 means ten items.

Numbers are symbolic labels for quantities.

### **1.2 The three families of basic numbers**

1. **Natural numbers**: 1, 2, 3, 4…  
    Used for counting.
2. **Whole numbers**: 0, 1, 2, 3…  
    Same as naturals, but include zero.
3. **Integers**: …–3, –2, –1, 0, 1, 2, 3…  
    These include negatives.

### **1.3 Place value**

In a number like **4,382**, every digit has a “position” that changes its meaning:

- 4 → thousands
- 3 → hundreds
- 8 → tens
- 2 → ones

This positional structure is the core of our number system.
