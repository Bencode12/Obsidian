## The Idea of Quantity

> [!tip] Numbers are Abstract
> 
> A number like "5" doesn't exist physically — it's a concept that represents any collection of five things, whether apples, stars, or ideas.

|Number|Meaning|Example|
|---|---|---|
|1|a single item|one apple|
|2|a pair|two hands|
|5|a handful|five fingers|
|10|a decade|ten years|

## The Three Families of Basic Numbers

|Family|Set|Used for|
|---|---|---|
|**Natural numbers**|$1, 2, 3, 4, \ldots$|Counting|
|**Whole numbers**|$0, 1, 2, 3, \ldots$|Counting including zero|
|**Integers**|$\ldots, -3, -2, -1, 0, 1, 2, 3, \ldots$|Counting with negatives|

> [!info] Number System Hierarchy
> 
> Each number system builds on the previous one to solve limitations:
> - **Natural numbers** can't represent "nothing" → add zero → **Whole numbers**
> - **Whole numbers** can't represent debt → add negatives → **Integers**
> - **Integers** can't represent parts → add fractions → **Rational numbers**

## Place Value

Every digit in a number has a **position** that changes its value. In $4{,}382$:

|Digit|Position|Value|
|---|---|---|
|4|Thousands|$4000$|
|3|Hundreds|$300$|
|8|Tens|$80$|
|2|Ones|$2$|

> [!example] Place Value in Action
> 
> The number 4,382 means:
> 
> $$4 \times 1000 + 3 \times 100 + 8 \times 10 + 2 \times 1 = 4000 + 300 + 80 + 2$$
> 
> If we rearrange digits to 2,834, the value changes completely because each digit's position determines its contribution.
