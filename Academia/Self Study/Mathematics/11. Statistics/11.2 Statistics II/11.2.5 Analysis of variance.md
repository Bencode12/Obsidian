## One-Way ANOVA

### Purpose

Compare means across $k \geq 3$ groups simultaneously.

Tests: $H_0: \mu_1 = \mu_2 = ... = \mu_k$ vs $H_1$: At least one differs

### Model

$$Y_{ij} = \mu_i + \epsilon_{ij} = \mu + \tau_i + \epsilon_{ij}$$

where:
- $\mu$: overall mean
- $\tau_i$: treatment effect for group $i$
- $\epsilon_{ij} \sim N(0, \sigma^2)$: random error

Assumptions: Normality, equal variances, independence

---

## Sum of Squares Decomposition

### Total Variation

$$SS_T = \sum_{i=1}^k\sum_{j=1}^{n_i}(Y_{ij} - \bar{Y}_{..})^2$$

### Between-Group Variation

$$SS_B = \sum_{i=1}^k n_i(\bar{Y}_{i.} - \bar{Y}_{..})^2$$

Measures variation due to different group means.

### Within-Group Variation

$$SS_W = \sum_{i=1}^k\sum_{j=1}^{n_i}(Y_{ij} - \bar{Y}_{i.})^2$$

Measures variation within each group (error).

### Decomposition

$$SS_T = SS_B + SS_W$$

Total variation = Between-group + Within-group

---

## F-Test

### Mean Squares

$$MS_B = \frac{SS_B}{k-1}, \quad MS_W = \frac{SS_W}{N-k}$$

where $N = \sum n_i$ is total sample size.

### Test Statistic

$$F = \frac{MS_B}{MS_W} \sim F_{k-1, N-k} \text{ under } H_0$$

Large $F$ → between-group variation dominates → reject $H_0$!

### ANOVA Table

|Source|SS|df|MS|F|
|---|---|---|---|---|
|Between|$SS_B$|$k-1$|$MS_B$|$MS_B/MS_W$|
|Within|$SS_W$|$N-k$|$MS_W$||
|Total|$SS_T$|$N-1$|||

## Post-Hoc Comparisons

If ANOVA rejects $H_0$, which groups differ?

### Tukey's HSD (Honestly Significant Difference)

For all pairwise comparisons:

$$\text{HSD} = q_{\alpha,k,N-k}\sqrt{\frac{MS_W}{2}\left(\frac{1}{n_i} + \frac{1}{n_j}\right)}$$

Reject if $|\bar{Y}_i - \bar{Y}_j| > \text{HSD}$

Controls family-wise error rate!

### Bonferroni Correction

For $m$ comparisons, use significance level $\alpha/m$ for each test.

Conservative but simple!

### Scheffé's Method

Most flexible—allows any contrast, not just pairwise.

More conservative than Tukey for pairwise comparisons.

## Two-Way ANOVA

### Model with Interaction

$$Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \epsilon_{ijk}$$

- $\alpha_i$: effect of factor A (level $i$)
- $\beta_j$: effect of factor B (level $j$)
- $(\alpha\beta)_{ij}$: interaction effect

### Three Hypothesis Tests

1. **Main effect A**: $H_0: \alpha_1 = ... = \alpha_a = 0$
2. **Main effect B**: $H_0: \beta_1 = ... = \beta_b = 0$
3. **Interaction**: $H_0: (\alpha\beta)_{ij} = 0$ for all $i,j$

### Interaction Plot

Plot means for each combination of factors.

- Parallel lines: no interaction
- Non-parallel lines: interaction present!

```desmos-graph
y = x
y = x + 2
x = [1, 4]
```

```desmos-graph

y = -x + 5
x = [1, 4]
```

> [!example] Visual Interpretation
> Top: parallel lines (no interaction). Bottom: crossing lines (strong interaction)—effect of one factor depends on the other!

## Assumptions and Diagnostics

### Key Assumptions

1. **Normality**: Residuals normally distributed
2. **Homoscedasticity**: Equal variances across groups
3. **Independence**: Observations independent

### Diagnostic Plots

- **Residuals vs fitted**: Check for patterns (violates linearity/equal variance)
- **Q-Q plot**: Check normality
- **Box plots by group**: Check equal variance visually

### Remedies

- Non-normality: Transform data (log, square root) or use non-parametric test
- Unequal variances: Welch's ANOVA or transform
- Dependence: Use repeated measures ANOVA

## Kruskal-Wallis Test (Non-Parametric ANOVA)

When normality assumption fails!

**Procedure**:
1. Rank all observations together
2. Compute rank sums for each group
3. Test statistic:
   $$H = \frac{12}{N(N+1)}\sum_{i=1}^k \frac{R_i^2}{n_i} - 3(N+1)$$

Under $H_0$: $H \sim \chi^2_{k-1}$ approximately.

## Common Mistakes

> [!warning] Running Multiple T-Tests Instead of ANOVA
> Increases Type I error! For 3 groups, three t-tests give ~14% chance of false positive (not 5%).

> [!warning] Ignoring Interaction Effects
> In two-way ANOVA, always check interaction first. If significant, main effects may be misleading!

> [!warning] Forgetting Post-Hoc Corrections
> Multiple comparisons inflate error rate. Always adjust p-values or use methods like Tukey's HSD.

> [!warning] Assuming Equal Sample Sizes Required
> ANOVA works with unequal $n_i$, but power is reduced. Balanced designs are optimal.
