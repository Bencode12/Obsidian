## **Lesson 11.2.1 — Hypothesis Testing**

Hypothesis testing is the formal framework for **making decisions under uncertainty** using data. It allows us to test claims about population parameters using sample evidence.

---

## **11.2.1.1 Core Idea**

We start with a claim about a parameter (\theta), then use data to decide whether to **reject** it.

Two hypotheses:

- **Null hypothesis** (H_0): baseline assumption
    
- **Alternative hypothesis** (H_1) or (H_a): competing claim
    

---

## **11.2.1.2 Types of Tests**

1. **Two-sided test:**  
    [  
    H_0: \theta = \theta_0 \quad \text{vs} \quad H_1: \theta \neq \theta_0  
    ]
    
2. **Left-tailed test:**  
    [  
    H_0: \theta = \theta_0 \quad \text{vs} \quad H_1: \theta < \theta_0  
    ]
    
3. **Right-tailed test:**  
    [  
    H_0: \theta = \theta_0 \quad \text{vs} \quad H_1: \theta > \theta_0  
    ]
    

---

## **11.2.1.3 Test Statistic**

A **test statistic** measures how far the sample is from what (H_0) predicts.

### Example (mean, known variance):

[  
Z = \frac{\bar{X} - \mu_0}{\sigma / \sqrt{n}}  
]

- Under (H_0): (Z \sim N(0,1))
    

---

## **11.2.1.4 Rejection Region**

We reject (H_0) if the test statistic falls in a **critical region**.

- Significance level: (\alpha) (e.g., 0.05)
    
- Critical value: (z_{\alpha/2}), (z_\alpha)
    

Example (two-sided test):

[  
\text{Reject } H_0 \text{ if } |Z| > z_{\alpha/2}  
]

---

## **11.2.1.5 p-value**

The **p-value** is:

[  
\text{p-value} = P(\text{observing data as extreme or more extreme than current}, H_0)  
]

Decision rule:

- If (p \le \alpha): reject (H_0)
    
- If (p > \alpha): fail to reject (H_0)
    

---

## **11.2.1.6 Errors**

1. **Type I Error** (False Positive):
    

- Reject (H_0) when it is true
    
- Probability = (\alpha)
    

2. **Type II Error** (False Negative):
    

- Fail to reject (H_0) when it is false
    
- Probability = (\beta)
    

3. **Power of test:**  
    [  
    \text{Power} = 1 - \beta  
    ]
    

- Probability of correctly rejecting false (H_0)
    

---

## **11.2.1.7 Common Tests**

1. **Z-test:**
    

- Known variance or large (n)
    

2. **t-test:**
    

- Unknown variance, small sample
    

3. **Chi-square test:**
    

- Variance, goodness-of-fit
    

4. **Proportion test:**
    

- Binary outcomes
    

---

## **11.2.1.8 Example**

Test:  
[  
H_0: \mu = 100, \quad H_1: \mu \neq 100  
]

Given:

- (\bar{X} = 104), (\sigma = 10), (n = 25)
    

Compute:

[  
Z = \frac{104 - 100}{10/5} = \frac{4}{2} = 2  
]

At (\alpha = 0.05), critical value:

[  
z_{0.025} \approx 1.96  
]

Since (2 > 1.96):

✅ Reject (H_0)

---

## **11.2.1.9 Connection to Confidence Intervals**

- If (\theta_0) is **outside the confidence interval**, reject (H_0)
    
- If inside, fail to reject
    

> Hypothesis testing and confidence intervals are mathematically equivalent frameworks.
