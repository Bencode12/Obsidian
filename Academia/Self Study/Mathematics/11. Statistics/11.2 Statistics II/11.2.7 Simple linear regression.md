## Simple Linear Regression

### Model

$$Y_i = \beta_0 + \beta_1 X_i + \epsilon_i$$

where:
- $\beta_0$: intercept
- $\beta_1$: slope
- $\epsilon_i \sim N(0, \sigma^2)$: random error (i.i.d.)

Assumptions: Linearity, independence, homoscedasticity, normality

### Least Squares Estimation

Minimize sum of squared residuals:

$$SSE = \sum_{i=1}^n (Y_i - \hat{Y}_i)^2 = \sum_{i=1}^n (Y_i - \beta_0 - \beta_1 X_i)^2$$

**Solutions**:
$$\hat{\beta}_1 = \frac{\sum(X_i - \bar{X})(Y_i - \bar{Y})}{\sum(X_i - \bar{X})^2} = \frac{S_{XY}}{S_{XX}}$$

$$\hat{\beta}_0 = \bar{Y} - \hat{\beta}_1\bar{X}$$

```desmos-graph
y = 2x + 1
y = 2x + 1 + normaldist(0, 1)
x = [0, 5]
```

> [!example] Visual Interpretation
> Blue line: true relationship. Orange points: observed data with noise. Regression finds best-fitting line.

## Properties of Estimators

### Unbiasedness

$$E[\hat{\beta}_0] = \beta_0, \quad E[\hat{\beta}_1] = \beta_1$$

### Variances

$$\text{Var}(\hat{\beta}_1) = \frac{\sigma^2}{S_{XX}}$$

$$\text{Var}(\hat{\beta}_0) = \sigma^2\left(\frac{1}{n} + \frac{\bar{X}^2}{S_{XX}}\right)$$

More spread in $X$ → more precise slope estimate!

### Sampling Distributions

$$\hat{\beta}_1 \sim N\left(\beta_1, \frac{\sigma^2}{S_{XX}}\right)$$

$$\hat{\beta}_0 \sim N\left(\beta_0, \sigma^2\left(\frac{1}{n} + \frac{\bar{X}^2}{S_{XX}}\right)\right)$$

---

## Inference for Regression Coefficients

### Estimating Error Variance

$$\hat{\sigma}^2 = MSE = \frac{SSE}{n-2} = \frac{\sum(Y_i - \hat{Y}_i)^2}{n-2}$$

Unbiased estimator of $\sigma^2$.

### T-Tests for Coefficients

Test $H_0: \beta_1 = 0$ (no linear relationship):

$$T = \frac{\hat{\beta}_1}{SE(\hat{\beta}_1)} = \frac{\hat{\beta}_1}{\hat{\sigma}/\sqrt{S_{XX}}} \sim t_{n-2}$$

Similarly for intercept: $T = \frac{\hat{\beta}_0}{SE(\hat{\beta}_0)} \sim t_{n-2}$

### Confidence Intervals

$$\hat{\beta}_1 \pm t_{\alpha/2, n-2} \cdot SE(\hat{\beta}_1)$$

---

## Analysis of Variance for Regression

### Sum of Squares Decomposition

$$SS_T = SS_R + SS_E$$

- **Total**: $SS_T = \sum(Y_i - \bar{Y})^2$
- **Regression**: $SS_R = \sum(\hat{Y}_i - \bar{Y})^2$
- **Error**: $SS_E = \sum(Y_i - \hat{Y}_i)^2$

### F-Test

$$F = \frac{MS_R}{MS_E} = \frac{SS_R/1}{SS_E/(n-2)} \sim F_{1, n-2}$$

Tests overall significance of regression.

Note: $F = T^2$ for simple linear regression!

### R-Squared

$$R^2 = \frac{SS_R}{SS_T} = 1 - \frac{SS_E}{SS_T}$$

Proportion of variance explained by model.

Range: [0, 1], where 1 = perfect fit.

## Prediction and Confidence Intervals

### Mean Response

CI for $E[Y|X=x_0] = \beta_0 + \beta_1 x_0$:

$$\hat{Y}_0 \pm t_{\alpha/2, n-2} \cdot \hat{\sigma}\sqrt{\frac{1}{n} + \frac{(x_0-\bar{X})^2}{S_{XX}}}$$

### Individual Prediction

Prediction interval for new observation $Y_0$:

$$\hat{Y}_0 \pm t_{\alpha/2, n-2} \cdot \hat{\sigma}\sqrt{1 + \frac{1}{n} + \frac{(x_0-\bar{X})^2}{S_{XX}}}$$

Wider than CI—accounts for both estimation error and random variation!

```desmos-graph
y = 2x + 1
y = 2x + 1 + 2
y = 2x + 1 - 2
x = [0, 5]
```

> [!example] Visual Interpretation
> Middle line: regression line. Outer bands: prediction intervals. Notice they widen at extremes!

## Correlation

### Pearson Correlation Coefficient

$$r = \frac{\sum(X_i - \bar{X})(Y_i - \bar{Y})}{\sqrt{\sum(X_i - \bar{X})^2\sum(Y_i - \bar{Y})^2}} = \frac{S_{XY}}{\sqrt{S_{XX}S_{YY}}}$$

Properties:
- Range: [-1, 1]
- $r = 1$: perfect positive linear relationship
- $r = -1$: perfect negative linear relationship
- $r = 0$: no linear relationship

### Relationship to Regression

$$R^2 = r^2$$

In simple linear regression, coefficient of determination equals squared correlation!

### Test for Correlation

Test $H_0: \rho = 0$:

$$T = r\sqrt{\frac{n-2}{1-r^2}} \sim t_{n-2}$$

## Residual Analysis

### Checking Assumptions

1. **Linearity**: Residuals vs fitted plot should show no pattern
2. **Constant variance**: Residual spread should be uniform
3. **Normality**: Q-Q plot of residuals should be linear
4. **Independence**: No autocorrelation (Durbin-Watson test)

### Standardized Residuals

$$r_i = \frac{e_i}{\hat{\sigma}\sqrt{1-h_{ii}}}$$

where $h_{ii}$ are leverage values.

Identify outliers: $|r_i| > 2$ or $|r_i| > 3$

---

## Common Mistakes

> [!warning] Assuming Causation from Correlation
> Correlation ≠ causation! Confounding variables may explain relationship.

> [!warning] Extrapolating Beyond Data Range
> Regression model only valid within range of observed $X$ values.

> [!warning] Ignoring Outliers and Influential Points
> Single observations can dramatically affect regression. Check Cook's distance!

> [!warning] Using R² as Sole Measure of Fit
> High R² doesn't guarantee good model. Check residual plots and assumptions!
