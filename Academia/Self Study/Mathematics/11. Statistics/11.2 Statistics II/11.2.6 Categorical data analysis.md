## Chi-Squared Goodness-of-Fit Test

### Purpose

Test whether observed data follows a specified distribution.

**Hypotheses**:
- $H_0$: Data follows distribution with probabilities $p_1, ..., p_k$
- $H_1$: Data doesn't follow specified distribution

### Test Statistic

$$\chi^2 = \sum_{i=1}^k \frac{(O_i - E_i)^2}{E_i}$$

where:
- $O_i$: observed frequency in category $i$
- $E_i = np_i$: expected frequency under $H_0$

Under $H_0$: $\chi^2 \sim \chi^2_{k-1-m}$ where $m$ = number of estimated parameters

### Requirements

- All $E_i \geq 5$ (or combine categories)
- Independent observations
- Categorical data

### Example

Test if die is fair (roll 60 times):

|Face|Observed|Expected|$(O-E)^2/E$|
|---|---|---|---|
|1|8|10|0.4|
|2|12|10|0.4|
|3|9|10|0.1|
|4|11|10|0.1|
|5|10|10|0|
|6|10|10|0|

$\chi^2 = 1.0$, df = 5, p-value > 0.05 → Don't reject fairness!

## Test of Independence

### Contingency Tables

Test whether two categorical variables are independent.

**Example**: Smoking vs Lung Cancer

||Cancer|No Cancer|Total|
|---|---|---|---|
|Smoker|60|140|200|
|Non-smoker|20|180|200|
|Total|80|320|400|

### Hypotheses

- $H_0$: Variables are independent
- $H_1$: Variables are associated

### Expected Frequencies

$$E_{ij} = \frac{(\text{row total}_i)(\text{column total}_j)}{\text{grand total}}$$

For smoker-cancer cell: $E_{11} = \frac{200 \times 80}{400} = 40$

### Test Statistic

$$\chi^2 = \sum_{i,j} \frac{(O_{ij} - E_{ij})^2}{E_{ij}} \sim \chi^2_{(r-1)(c-1)}$$

where $r$ = rows, $c$ = columns

### Degrees of Freedom

df = $(r-1)(c-1)$

For 2×2 table: df = 1

## Test of Homogeneity

### Purpose

Test whether multiple populations have same distribution across categories.

Similar to independence test, but sampling scheme differs:
- **Independence**: One sample, two variables measured
- **Homogeneity**: Multiple samples, one variable measured

Same test statistic and procedure!

## Fisher's Exact Test

### When to Use

For 2×2 tables with small sample sizes (expected counts < 5).

Chi-squared approximation fails—use exact distribution!

### Hypergeometric Distribution

Given fixed margins, the cell counts follow hypergeometric distribution.

**P-value**: Sum probabilities of all tables as or more extreme than observed.

### Example

||Success|Failure|Total|
|---|---|---|---|
|Group A|8|2|10|
|Group B|2|8|10|
|Total|10|10|20|

Compute exact p-value using hypergeometric probabilities.

## Measures of Association

### Phi Coefficient (2×2 Tables)

$$\phi = \sqrt{\frac{\chi^2}{n}}$$

Range: [0, 1], where 0 = no association

### Cramer's V (General Tables)

$$V = \sqrt{\frac{\chi^2}{n \cdot \min(r-1, c-1)}}$$

Standardized measure of association strength.

### Interpretation

|Value|Strength|
|---|---|
|0 - 0.1|Negligible|
|0.1 - 0.3|Weak|
|0.3 - 0.5|Moderate|
|0.5 - 1.0|Strong|

## Odds Ratios

### Definition

For 2×2 table:

||Outcome+|Outcome-|
|---|---|---|
|Exposed|$a$|$b$|
|Unexposed|$c$|$d$|

**Odds ratio**:
$$OR = \frac{a/b}{c/d} = \frac{ad}{bc}$$

- OR = 1: No association
- OR > 1: Positive association
- OR < 1: Negative association

### Confidence Interval

$$\log(OR) \pm z_{\alpha/2}\sqrt{\frac{1}{a} + \frac{1}{b} + \frac{1}{c} + \frac{1}{d}}$$

Exponentiate to get CI for OR.

## Cochran's Rule

### Guidelines for Chi-Squared Tests

Validity requires:
- No more than 20% of expected counts < 5
- All expected counts ≥ 1

If violated:
- Combine categories
- Use Fisher's exact test
- Increase sample size

## Common Mistakes

> [!warning] Using Chi-Squared on Continuous Data
> Must categorize first! Or use appropriate continuous tests (KS test, etc.).

> [!warning] Ignoring Small Expected Counts
> Chi-squared approximation fails with small $E_i$. Check assumptions!

> [!warning] Confusing Statistical Significance with Practical Importance
> Large samples can give significant chi-squared with weak association. Check effect size (Cramer's V)!

> [!warning] Treating Ordinal Data as Nominal
> If categories have order, use more powerful tests (Cochran-Armitage trend test).
