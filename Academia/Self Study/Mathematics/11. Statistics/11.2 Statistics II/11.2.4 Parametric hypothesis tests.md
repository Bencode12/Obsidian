## One-Sample Tests

### Z-Test (Known Variance)

**Hypotheses**: $H_0: \mu = \mu_0$ vs $H_1: \mu \neq \mu_0$ (or one-sided)

**Test statistic**:
$$Z = \frac{\bar{X} - \mu_0}{\sigma/\sqrt{n}} \sim N(0,1) \text{ under } H_0$$

**Decision rule**: Reject $H_0$ if $|Z| > z_{\alpha/2}$

Requires: Normal population OR large sample (CLT), known $\sigma$

### T-Test (Unknown Variance)

**Test statistic**:
$$T = \frac{\bar{X} - \mu_0}{S/\sqrt{n}} \sim t_{n-1} \text{ under } H_0$$

More realistic—uses sample standard deviation!

```desmos-graph
y = normaldist(0, 1)
y = studenttdist(10)
x = [-4, 4]
```

> [!example] Visual Interpretation
> Notice t-distribution has heavier tails. This makes it harder to reject $H_0$, accounting for uncertainty in estimating $\sigma$.

## Two-Sample Tests

### Independent Samples (Equal Variances)

**Pooled variance**:
$$S_p^2 = \frac{(n_1-1)S_1^2 + (n_2-1)S_2^2}{n_1+n_2-2}$$

**Test statistic**:
$$T = \frac{\bar{X}_1 - \bar{X}_2}{S_p\sqrt{\frac{1}{n_1} + \frac{1}{n_2}}} \sim t_{n_1+n_2-2}$$

### Independent Samples (Unequal Variances)

**Welch's t-test**:
$$T = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}}}$$

Approximate degrees of freedom (Welch-Satterthwaite):
$$\nu \approx \frac{\left(\frac{S_1^2}{n_1} + \frac{S_2^2}{n_2}\right)^2}{\frac{(S_1^2/n_1)^2}{n_1-1} + \frac{(S_2^2/n_2)^2}{n_2-1}}$$

More robust when variances differ!

### Paired T-Test

For matched pairs or before-after measurements.

Compute differences: $D_i = X_{i1} - X_{i2}$

Then use one-sample t-test on differences:
$$T = \frac{\bar{D}}{S_D/\sqrt{n}} \sim t_{n-1}$$

Eliminates between-subject variability!

## Tests for Variance

### Chi-Squared Test (One Sample)

**Hypotheses**: $H_0: \sigma^2 = \sigma_0^2$

**Test statistic**:
$$\chi^2 = \frac{(n-1)S^2}{\sigma_0^2} \sim \chi^2_{n-1} \text{ under } H_0$$

### F-Test (Two Samples)

**Hypotheses**: $H_0: \sigma_1^2 = \sigma_2^2$

**Test statistic**:
$$F = \frac{S_1^2}{S_2^2} \sim F_{n_1-1, n_2-1} \text{ under } H_0$$

Sensitive to non-normality! Use Levene's test for robustness.

## Proportion Tests

### One-Sample Proportion

**Test statistic** (large sample):
$$Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}} \sim N(0,1)$$

where $\hat{p} = X/n$ is sample proportion.

### Two-Sample Proportion

**Pooled proportion**: $\hat{p} = \frac{X_1 + X_2}{n_1 + n_2}$

**Test statistic**:
$$Z = \frac{\hat{p}_1 - \hat{p}_2}{\sqrt{\hat{p}(1-\hat{p})\left(\frac{1}{n_1} + \frac{1}{n_2}\right)}}$$

## Power and Sample Size

### Power Function

**Power** = $P(\text{Reject } H_0 | H_1 \text{ is true}) = 1 - \beta$

Depends on:
- True parameter value
- Sample size $n$
- Significance level $\alpha$
- Variability

### Computing Power (One-Sample Z-Test)

For $H_1: \mu = \mu_1$:

$$\text{Power} = \Phi\left(-z_{\alpha/2} + \frac{|\mu_1 - \mu_0|}{\sigma/\sqrt{n}}\right) + \Phi\left(-z_{\alpha/2} - \frac{|\mu_1 - \mu_0|}{\sigma/\sqrt{n}}\right)$$

where $\Phi$ is standard normal CDF.

### Sample Size Calculation

To achieve power $1-\beta$ for detecting effect size $\delta = |\mu_1 - \mu_0|$:

$$n = \left(\frac{(z_{\alpha/2} + z_\beta)\sigma}{\delta}\right)^2$$

Larger effect → smaller sample needed!

```desmos-graph
y = normaldist(0, 1)
y = normaldist(2, 1)
x = [-3, 6]
```

> [!example] Visual Interpretation
> Blue: null distribution. Orange: alternative. Power is area under orange curve beyond critical value.

## Type I and Type II Errors

|Decision|$H_0$ True|$H_0$ False|
|---|---|---|
|Don't reject|Correct ($1-\alpha$)|Type II error ($\beta$)|
|Reject|Type I error ($\alpha$)|Correct (Power = $1-\beta$)|

### Trade-offs

- Decreasing $\alpha$ increases $\beta$ (for fixed $n$)
- Increasing $n$ decreases both errors
- Can't minimize both simultaneously without more data!

## Common Mistakes

> [!warning] Confusing Statistical and Practical Significance
> Small p-value doesn't mean large effect! Always report effect size and confidence intervals.

> [!warning] Using Wrong Test
> Check assumptions: normality, equal variances, independence. Violations invalidate results.

> [!warning] P-Hacking
> Don't try multiple tests until you get significance! Pre-specify hypotheses and analysis plan.

> [!warning] Ignoring Power
> A non-significant result doesn't prove $H_0$—it might just be underpowered!
