## Multiple Linear Regression

### Model

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + ... + \beta_p X_p + \epsilon$$

Or in matrix form:
$$\mathbf{Y} = \mathbf{X}\boldsymbol{\beta} + \boldsymbol{\epsilon}$$

where:
- $\mathbf{Y}$: $n \times 1$ response vector
- $\mathbf{X}$: $n \times (p+1)$ design matrix (includes column of 1s)
- $\boldsymbol{\beta}$: $(p+1) \times 1$ coefficient vector
- $\boldsymbol{\epsilon} \sim N(\mathbf{0}, \sigma^2\mathbf{I})$

---

## Least Squares Estimation

### Normal Equations

Minimize $SSE = (\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})^T(\mathbf{Y} - \mathbf{X}\boldsymbol{\beta})$

**Solution**:
$$\hat{\boldsymbol{\beta}} = (\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T\mathbf{Y}$$

Requires $\mathbf{X}^T\mathbf{X}$ to be invertible (no perfect multicollinearity).

### Properties

- **Unbiased**: $E[\hat{\boldsymbol{\beta}}] = \boldsymbol{\beta}$
- **Variance**: $\text{Var}(\hat{\boldsymbol{\beta}}) = \sigma^2(\mathbf{X}^T\mathbf{X})^{-1}$
- **BLUE**: Best Linear Unbiased Estimator (Gauss-Markov theorem)

---

## Inference for Multiple Regression

### Estimating Error Variance

$$\hat{\sigma}^2 = MSE = \frac{SSE}{n-p-1} = \frac{\mathbf{Y}^T(\mathbf{I}-\mathbf{H})\mathbf{Y}}{n-p-1}$$

where $\mathbf{H} = \mathbf{X}(\mathbf{X}^T\mathbf{X})^{-1}\mathbf{X}^T$ is the hat matrix.

### T-Tests for Individual Coefficients

Test $H_0: \beta_j = 0$:

$$T = \frac{\hat{\beta}_j}{SE(\hat{\beta}_j)} \sim t_{n-p-1}$$

where $SE(\hat{\beta}_j) = \hat{\sigma}\sqrt{[(\mathbf{X}^T\mathbf{X})^{-1}]_{jj}}$

### F-Test for Overall Significance

Test $H_0: \beta_1 = \beta_2 = ... = \beta_p = 0$:

$$F = \frac{MS_R}{MS_E} = \frac{SS_R/p}{SS_E/(n-p-1)} \sim F_{p, n-p-1}$$

---

## Model Selection

### Adjusted R-Squared

$$R^2_{adj} = 1 - \frac{SS_E/(n-p-1)}{SS_T/(n-1)} = 1 - (1-R^2)\frac{n-1}{n-p-1}$$

Penalizes adding unnecessary predictors!

Unlike $R^2$, adjusted $R^2$ can decrease when adding variables.

### Information Criteria

**AIC (Akaike Information Criterion)**:
$$AIC = n\log(SSE/n) + 2(p+1)$$

**BIC (Bayesian Information Criterion)**:
$$BIC = n\log(SSE/n) + (p+1)\log(n)$$

Lower values indicate better models. BIC penalizes complexity more heavily.

### Stepwise Selection

- **Forward**: Start with no variables, add most significant
- **Backward**: Start with all variables, remove least significant
- **Stepwise**: Combination of forward and backward

## Multicollinearity

### Problem

When predictors are highly correlated:
- Large standard errors for coefficients
- Unstable estimates
- Difficult interpretation

### Detection

**Variance Inflation Factor (VIF)**:
$$VIF_j = \frac{1}{1-R_j^2}$$

where $R_j^2$ is $R^2$ from regressing $X_j$ on all other predictors.

- VIF > 5 or 10: problematic multicollinearity

### Remedies

- Remove redundant variables
- Combine correlated variables
- Use ridge regression or principal components

---

## Diagnostic Measures

### Leverage

Measures how far an observation is from the center of predictor space.

$$h_{ii} = [\mathbf{H}]_{ii}$$

High leverage: $h_{ii} > 2(p+1)/n$

### Cook's Distance

Measures influence of each observation on all coefficients:

$$D_i = \frac{e_i^2}{(p+1)MSE}\frac{h_{ii}}{(1-h_{ii})^2}$$

Influential if $D_i > 1$ or $D_i > 4/n$

### DFFITS and DFBETAS

- **DFFITS**: Change in fitted value when observation removed
- **DFBETAS**: Change in each coefficient when observation removed

## Polynomial Regression

### Model

Include polynomial terms:
$$Y = \beta_0 + \beta_1 X + \beta_2 X^2 + ... + \beta_k X^k + \epsilon$$

Still linear in parameters—use multiple regression!

### Orthogonal Polynomials

Use orthogonal polynomials to reduce multicollinearity between powers of $X$.

```desmos-graph
y = x
y = x^2
y = x^3
x = [-2, 2]
```

> [!example] Visual Interpretation
> Different polynomial degrees capture different curvature patterns. Choose degree based on data and theory.

## Interaction Terms

### Model with Interaction

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \beta_3 X_1 X_2 + \epsilon$$

The effect of $X_1$ depends on $X_2$ (and vice versa).

### Interpretation

- $\frac{\partial Y}{\partial X_1} = \beta_1 + \beta_3 X_2$
- Effect of $X_1$ changes with level of $X_2$

Always include main effects when including interaction!

---

## Common Mistakes

> [!warning] Overfitting with Too Many Predictors
> High $R^2$ doesn't mean good model! Use adjusted $R^2$, AIC, BIC, or cross-validation.

> [!warning] Ignoring Multicollinearity
> Check VIFs! Correlated predictors make individual coefficients unreliable.

> [!warning] Forgetting to Check Assumptions
> Residual plots are essential! Linearity, constant variance, normality, independence.

> [!warning] Interpreting Coefficients Without Context
> Coefficient $\beta_j$ is effect of $X_j$ holding all other variables constant. This may not be realistic!
