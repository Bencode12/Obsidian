## Motivation

### Parametric vs Non-parametric

**Parametric methods** assume a specific distribution (e.g., normal, exponential):
- Powerful when assumptions hold
- Can fail dramatically when assumptions are violated

**Non-parametric methods** make minimal assumptions:
- More robust to model misspecification
- Often less efficient when parametric assumptions are correct
- Based on ranks, signs, or empirical distributions

---

## Sign Test

### Purpose

Test whether the median of a distribution equals a specified value.

### Procedure

For testing $H_0: \text{median} = m_0$:
1. Count observations above $m_0$ (call this $S^+$)
2. Count observations below $m_0$ (call this $S^-$)
3. Under $H_0$, $S^+ \sim \text{Binomial}(n, 0.5)$

### Example

Data: {3, 7, 8, 12, 15}, test if median = 10

- Above 10: {12, 15} → $S^+ = 2$
- Below 10: {3, 7, 8} → $S^- = 3$
- p-value from Binomial(5, 0.5)

Simple but wastes information (only uses signs, not magnitudes)!

## Wilcoxon Signed-Rank Test

### Purpose

More powerful alternative to sign test for paired data or one-sample location.

### Procedure

For paired differences $D_i = X_i - Y_i$:
1. Compute absolute values $|D_i|$
2. Rank them from smallest to largest
3. Assign signs back to ranks
4. Test statistic: $W = \sum$ of positive ranks

Under $H_0$, $W$ has known distribution (tabulated).

### Advantages

- Uses magnitude information (via ranks)
- More powerful than sign test
- Still robust to outliers

```desmos-graph
y = normaldist(0, 1)
x = [-4, 4]
```

> [!example] Visual Interpretation
> Unlike t-test which assumes normality, Wilcoxon test works for any symmetric distribution.

## Mann-Whitney U Test (Wilcoxon Rank-Sum)

### Purpose

Compare two independent samples without assuming normality.

Tests: $H_0$: Both samples come from same distribution

### Procedure

1. Combine both samples and rank all observations
2. Sum ranks for each group: $R_1$ and $R_2$
3. Compute U statistics:
   $$U_1 = R_1 - \frac{n_1(n_1+1)}{2}$$
   $$U_2 = R_2 - \frac{n_2(n_2+1)}{2}$$
4. Use smaller U for testing

### Asymptotic Distribution

For large samples, $U$ is approximately normal:
$$Z = \frac{U - \mu_U}{\sigma_U} \sim N(0,1)$$

where $\mu_U = \frac{n_1 n_2}{2}$ and $\sigma_U = \sqrt{\frac{n_1 n_2(n_1+n_2+1)}{12}}$

## Kruskal-Wallis Test

### Purpose

Non-parametric alternative to one-way ANOVA.

Tests: $H_0$: All groups have same distribution

### Procedure

1. Rank all observations across all groups
2. Compute rank sums for each group
3. Test statistic:
   $$H = \frac{12}{N(N+1)}\sum_{i=1}^k \frac{R_i^2}{n_i} - 3(N+1)$$

Under $H_0$, $H \sim \chi^2_{k-1}$ approximately.

## Spearman's Rank Correlation

### Definition

**Spearman's $\rho$** measures monotonic association using ranks:

$$\rho_s = 1 - \frac{6\sum d_i^2}{n(n^2-1)}$$

where $d_i$ is the difference between ranks of $X_i$ and $Y_i$.

### Comparison with Pearson

|Property|Pearson $r$|Spearman $\rho_s$|
|---|---|---|
|Measures|Linear association|Monotonic association|
|Assumptions|Normality, linearity|None|
|Robustness|Sensitive to outliers|Robust|
|Range|[-1, 1]|[-1, 1]|

### Example

If $Y = X^3$, Pearson may be low but Spearman = 1 (perfect monotonic)!

```desmos-graph
y = x^3
x = [-2, 2]
```

## Kolmogorov-Smirnov Test

### One-Sample KS Test

Tests if sample comes from a specified distribution.

**Test statistic**:
$$D_n = \sup_x |F_n(x) - F_0(x)|$$

where $F_n$ is empirical CDF and $F_0$ is hypothesized CDF.

### Two-Sample KS Test

Tests if two samples come from same distribution.

**Test statistic**:
$$D_{n,m} = \sup_x |F_n(x) - G_m(x)|$$

### Advantages

- Distribution-free (under $H_0$)
- Sensitive to differences in location, scale, and shape
- Works for continuous distributions

## Bootstrap Methods

### Basic Idea

Use resampling to estimate sampling distributions without parametric assumptions!

### Procedure

1. Draw bootstrap sample (with replacement) from original data
2. Compute statistic of interest
3. Repeat many times (e.g., 1000)
4. Use bootstrap distribution for inference

### Applications

- Confidence intervals
- Standard error estimation
- Bias correction
- Hypothesis testing

### Types

|Type|Description|
|---|---|
|Non-parametric|Resample from empirical distribution|
|Parametric|Resample from fitted parametric model|
|Block bootstrap|For time series data|

## Kernel Density Estimation

### Purpose

Estimate probability density function without assuming parametric form.

### Formula

$$\hat{f}_h(x) = \frac{1}{nh}\sum_{i=1}^n K\left(\frac{x-X_i}{h}\right)$$

where:
- $K$ is kernel function (e.g., Gaussian)
- $h$ is bandwidth (smoothing parameter)

### Bandwidth Selection

- Small $h$: undersmoothed (noisy)
- Large $h$: oversmoothed (misses features)
- Optimal: balance bias and variance

```desmos-graph
y = exp(-x^2/2)/sqrt(2*pi)
y = 0.5*exp(-(x-2)^2/0.5)/sqrt(pi) + 0.5*exp(-(x+2)^2/0.5)/sqrt(pi)
x = [-5, 5]
```

> [!example] Visual Interpretation
> Top: unimodal density. Bottom: bimodal mixture—kernel density can capture complex shapes!

## Common Mistakes

> [!warning] Assuming Non-parametric Means Assumption-Free
> Non-parametric still makes assumptions (e.g., continuity, symmetry for some tests). Just fewer assumptions!

> [!warning] Using Non-parametric Tests Unnecessarily
> If parametric assumptions hold, parametric tests are more powerful. Don't sacrifice efficiency without reason.

> [!warning] Misinterpreting Rank Tests
> Wilcoxon tests location shift under symmetry assumption. Without symmetry, it tests stochastic dominance.
