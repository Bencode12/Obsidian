## Time Series Components

### Decomposition

Time series can be decomposed into:

$$Y_t = T_t + S_t + C_t + I_t$$

- **Trend** ($T_t$): Long-term direction
- **Seasonal** ($S_t$): Regular periodic patterns
- **Cyclical** ($C_t$): Longer-term oscillations
- **Irregular** ($I_t$): Random noise

### Additive vs Multiplicative

**Additive**: $Y_t = T_t + S_t + I_t$

Use when seasonal variation is constant over time.

**Multiplicative**: $Y_t = T_t \times S_t \times I_t$

Use when seasonal variation increases with level.

Take logs to convert: $\log Y_t = \log T_t + \log S_t + \log I_t$

```desmos-graph
y = (1 + 0.05x)*(1 + 0.3*sin(2*pi*x/12))
x = [0, 60]
```

> Top: additive model (constant amplitude). Bottom: multiplicative model (increasing amplitude with trend).

## Stationarity

### Weak Stationarity

A process is **weakly stationary** if:
1. Constant mean: $E[Y_t] = \mu$ for all $t$
2. Constant variance: $\text{Var}(Y_t) = \sigma^2$ for all $t$
3. Autocovariance depends only on lag: $\text{Cov}(Y_t, Y_{t+h}) = \gamma(h)$

Most time series methods require stationarity!

### Achieving Stationarity

**Differencing**:
$$\Delta Y_t = Y_t - Y_{t-1}$$

Remove trend by taking first differences.

**Seasonal differencing**:
$$\Delta_s Y_t = Y_t - Y_{t-s}$$

Remove seasonal patterns (e.g., $s=12$ for monthly data).

**Transformations**: Log, square root to stabilize variance.

## Autocorrelation

### Autocovariance Function

$$\gamma(h) = \text{Cov}(Y_t, Y_{t+h})$$

Measures linear dependence at lag $h$.

### Autocorrelation Function (ACF)

$$\rho(h) = \frac{\gamma(h)}{\gamma(0)}$$

Standardized to [-1, 1].

### Sample ACF

$$r(h) = \frac{\sum_{t=1}^{n-h}(Y_t - \bar{Y})(Y_{t+h} - \bar{Y})}{\sum_{t=1}^n(Y_t - \bar{Y})^2}$$

Plot ACF to identify patterns!

## ARIMA Models

### Autoregressive (AR) Models

**AR(p)**:
$$Y_t = c + \phi_1 Y_{t-1} + \phi_2 Y_{t-2} + ... + \phi_p Y_{t-p} + \epsilon_t$$

Current value depends on $p$ previous values.

$\epsilon_t \sim WN(0, \sigma^2)$: white noise

### Moving Average (MA) Models

**MA(q)**:
$$Y_t = \mu + \epsilon_t + \theta_1\epsilon_{t-1} + \theta_2\epsilon_{t-2} + ... + \theta_q\epsilon_{t-q}$$

Current value depends on $q$ previous errors.

### ARMA Models

Combine AR and MA:

**ARMA(p,q)**:
$$Y_t = c + \sum_{i=1}^p \phi_i Y_{t-i} + \epsilon_t + \sum_{j=1}^q \theta_j\epsilon_{t-j}$$

### ARIMA Models

**ARIMA(p,d,q)**: Apply ARMA(p,q) to $d$-times differenced series.

- $p$: AR order
- $d$: degree of differencing
- $q$: MA order

Most flexible class of models!

## Model Identification

### Using ACF and PACF

**Partial Autocorrelation Function (PACF)**: Correlation at lag $h$ after removing effects of intermediate lags.

|Model|ACF|PACF|
|---|---|---|
|AR(p)|Tails off|Cuts off after lag $p$|
|MA(q)|Cuts off after lag $q$|Tails off|
|ARMA(p,q)|Tails off|Tails off|

### Information Criteria

**AIC**:
$$AIC = n\log(\hat{\sigma}^2) + 2(p+q+1)$$

**BIC**:
$$BIC = n\log(\hat{\sigma}^2) + (p+q+1)\log(n)$$

Choose model with minimum AIC/BIC.

## Parameter Estimation

### Maximum Likelihood

For Gaussian ARIMA models, use conditional or exact likelihood.

Numerical optimization required!

### Method of Moments

Match sample autocorrelations to theoretical values.

Less efficient than MLE but simpler.

### Yule-Walker Equations (for AR models)

$$\rho(h) = \phi_1\rho(h-1) + ... + \phi_p\rho(h-p)$$

Solve system of equations for $\phi$ parameters.

## Forecasting

### Point Forecasts

For ARIMA models, forecast recursively:

$$\hat{Y}_{n+h} = E[Y_{n+h}|Y_1,...,Y_n]$$

Replace unknown future errors with zero.

### Prediction Intervals

Account for uncertainty in forecasts:

$$\hat{Y}_{n+h} \pm z_{\alpha/2}\sqrt{\text{Var}(e_{n+h})}$$

Intervals widen as forecast horizon increases!

### Forecast Evaluation

**Mean Squared Error (MSE)**:
$$MSE = \frac{1}{m}\sum_{i=1}^m (Y_{n+i} - \hat{Y}_{n+i})^2$$

**Mean Absolute Percentage Error (MAPE)**:
$$MAPE = \frac{100}{m}\sum_{i=1}^m \left|\frac{Y_{n+i} - \hat{Y}_{n+i}}{Y_{n+i}}\right|$$

Lower values indicate better forecasts.

## Seasonal Models

### SARIMA

**SARIMA(p,d,q)(P,D,Q)_s**:

Includes seasonal components with period $s$.

Example: SARIMA(1,1,1)(1,1,1)₁₂ for monthly data with annual seasonality.

### Seasonal Decomposition

**STL (Seasonal-Trend decomposition using LOESS)**:

Robust decomposition method that handles changing seasonality.

More flexible than classical decomposition!

## Common Mistakes

> [!warning] Ignoring Non-Stationarity
> Always check for trends and seasonality before modeling. Non-stationary data gives spurious results!

> [!warning] Overfitting with High Orders
> Complex models may fit past well but forecast poorly. Use parsimony principle.

> [!warning] Forgetting to Check Residuals
> Residuals should be white noise. Check ACF of residuals for remaining patterns.

> [!warning] Extrapolating Too Far
> Forecasts become unreliable as horizon increases. Uncertainty grows rapidly!
