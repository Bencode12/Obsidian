## Likelihood Function

### Definition

For i.i.d. data $X_1, ..., X_n$ with PDF/PMF $f(x|\theta)$:

$$L(\theta; \mathbf{x}) = \prod_{i=1}^n f(x_i|\theta)$$

**Log-likelihood**:
$$\ell(\theta; \mathbf{x}) = \log L(\theta; \mathbf{x}) = \sum_{i=1}^n \log f(x_i|\theta)$$

Easier to work with (products become sums)!

---

## Score Function and Fisher Information

### Score Function

$$U(\theta) = \frac{\partial}{\partial\theta}\ell(\theta; \mathbf{x})$$

First derivative of log-likelihood.

**Property**: $E[U(\theta)] = 0$ under regularity conditions.

### Fisher Information

$$I_n(\theta) = -E\left[\frac{\partial^2}{\partial\theta^2}\ell(\theta; \mathbf{x})\right] = \text{Var}(U(\theta))$$

For i.i.d. data: $I_n(\theta) = nI_1(\theta)$

Measures amount of information about $\theta$ in the data!

---

## Maximum Likelihood Estimation

### MLE Definition

$$\hat{\theta}_{MLE} = \arg\max_\theta L(\theta; \mathbf{x}) = \arg\max_\theta \ell(\theta; \mathbf{x})$$

Usually found by solving score equation:
$$U(\hat{\theta}) = 0$$

### Examples

**Normal mean** ($\sigma^2$ known):
$$\ell(\mu) = -\frac{n}{2}\log(2\pi\sigma^2) - \frac{\sum(x_i-\mu)^2}{2\sigma^2}$$

$$\frac{\partial\ell}{\partial\mu} = \frac{\sum(x_i-\mu)}{\sigma^2} = 0 \implies \hat{\mu} = \bar{x}$$

**Exponential rate**:
$$\ell(\lambda) = n\log\lambda - \lambda\sum x_i$$

$$\frac{\partial\ell}{\partial\lambda} = \frac{n}{\lambda} - \sum x_i = 0 \implies \hat{\lambda} = \frac{1}{\bar{x}}$$

---

## Properties of MLE

### Asymptotic Properties

Under regularity conditions:

1. **Consistency**: $\hat{\theta}_{MLE} \xrightarrow{P} \theta$

2. **Asymptotic Normality**:
   $$\sqrt{n}(\hat{\theta}_{MLE} - \theta) \xrightarrow{d} N\left(0, \frac{1}{I_1(\theta)}\right)$$

3. **Asymptotic Efficiency**: Achieves Cramér-Rao lower bound

4. **Invariance**: If $\hat{\theta}$ is MLE of $\theta$, then $g(\hat{\theta})$ is MLE of $g(\theta)$

### Finite Sample Properties

- May be biased (but bias → 0 as $n \to \infty$)
- Not always exists or unique
- May not have closed-form solution (requires numerical optimization)

## Hypothesis Testing with Likelihood

### Wald Test

$$W = \frac{(\hat{\theta} - \theta_0)^2}{\widehat{\text{Var}}(\hat{\theta})} \sim \chi^2_1 \text{ under } H_0$$

Uses estimated variance at MLE.

### Score Test (Rao's Test)

$$S = \frac{U(\theta_0)^2}{I_n(\theta_0)} \sim \chi^2_1 \text{ under } H_0$$

Only requires fitting null model!

### Likelihood Ratio Test

$$\Lambda = -2\log\left(\frac{L(\theta_0)}{L(\hat{\theta})}\right) = 2[\ell(\hat{\theta}) - \ell(\theta_0)] \sim \chi^2_1 \text{ under } H_0$$

Most reliable of the three!

### Comparison

|Test|Requires|Power|
|---|---|---|
|Wald|Full model|Good|
|Score|Null model only|Good|
|LRT|Both models|Best|

All three are asymptotically equivalent!

## Confidence Intervals from Likelihood

### Wald Interval

$$\hat{\theta} \pm z_{\alpha/2} \cdot SE(\hat{\theta})$$

where $SE(\hat{\theta}) = 1/\sqrt{I_n(\hat{\theta})}$

### Likelihood-Based Interval

Find $\theta$ values where:
$$2[\ell(\hat{\theta}) - \ell(\theta)] \leq \chi^2_{1, \alpha}$$

More accurate than Wald interval, especially for small samples!

### Profile Likelihood

For multi-parameter models, profile out nuisance parameters:

$$\ell_p(\theta_1) = \max_{\theta_2}\ell(\theta_1, \theta_2)$$

Then use profile likelihood for inference on $\theta_1$.

## Numerical Optimization

### When Closed Form Doesn't Exist

Many MLE problems require numerical methods:

**Newton-Raphson**:
$$\theta^{(k+1)} = \theta^{(k)} - \frac{U(\theta^{(k)})}{U'(\theta^{(k)})}$$

**Fisher Scoring**:
$$\theta^{(k+1)} = \theta^{(k)} + \frac{U(\theta^{(k)})}{I_n(\theta^{(k)})}$$

Replace second derivative with expected information.

### EM Algorithm

For models with missing/latent data:

1. **E-step**: Compute expected log-likelihood given current parameters
2. **M-step**: Maximize expected log-likelihood

Iterate until convergence!

## Generalized Linear Models (GLMs)

### Structure

GLMs extend linear regression to non-normal responses:

1. **Random component**: $Y_i$ from exponential family
2. **Systematic component**: $\eta_i = \mathbf{x}_i^T\boldsymbol{\beta}$
3. **Link function**: $g(E[Y_i]) = \eta_i$

### Common GLMs

|Model|Distribution|Link|Use Case|
|---|---|---|---|
|Linear regression|Normal|Identity|Continuous response|
|Logistic regression|Binomial|Logit|Binary outcome|
|Poisson regression|Poisson|Log|Count data|
|Gamma regression|Gamma|Log/Inverse|Positive continuous|

### Logistic Regression Example

$$\log\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1 X$$

$$p = \frac{e^{\beta_0 + \beta_1 X}}{1 + e^{\beta_0 + \beta_1 X}}$$

No closed-form MLE—use iterative methods!

> [!example] Visual Interpretation
> Logistic function maps linear predictor to probability between 0 and 1. S-shaped curve is characteristic of logistic regression.

## Common Mistakes

> [!warning] Assuming MLE is Always Best
> MLE can be poor for small samples or with outliers. Consider robust alternatives!

> [!warning] Ignoring Regularity Conditions
> Asymptotic properties require smoothness, identifiability, and support conditions.

> [!warning] Using Wald Tests Exclusively
> LRT is generally more reliable, especially for small samples or boundary parameters.

> [!warning] Forgetting to Check Convergence
> Numerical optimization may fail to converge or find local maximum. Check multiple starting values!
