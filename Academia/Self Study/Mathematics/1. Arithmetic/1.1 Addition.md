## What is Addition?

Addition means **combining quantities**. Symbol: $+$

$$2 + 3 = 5$$

Two objects combined with three objects make five objects.

---

## Addition on a Number Line

Start at the first number and **move right** by the second number.

$$4 + 3 \quad \longrightarrow \quad \text{start at } 4, \text{ move } 3 \text{ steps right, land on } 7$$

---

## Properties of Addition

### Commutative Property — order does not matter

$$a + b = b + a$$

|Example|Result|
|---|---|
|$2 + 5$|$7$|
|$5 + 2$|$7$|

### Associative Property — grouping does not matter

$$( a + b ) + c = a + ( b + c )$$

|Example|Result|
|---|---|
|$(1 + 2) + 3$|$6$|
|$1 + (2 + 3)$|$6$|

### Identity Property — adding zero

Zero is the **additive identity** — it does not change the number.

$$a + 0 = a \qquad 0 + a = a$$

> [!tip] Why these properties matter Commutative and associative properties mean you can **reorder and regroup** any addition to make calculation easier — e.g. add the easy pairs first.
