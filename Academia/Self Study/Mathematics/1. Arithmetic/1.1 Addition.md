### **1.1.1 What is addition?**

Addition means **combining quantities**.

Symbol: **+**

Example:

- 2 + 3 = 5  
    Two objects combined with three objects make five objects.

### **1.1.2 Addition on a number line**

Start at the first number and move **right**.

Example:

- 4 + 3  
    Start at 4 → move 3 steps right → land on 7

### **1.1.3 Order does not matter (Commutative property)**

- 2 + 5 = 7
- 5 + 2 = 7

So:  
a + b = b + a

### **1.1.4 Grouping does not matter (Associative property)**

- (1 + 2) + 3 = 6
- 1 + (2 + 3) = 6

So:  
(a + b) + c = a + (b + c)

### **1.1.5 Adding zero**

Zero is the **identity** for addition.

- 7 + 0 = 7
- 0 + 7 = 7

Zero does not change the number.
