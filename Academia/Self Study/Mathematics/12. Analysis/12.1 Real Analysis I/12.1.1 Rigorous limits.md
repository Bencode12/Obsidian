## Definition

The **limit** of a function $f(x)$ as $x$ approaches $a$ is $L$, written:

$$\lim_{x \to a} f(x) = L$$

if for every $\epsilon > 0$, there exists $\delta > 0$ such that whenever $0 < |x - a| < \delta$, we have $|f(x) - L| < \epsilon$.

> [!info] Tao Reference
> See *Analysis I* (4th ed.), Chapter 6: Limits of Functions, Sections 6.1-6.3 (pp. 155-170)

> [!warning] Common Misconception
> The limit describes behavior **near** $a$, not **at** $a$. The function doesn't even need to be defined at $a$!

---

## Key Concepts

### Epsilon-Delta Definition

|Component|Meaning|
|---|---|
|$\epsilon$|Tolerance around $L$ (output)|
|$\delta$|Neighborhood around $a$ (input)|
|$0 < |x - a|$|Excludes the point $a$ itself|

**Intuition**: No matter how small a tolerance $\epsilon$ you specify around $L$, I can find a neighborhood $\delta$ around $a$ where all function values stay within that tolerance.

```desmos-graph
y = x^2
x = 1
y = 1
```

> [!example] Visual Interpretation
> The epsilon-delta definition means: for any horizontal band around $L$ (height $2\epsilon$), we can find a vertical band around $a$ (width $2\delta$) such that the function stays within the horizontal band.

### Proof Strategy Template

To prove $\lim_{x \to a} f(x) = L$:

1. **Start**: "Let $\epsilon > 0$ be given"
2. **Choose**: Find $\delta$ in terms of $\epsilon$
3. **Assume**: "Suppose $0 < |x - a| < \delta$"
4. **Show**: Prove $|f(x) - L| < \epsilon$
5. **Conclude**: "Therefore, the limit equals $L$"

## Limit Laws

If $\lim_{x \to a} f(x) = L$ and $\lim_{x \to a} g(x) = M$, then:

|Law|Formula|
|---|---|
|Sum|$\lim_{x \to a} [f(x) + g(x)] = L + M$|
|Product|$\lim_{x \to a} [f(x) \cdot g(x)] = L \cdot M$|
|Quotient|$\lim_{x \to a} \frac{f(x)}{g(x)} = \frac{L}{M}$ (if $M \neq 0$)|
|Power|$\lim_{x \to a} [f(x)]^n = L^n$|

> [!example] Applying Limit Laws
> $\lim_{x \to 2} (3x^2 + 2x - 1) = 3(4) + 2(2) - 1 = 15$

---

## Squeeze Theorem

If $g(x) \leq f(x) \leq h(x)$ near $a$ and $\lim_{x \to a} g(x) = \lim_{x \to a} h(x) = L$, then:

$$\lim_{x \to a} f(x) = L$$

> [!example] Classic Application
> Show $\lim_{x \to 0} x \sin(1/x) = 0$
> 
> Since $-1 \leq \sin(1/x) \leq 1$, we have $-|x| \leq x \sin(1/x) \leq |x|$
> Both bounds approach 0, so the limit is 0 by squeeze theorem.

## Worked Examples

### Linear Function

**Prove**: $\lim_{x \to 2} (3x - 1) = 5$

Let $\epsilon > 0$ be given. Choose $\delta = \epsilon/3$.

Suppose $0 < |x - 2| < \delta$. Then:
$$|(3x - 1) - 5| = |3x - 6| = 3|x - 2| < 3\delta = \epsilon$$

Therefore, $\lim_{x \to 2} (3x - 1) = 5$. ∎

### Quadratic Function

**Prove**: $\lim_{x \to 1} x^2 = 1$

Let $\epsilon > 0$ be given. We need to bound $|x^2 - 1| = |x - 1||x + 1|$.

First, restrict $\delta \leq 1$. Then $|x - 1| < 1$ implies $0 < x < 2$, so $|x + 1| < 3$.

Choose $\delta = \min\{1, \epsilon/3\}$.

Suppose $0 < |x - 1| < \delta$. Then:
$$|x^2 - 1| = |x - 1||x + 1| < \delta \cdot 3 \leq \frac{\epsilon}{3} \cdot 3 = \epsilon$$

Therefore, $\lim_{x \to 1} x^2 = 1$. ∎

## Common Pitfalls

> [!warning] Mistake: Delta Depends on $x$
> $\delta$ can only depend on $\epsilon$ and $a$, **never** on $x$!

> [!warning] Mistake: Forgetting $0 < |x - a|$
> The condition $x \neq a$ is crucial. The limit doesn't care about $f(a)$.

> [!warning] Mistake: Not Handling All $\epsilon$
> Your proof must work for **arbitrarily small** $\epsilon > 0$, not just specific values.
