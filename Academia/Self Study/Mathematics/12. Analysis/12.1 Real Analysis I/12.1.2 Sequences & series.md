## Definition

A **sequence** is a function $a: \mathbb{N} \to \mathbb{R}$, typically written as $(a_n)_{n=1}^{\infty}$ or simply $(a_n)$.

> [!info] Tao Reference
> See *Analysis I* (4th ed.), Chapter 6: Limits of Functions, Sections 6.1-6.2 (pp. 155-165) for sequence convergence

---

## Convergence

### Definition

A sequence $(a_n)$ **converges** to $L$ if:

$$\forall \epsilon > 0, \exists N \in \mathbb{N} \text{ such that } n \geq N \implies |a_n - L| < \epsilon$$

We write $\lim_{n \to \infty} a_n = L$ or $a_n \to L$.

If no such $L$ exists, the sequence **diverges**.

```desmos-graph
y = 1/x
x = [1, 20]
```

> [!example] Visual Interpretation
> Convergence means: eventually (for large enough $n$), all terms stay within any specified tolerance $\epsilon$ of the limit $L$.

### Examples

**Example 1**: $a_n = \frac{1}{n}$ converges to 0.

Proof: Given $\epsilon > 0$, choose $N > 1/\epsilon$. Then for $n \geq N$:
$$\left|\frac{1}{n} - 0\right| = \frac{1}{n} \leq \frac{1}{N} < \epsilon$$

**Example 2**: $a_n = (-1)^n$ diverges (oscillates between -1 and 1).

**Example 3**: $a_n = \frac{n}{n+1}$ converges to 1.

## Key Properties

### Uniqueness of Limits

If $a_n \to L$ and $a_n \to M$, then $L = M$.

### Boundedness

Every convergent sequence is bounded: if $a_n \to L$, then $\exists M$ such that $|a_n| \leq M$ for all $n$.

> [!warning] Converse is False!
> Bounded sequences need NOT converge. Example: $(-1)^n$ is bounded but divergent.

### Algebraic Operations

If $a_n \to L$ and $b_n \to M$, then:

|Operation|Result|
|---|---|
|Sum|$(a_n + b_n) \to (L + M)$|
|Product|$(a_n \cdot b_n) \to (L \cdot M)$|
|Quotient|$a_n/b_n \to L/M$ (if $M \neq 0$)|
|Scalar multiple|$(ca_n) \to cL$|

## Cauchy Sequences

### Definition

A sequence $(a_n)$ is **Cauchy** if:

$$\forall \epsilon > 0, \exists N \in \mathbb{N} \text{ such that } m,n \geq N \implies |a_m - a_n| < \epsilon$$

### Cauchy Criterion

**Theorem**: A sequence of real numbers converges if and only if it is Cauchy.

This characterizes the **completeness** of $\mathbb{R}$!

> [!example] Why This Matters
> In incomplete spaces (like $\mathbb{Q}$), Cauchy sequences may not converge. The real numbers are complete—every Cauchy sequence has a limit in $\mathbb{R}$.

## Monotone Convergence Theorem

### Definitions

- **Monotone increasing**: $a_n \leq a_{n+1}$ for all $n$
- **Monotone decreasing**: $a_n \geq a_{n+1}$ for all $n$
- **Bounded above**: $\exists M$ such that $a_n \leq M$ for all $n$
- **Bounded below**: $\exists m$ such that $a_n \geq m$ for all $n$

### Theorem

**Monotone Convergence Theorem**: Every bounded monotone sequence converges.

Specifically:
- If $(a_n)$ is increasing and bounded above, then $a_n \to \sup\{a_n\}$
- If $(a_n)$ is decreasing and bounded below, then $a_n \to \inf\{a_n\}$

```desmos-graph
y = 1 - 1/x
x = [1, 20]
```

> [!example] Application
> The sequence $a_n = (1 + 1/n)^n$ is increasing and bounded above by 3, so it converges. Its limit is $e \approx 2.718$.

## Subsequences

### Definition

A **subsequence** of $(a_n)$ is a sequence $(a_{n_k})$ where $n_1 < n_2 < n_3 < ...$

### Bolzano-Weierstrass Theorem

**Theorem**: Every bounded sequence has a convergent subsequence.

This is a fundamental compactness result!

### Properties

- If $a_n \to L$, then every subsequence also converges to $L$
- If two subsequences converge to different limits, the original sequence diverges

## Series

### Definition

An infinite **series** is the sum of a sequence:

$$\sum_{n=1}^{\infty} a_n = a_1 + a_2 + a_3 + ...$$

The **partial sums** are:
$$S_N = \sum_{n=1}^{N} a_n$$

The series **converges** if the sequence of partial sums $(S_N)$ converges.

### Geometric Series

$$\sum_{n=0}^{\infty} ar^n = \frac{a}{1-r} \quad \text{if } |r| < 1$$

Diverges if $|r| \geq 1$.

### Harmonic Series

$$\sum_{n=1}^{\infty} \frac{1}{n} = 1 + \frac{1}{2} + \frac{1}{3} + ...$$

**Diverges** (grows like $\ln n$), even though terms approach 0!

## Convergence Tests

### Necessary Condition

If $\sum a_n$ converges, then $a_n \to 0$.

> [!warning] Not Sufficient!
> The converse is false. The harmonic series has $1/n \to 0$ but diverges.

### Comparison Test

If $0 \leq a_n \leq b_n$ for all $n$:
- If $\sum b_n$ converges, then $\sum a_n$ converges
- If $\sum a_n$ diverges, then $\sum b_n$ diverges

### Ratio Test

Let $L = \lim_{n \to \infty} \left|\frac{a_{n+1}}{a_n}\right|$:
- If $L < 1$: series converges absolutely
- If $L > 1$: series diverges
- If $L = 1$: test is inconclusive

### Root Test

Let $L = \lim_{n \to \infty} \sqrt[n]{|a_n|}$:
- If $L < 1$: series converges absolutely
- If $L > 1$: series diverges
- If $L = 1$: test is inconclusive

### Integral Test

If $f(x)$ is positive, continuous, and decreasing for $x \geq 1$, and $a_n = f(n)$, then:

$$\sum_{n=1}^{\infty} a_n \text{ converges } \iff \int_1^{\infty} f(x)\,dx \text{ converges}$$

## Common Mistakes

> [!warning] Confusing Sequence and Series
> Sequence: list of numbers $(a_n)$  
> Series: sum of numbers $\sum a_n$  
> A sequence can converge while its series diverges (e.g., $1/n$)!

> [!warning] Assuming $a_n \to 0$ Implies Convergence
> This is necessary but NOT sufficient for series convergence. Always use a convergence test!

> [!warning] Misapplying Tests
> Each test has specific conditions. Check hypotheses before applying!

## Practice Problems

Test convergence:

- Determine if $a_n = \frac{n^2}{n^2 + 1}$ converges. If so, find the limit
- Show that $a_n = \sin(n)$ diverges
- Test $\sum_{n=1}^{\infty} \frac{1}{n^2}$ for convergence
- Does $\sum_{n=1}^{\infty} \frac{(-1)^n}{n}$ converge?

Apply theorems:

- Use the Monotone Convergence Theorem to show $(1 + 1/n)^n$ converges
- Prove that every Cauchy sequence is bounded
- Find a convergent subsequence of $(-1)^n \cdot n/(n+1)$

Conceptual:

- Why is completeness of $\mathbb{R}$ important for analysis?
- Explain the difference between absolute and conditional convergence
- What does the Bolzano-Weierstrass theorem tell us about bounded sets?

Challenge:

- Prove the Cauchy criterion for convergence
- Show that rearranging a conditionally convergent series can change its sum (Riemann Rearrangement Theorem)
- Research: What is the difference between pointwise and uniform convergence of sequences of functions?
