## Definition

The **derivative** of $f$ at $a$ is:

$$f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}$$

if this limit exists. We say $f$ is **differentiable** at $a$.

Equivalently:
$$f'(a) = \lim_{x \to a} \frac{f(x) - f(a)}{x - a}$$

> [!info] Tao Reference
> See *Analysis I* (4th ed.), Chapter 10: Differentiation of Functions, Sections 10.1-10.3 (pp. 251-270)

> [!example] Geometric Interpretation
> The derivative is the slope of the tangent line to the graph at $(a, f(a))$.

```desmos-graph
x = 1
```

## Differentiability Implies Continuity

**Theorem**: If $f$ is differentiable at $a$, then $f$ is continuous at $a$.

**Proof**: 
$$\begin{align*}
\lim_{x \to a} [f(x) - f(a)] &= \lim_{x \to a} \frac{f(x) - f(a)}{x - a} \cdot (x - a) \\
&= f'(a) \cdot 0 = 0
\end{align*}$$

Therefore $\lim_{x \to a} f(x) = f(a)$. ∎

> [!warning] Converse is False!
> Continuous does NOT imply differentiable!  
> Example: $f(x) = |x|$ is continuous at 0 but not differentiable there.

## Basic Derivative Rules

### Power Rule

$$\frac{d}{dx}[x^n] = nx^{n-1}$$

### Linearity

$$\frac{d}{dx}[af(x) + bg(x)] = af'(x) + bg'(x)$$

### Product Rule

$$\frac{d}{dx}[f(x)g(x)] = f'(x)g(x) + f(x)g'(x)$$

### Quotient Rule

$$\frac{d}{dx}\left[\frac{f(x)}{g(x)}\right] = \frac{f'(x)g(x) - f(x)g'(x)}{[g(x)]^2}$$

### Chain Rule

$$\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)$$

## Higher-Order Derivatives

### Notation

- First derivative: $f'(x)$ or $\frac{df}{dx}$
- Second derivative: $f''(x)$ or $\frac{d^2f}{dx^2}$
- $n$-th derivative: $f^{(n)}(x)$ or $\frac{d^nf}{dx^n}$

### Examples

For $f(x) = x^4$:
- $f'(x) = 4x^3$
- $f''(x) = 12x^2$
- $f'''(x) = 24x$
- $f^{(4)}(x) = 24$
- $f^{(n)}(x) = 0$ for $n \geq 5$

### Smoothness Classes

- $C^0$: continuous functions
- $C^1$: continuously differentiable ($f'$ exists and is continuous)
- $C^n$: $n$ times continuously differentiable
- $C^\infty$: infinitely differentiable (smooth)
- $C^\omega$: analytic (equal to its Taylor series)

## Critical Points and Extrema

### Definitions

- **Critical point**: $c$ where $f'(c) = 0$ or $f'(c)$ doesn't exist
- **Local maximum**: $f(c) \geq f(x)$ for all $x$ near $c$
- **Local minimum**: $f(c) \leq f(x)$ for all $x$ near $c$

### First Derivative Test

If $c$ is a critical point:
- If $f'$ changes from positive to negative at $c$: local maximum
- If $f'$ changes from negative to positive at $c$: local minimum
- If $f'$ doesn't change sign: neither

### Second Derivative Test

If $f'(c) = 0$ and $f''$ exists:
- If $f''(c) > 0$: local minimum
- If $f''(c) < 0$: local maximum
- If $f''(c) = 0$: test inconclusive

## Common Mistakes

> [!warning] Assuming Differentiability
> Always verify that $f$ is differentiable before applying rules. Check for corners, cusps, vertical tangents!

> [!warning] Misapplying Chain Rule
> Don't forget the inner derivative! $\frac{d}{dx}[f(g(x))] \neq f'(g(x))$.

> [!warning] Confusing $f'$ with $f$
> The derivative describes the RATE OF CHANGE, not the function value itself.

> [!warning] Forgetting Domain Restrictions
> $\frac{d}{dx}[\ln x] = 1/x$ only for $x > 0$!
