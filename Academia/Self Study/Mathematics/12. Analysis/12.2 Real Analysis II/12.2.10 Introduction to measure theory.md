## Motivation

### Limitations of Riemann Integration

The Riemann integral has serious limitations:
- Cannot integrate highly discontinuous functions (e.g., Dirichlet function)
- Poor behavior with limits: $\lim \int f_n \neq \int \lim f_n$ in general
- Cannot handle unbounded domains naturally

**Lebesgue integration** solves these problems!

> [!info] Tao Reference
> See *Analysis II* (4th ed.), Chapter 11: The Riemann Integral, Section 11.12 (pp. 335-340) and Introduction to Measure Theory

## Outer Measure

### Definition

The **Lebesgue outer measure** of $E \subseteq \mathbb{R}$ is:

$$m^*(E) = \inf\left\{\sum_{n=1}^\infty |I_n| : E \subseteq \bigcup_{n=1}^\infty I_n\right\}$$

where $I_n$ are open intervals.

We cover $E$ with countably many intervals and take the infimum of total lengths.

### Properties

1. $m^*(\emptyset) = 0$
2. Monotonicity: $A \subseteq B \implies m^*(A) \leq m^*(B)$
3. Countable subadditivity: $m^*\left(\bigcup_{n=1}^\infty E_n\right) \leq \sum_{n=1}^\infty m^*(E_n)$

```desmos-graph
y = 0
y = 1
```

> [!example] Visual Interpretation
> Outer measure approximates the "size" of a set by covering it with intervals. For an interval $[a,b]$, we get $m^*([a,b]) = b-a$.

## Measurable Sets

### Carathéodory's Criterion

A set $E$ is **Lebesgue measurable** if for every $A \subseteq \mathbb{R}$:

$$m^*(A) = m^*(A \cap E) + m^*(A \cap E^c)$$

Intuitively: $E$ splits every test set "nicely."

### Alternative Definition (for intuition)

$E$ is measurable if it can be approximated by open sets from outside and closed sets from inside.

---

## Lebesgue Measure

### Definition

For measurable sets, define **Lebesgue measure**:

$$m(E) = m^*(E)$$

### Examples

|Set|Measure|
|---|---|
|Interval $[a,b]$|$b - a$|
|Single point $\{x\}$|$0$|
|Rational numbers $\mathbb{Q}$|$0$|
|Cantor set|$0$|
|$[0,1] \setminus \mathbb{Q}$|$1$|

---

## Key Properties

### Countable Additivity

If $\{E_n\}$ are disjoint measurable sets:

$$m\left(\bigcup_{n=1}^\infty E_n\right) = \sum_{n=1}^\infty m(E_n)$$

This is MUCH stronger than finite additivity!

### Translation Invariance

$$m(E + x) = m(E)$$

Shifting a set doesn't change its measure.

### Completeness

If $m(E) = 0$ and $F \subseteq E$, then $F$ is measurable with $m(F) = 0$.

## Almost Everywhere

### Definition

A property holds **almost everywhere** (a.e.) if it holds except on a set of measure zero.

### Examples

- A function continuous a.e. may have countably many discontinuities
- Two functions equal a.e. have the same Lebesgue integral
- "Most" real numbers are irrational ($\mathbb{Q}$ has measure zero)

### Notation

Write "$P(x)$ holds a.e." or "$f = g$ a.e."

## Lebesgue Integral (Overview)

### For Simple Functions

A **simple function** takes finitely many values:
$$s(x) = \sum_{i=1}^n a_i \chi_{E_i}(x)$$

Define:
$$\int s\,dm = \sum_{i=1}^n a_i m(E_i)$$

### For Non-negative Functions

$$\int f\,dm = \sup\left\{\int s\,dm : 0 \leq s \leq f, s \text{ simple}\right\}$$

Approximate from below by simple functions!

### For General Functions

Write $f = f^+ - f^-$ where $f^+ = \max(f,0)$ and $f^- = \max(-f,0)$.

Then:
$$\int f\,dm = \int f^+\,dm - \int f^-\,dm$$

(when both are finite)

## Convergence Theorems

### Monotone Convergence Theorem

If $0 \leq f_1 \leq f_2 \leq ...$ and $f_n \to f$ pointwise:

$$\lim_{n \to \infty} \int f_n\,dm = \int f\,dm$$

### Dominated Convergence Theorem

If $f_n \to f$ pointwise and $|f_n| \leq g$ for some integrable $g$:

$$\lim_{n \to \infty} \int f_n\,dm = \int f\,dm$$

These justify interchanging limits and integrals!

## Comparison with Riemann

|Property|Riemann|Lebesgue|
|---|---|---|
|Domain|Intervals|Any measurable set|
|Discontinuous functions|Limited|Much broader|
|Limit theorems|Weak|Strong (MCT, DCT)|
|Completeness|No|Yes|
|Fundamental theorem|Requires continuity|More general|

**Every Riemann integrable function is Lebesgue integrable, but not conversely!**

## Common Mistakes

> [!warning] Confusing Outer Measure with Measure
> Outer measure is defined for ALL sets, but only equals measure for measurable sets!

> [!warning] Assuming All Sets Are Measurable
> Non-measurable sets exist (require Axiom of Choice). Vitali sets are classic examples.

> [!warning] Thinking Measure Zero Means Empty
> $\mathbb{Q}$ has measure zero but is dense in $\mathbb{R}$!

## Practice Problems

Compute measures:

- Find the measure of the Cantor set
- Show that any countable set has measure zero
- What is the measure of $(0,1) \cup (2,3) \cup (4,5) \cup ...$?

Test measurability:

- Prove that open sets are measurable
- Show that the union of measurable sets is measurable
- Is every subset of a measure zero set measurable?

Conceptual:

- Why do we need measure theory beyond Riemann integration?
- Explain the significance of "almost everywhere"
- How does Lebesgue integration handle limits better?

Challenge:

- Construct a non-measurable set (Vitali set)
- Prove the Dominated Convergence Theorem
- Research: What is the relationship between Lebesgue measure and probability?
