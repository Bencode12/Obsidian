## Definition

A **metric space** is a pair $(X,d)$ where $X$ is a set and $d: X \times X \to \mathbb{R}$ is a **metric** (distance function) satisfying:

1. **Non-negativity**: $d(x,y) \geq 0$ and $d(x,y) = 0 \iff x = y$
2. **Symmetry**: $d(x,y) = d(y,x)$
3. **Triangle inequality**: $d(x,z) \leq d(x,y) + d(y,z)$

> [!info] Tao Reference
> See *Analysis II* (4th ed.), Chapter 12: Metric Spaces, Sections 12.1-12.3 (pp. 335-350)

---

## Examples

### Euclidean Space $\mathbb{R}^n$

$$d(x,y) = \|x-y\| = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}$$

### Discrete Metric

$$d(x,y) = \begin{cases} 0 & x = y \\ 1 & x \neq y \end{cases}$$

Every subset is both open and closed!

### Supremum Metric on $C([a,b])$

$$d(f,g) = \sup_{x \in [a,b]} |f(x) - g(x)|$$

### $L^1$ Metric

$$d(f,g) = \int_a^b |f(x) - g(x)|\,dx$$

## Open and Closed Sets

### Open Balls

The **open ball** centered at $x$ with radius $r$:

$$B(x,r) = \{y \in X : d(x,y) < r\}$$

### Open Sets

$U \subseteq X$ is **open** if for every $x \in U$, there exists $r > 0$ such that $B(x,r) \subseteq U$.

### Closed Sets

$F \subseteq X$ is **closed** if its complement $X \setminus F$ is open.

Equivalently: $F$ contains all its limit points.

```desmos-graph
(x-2)^2 + (y-3)^2 < 1
(x-2)^2 + (y-3)^2 = 1
```

> [!example] Visual Interpretation
> Open balls in $\mathbb{R}^2$ are disks without the boundary. Open sets are unions of open balls.

## Convergence and Continuity

### Convergent Sequences

$(x_n)$ **converges** to $x$ if:
$$\forall \epsilon > 0, \exists N \text{ such that } n \geq N \implies d(x_n, x) < \epsilon$$

### Cauchy Sequences

$(x_n)$ is **Cauchy** if:
$$\forall \epsilon > 0, \exists N \text{ such that } m,n \geq N \implies d(x_m, x_n) < \epsilon$$

### Continuous Functions

$f: (X,d_X) \to (Y,d_Y)$ is **continuous** at $x$ if:
$$\forall \epsilon > 0, \exists \delta > 0 \text{ such that } d_X(x,y) < \delta \implies d_Y(f(x),f(y)) < \epsilon$$

## Completeness

### Definition

A metric space is **complete** if every Cauchy sequence converges to a point in the space.

### Examples

|Space|Complete?|
|---|---|
|$\mathbb{R}^n$|Yes|
|$\mathbb{Q}$|No|
|$C([a,b])$ with sup norm|Yes|
|Polynomials with sup norm|No|

### Completion

Every metric space has a **completion**—a complete space containing it as a dense subspace.

Example: $\mathbb{R}$ is the completion of $\mathbb{Q}$.

---

## Compactness

### Definitions

- **Sequentially compact**: Every sequence has a convergent subsequence
- **Compact**: Every open cover has a finite subcover

In metric spaces, these are equivalent!

### Heine-Borel Theorem

**Theorem**: A subset of $\mathbb{R}^n$ is compact if and only if it is closed and bounded.

### Properties

Continuous images of compact sets are compact. Compact sets are:
- Closed
- Bounded
- Complete

---

## Connectedness

### Definition

$X$ is **connected** if it cannot be written as the union of two disjoint non-empty open sets.

### Path Connectedness

$X$ is **path connected** if any two points can be joined by a continuous path.

Path connected $\implies$ connected (but not conversely!).

### Example

The topologist's sine curve is connected but not path connected!

## Common Mistakes

> [!warning] Confusing Complete and Closed
> "Closed" is relative to the ambient space. "Complete" is intrinsic. $\mathbb{Q}$ is closed in itself but not complete!

> [!warning] Assuming Bounded Implies Compact
> Only true in $\mathbb{R}^n$ (Heine-Borel). In infinite dimensions, closed and bounded ≠ compact.

> [!warning] Mixing Up Open and Closed
> A set can be neither open nor closed, or both! Don't assume they're opposites.

## Practice Problems

Work with metrics:

- Verify that the discrete metric satisfies all axioms
- Show that $d(f,g) = \int_0^1 |f-g|$ is a metric on $C([0,1])$
- Prove the triangle inequality for the Euclidean metric

Test properties:

- Is $(0,1)$ complete with the standard metric?
- Show that $C([0,1])$ with the $L^1$ metric is not complete
- Determine if $\{(x,y) : xy = 1\}$ is connected in $\mathbb{R}^2$

Apply theorems:

- Use Heine-Borel to show $[0,1] \times [0,1]$ is compact
- Prove that a continuous function on a compact set is bounded
- Show that the image of a connected set under a continuous map is connected

Conceptual:

- Why is completeness important for analysis?
- Explain the relationship between compactness and extreme values
- What's the difference between connected and path connected?

Challenge:

- Prove that every compact metric space is complete
- Show that $\mathbb{R}$ is not compact but is complete
- Research: What is total boundedness and how does it relate to compactness?
