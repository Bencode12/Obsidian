## Statement

### Arzela-Ascoli Theorem

A subset $\mathcal{F} \subseteq C([a,b])$ is **relatively compact** (its closure is compact) if and only if:

1. **Pointwise bounded**: For each $x \in [a,b]$, the set $\{f(x) : f \in \mathcal{F}\}$ is bounded
2. **Equicontinuous**: For every $\epsilon > 0$, there exists $\delta > 0$ such that for all $f \in \mathcal{F}$:
   $$|x - y| < \delta \implies |f(x) - f(y)| < \epsilon$$

The same $\delta$ works for ALL functions in the family!

> [!info] Tao Reference
> See *Analysis II* (4th ed.), Chapter 14: Uniform Convergence, Section 14.4 (pp. 380-390)

## Understanding Equicontinuity

### Definition

A family $\mathcal{F}$ is **equicontinuous** at $x_0$ if:
$$\forall \epsilon > 0, \exists \delta > 0 \text{ such that } \forall f \in \mathcal{F}, |x - x_0| < \delta \implies |f(x) - f(x_0)| < \epsilon$$

Key: The SAME $\delta$ works for all functions!

### Comparison

|Property|Definition|
|---|---|
|Continuous|Each $f$ has its own $\delta_f$|
|Uniformly continuous|One $\delta$ works for all $x$ (for one $f$)|
|Equicontinuous|One $\delta$ works for all $f$ AND all $x$|

### Example

**Equicontinuous**: $\mathcal{F} = \{f_n(x) = \frac{\sin(nx)}{n}\}$

All functions have derivative bounded by 1, so they're Lipschitz with constant 1.

**Not equicontinuous**: $\mathcal{F} = \{f_n(x) = x^n\}$ on $[0,1]$

Near $x=1$, different functions need different $\delta$ values.

## Applications

### Existence of Solutions to ODEs

Used in the **Peano Existence Theorem**:

If $f(t,y)$ is continuous, then $y' = f(t,y)$ with $y(t_0) = y_0$ has a solution.

Proof constructs approximate solutions and uses Arzela-Ascoli to extract a convergent subsequence.

### Weierstrass Approximation

Every continuous function on $[a,b]$ can be uniformly approximated by polynomials.

Arzela-Ascoli helps prove this fundamental result!

### Compact Operators

In functional analysis, characterizes compact operators between Banach spaces.

---

## Proof Strategy

### Key Ideas

1. **Diagonal argument**: Extract subsequences converging at rational points
2. **Equicontinuity**: Extend pointwise convergence to uniform convergence
3. **Completeness**: Limit is continuous

### Outline

1. Use pointwise boundedness + diagonal argument to get pointwise convergence on a dense set
2. Use equicontinuity to upgrade to uniform convergence
3. Show the limit function is continuous
4. Conclude relative compactness

## Examples

### Example 1: Bounded Derivatives

If $\mathcal{F} = \{f \in C^1([a,b]) : |f'(x)| \leq M\}$, then $\mathcal{F}$ is equicontinuous.

By Mean Value Theorem: $|f(x) - f(y)| \leq M|x-y|$

If also pointwise bounded, Arzela-Ascoli applies!

### Example 2: Integral Operators

Consider $(Tf)(x) = \int_a^b K(x,t)f(t)\,dt$ where $K$ is continuous.

If $\|f\|_\infty \leq 1$, then $\{Tf\}$ is equicontinuous and bounded.

This shows $T$ is a compact operator!

## Variants

### On General Compact Spaces

If $X$ is compact metric space, then $\mathcal{F} \subseteq C(X)$ is relatively compact iff:
- Pointwise bounded
- Equicontinuous

### Local Version

For non-compact domains, we get local compactness (on compact subsets).

## Common Mistakes

> [!warning] Confusing Equicontinuity with Uniform Continuity
> Each function being uniformly continuous ≠ the family being equicontinuous!

> [!warning] Forgetting Pointwise Boundedness
> Both conditions are necessary! Equicontinuous but unbounded families aren't compact.

> [!warning] Assuming Pointwise Convergence Implies Uniform
> Need equicontinuity to upgrade from pointwise to uniform convergence!

## Practice Problems

Test equicontinuity:

- Is $\{f_n(x) = \frac{x}{n}\}$ equicontinuous on $\mathbb{R}$?
- Show that $\{f_n(x) = \cos(nx)/n\}$ is equicontinuous on $[0,2\pi]$
- Determine if $\{x^n\}$ is equicontinuous on $[0,1]$

Apply the theorem:

- Prove that $\{f \in C^1([0,1]) : |f'| \leq 1, f(0) = 0\}$ is relatively compact
- Show that solutions to $y' = f(x,y)$ form an equicontinuous family
- Use Arzela-Ascoli to prove Bolzano-Weierstrass for function sequences

Conceptual:

- Why is equicontinuity stronger than each function being continuous?
- Explain the role of the diagonal argument in the proof
- How does Arzela-Ascoli relate to Heine-Borel?
