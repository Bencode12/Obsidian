## Overview

The Lebesgue integral generalizes and improves upon the Riemann integral. Understanding their relationship clarifies when each is appropriate and why measure theory was developed.

---

## Key Differences

### Approach to Integration

|Aspect|Riemann Integral|Lebesgue Integral|
|---|---|---|
|**Partition**|Domain (x-axis)|Range (y-axis)|
|**Approximation**|Rectangles under curve|Horizontal slices|
|**Function class**|Bounded, few discontinuities|Measurable functions|
|**Limit theorems**|Limited|Powerful (MCT, DCT)|
|**Completeness**|Not complete|Complete ($L^p$ spaces)|

---

## When They Agree

### Theorem: Riemann ⇒ Lebesgue

If $f: [a,b] \to \mathbb{R}$ is Riemann integrable, then:
1. $f$ is Lebesgue measurable
2. $f$ is Lebesgue integrable
3. $\int_a^b f(x) \, dx = \int_{[a,b]} f \, d\mu$

### Characterization of Riemann Integrability

**Lebesgue's Criterion**: A bounded function $f: [a,b] \to \mathbb{R}$ is Riemann integrable if and only if:
- $f$ is continuous almost everywhere (discontinuities form a set of measure zero)

> [!example] Example: Step Function
> 
> $f(x) = \begin{cases} 0 & x < 1/2 \\ 1 & x \geq 1/2 \end{cases}$ on $[0,1]$
> 
> - Discontinuous at $x = 1/2$ (measure zero)
> - Both Riemann and Lebesgue integrable
> - $\int_0^1 f = 1/2$ in both theories

## When They Differ

### Case 1: Dirichlet Function

$$f(x) = \begin{cases} 1 & x \in \mathbb{Q} \\ 0 & x \notin \mathbb{Q} \end{cases}$$

On $[0,1]$:
- **Riemann**: Not integrable (upper sum = 1, lower sum = 0)
- **Lebesgue**: Integrable with $\int_0^1 f \, d\mu = 0$ (since $\mu(\mathbb{Q}) = 0$)

### Case 2: Unbounded Functions

$$f(x) = \frac{1}{\sqrt{x}} \text{ on } (0,1]$$

- **Riemann**: Improper integral $\lim_{\epsilon \to 0^+} \int_\epsilon^1 \frac{1}{\sqrt{x}} dx = 2$
- **Lebesgue**: Directly integrable, $\int_{(0,1]} \frac{1}{\sqrt{x}} d\mu = 2$

Lebesgue handles this without special "improper" machinery.

### Case 3: Limit Interchange

Consider $f_n(x) = n\chi_{(0,1/n)}(x)$ on $[0,1]$:

- Pointwise limit: $f_n \to 0$ everywhere except at 0
- $\int f_n = 1$ for all $n$, but $\int \lim f_n = 0$

**Riemann theory**: Cannot justify limit interchange  
**Lebesgue theory**: MCT/DCT tell us exactly when we can interchange

## Advantages of Lebesgue Integration

### 1. Better Convergence Theorems

Riemann integration lacks powerful tools like:
- Monotone Convergence Theorem
- Dominated Convergence Theorem
- Fatou's Lemma

These are essential for analysis, probability, and PDEs.

### 2. Complete Function Spaces

$L^p$ spaces are Banach spaces (complete), while Riemann-integrable functions don't form a complete space. This is crucial for:
- Fourier analysis
- Functional analysis
- Solving differential equations

### 3. Handles Pathological Functions

Lebesgue integration works for:
- Functions with dense discontinuities
- Functions defined on abstract measure spaces
- Probability distributions (including discrete and singular continuous)

### 4. Natural Framework for Probability

Probability theory is built on measure theory:
- Sample space = measure space with total measure 1
- Random variables = measurable functions
- Expectation = Lebesgue integral

---

## When Riemann is Preferable

Despite Lebesgue's superiority, Riemann integration still has uses:

1. **Pedagogical**: Easier to introduce conceptually
2. **Numerical computation**: Riemann sums directly relate to numerical methods
3. **Elementary calculus**: Sufficient for most engineering applications
4. **Geometric intuition**: Area under curve is more visual

## Practice Problems

1. Prove that every Riemann integrable function is Lebesgue integrable with the same value.
2. Show that the Dirichlet function is not Riemann integrable using upper and lower sums.
3. Give an example of a Lebesgue integrable function that is not Riemann integrable.
4. If $f$ is continuous on $[a,b]$, prove it's both Riemann and Lebesgue integrable.
5. Construct a function that is Lebesgue integrable but unbounded on every interval.
6. Show that if $f_n \to f$ uniformly on $[a,b]$ and each $f_n$ is Riemann integrable, then $\lim \int f_n = \int f$.
7. Explain why $L^2[0,1]$ is complete but the space of Riemann integrable functions is not.
8. Prove Lebesgue's criterion: $f$ is Riemann integrable iff it's continuous a.e.
9. Compare the computational complexity of approximating Riemann vs Lebesgue integrals numerically.
10. Discuss why probability theory requires Lebesgue integration rather than Riemann.
