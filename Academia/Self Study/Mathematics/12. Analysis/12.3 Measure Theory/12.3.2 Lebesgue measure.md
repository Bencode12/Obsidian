## Definition

The **Lebesgue measure** $\mu$ (or $m$) on $\mathbb{R}$ is a measure defined on the Borel σ-algebra $\mathcal{B}(\mathbb{R})$ that assigns to each interval its length:

$$\mu((a,b)) = b - a \quad \text{for } a < b$$

More generally, for any Borel set $E$, $\mu(E)$ represents the "length" or "size" of $E$.

---

## Construction via Outer Measure

### Lebesgue Outer Measure

For any set $E \subseteq \mathbb{R}$, the **outer measure** is:

$$\mu^*(E) = \inf \left\{\sum_{n=1}^{\infty} (b_n - a_n) : E \subseteq \bigcup_{n=1}^{\infty} (a_n, b_n)\right\}$$

This is the infimum of total lengths of countable coverings by open intervals.

### Carathéodory's Criterion

A set $E$ is **Lebesgue measurable** if for all $A \subseteq \mathbb{R}$:

$$\mu^*(A) = \mu^*(A \cap E) + \mu^*(A \cap E^c)$$

The collection of all Lebesgue measurable sets forms a σ-algebra containing the Borel σ-algebra.

---

## Properties of Lebesgue Measure

|Property|Statement|
|---|---|
|**Non-negativity**|$\mu(E) \geq 0$ for all measurable $E$|
|**Null empty set**|$\mu(\emptyset) = 0$|
|**Countable additivity**|If $E_i$ are disjoint, $\mu(\bigcup E_i) = \sum \mu(E_i)$|
|**Translation invariance**|$\mu(E + x) = \mu(E)$ for all $x \in \mathbb{R}$|
|**Monotonicity**|If $A \subseteq B$, then $\mu(A) \leq \mu(B)$|
|**Continuity from below**|If $E_1 \subseteq E_2 \subseteq \cdots$, then $\mu(\bigcup E_n) = \lim \mu(E_n)$|
|**Continuity from above**|If $E_1 \supseteq E_2 \supseteq \cdots$ and $\mu(E_1) < \infty$, then $\mu(\bigcap E_n) = \lim \mu(E_n)$|

## Examples

### Example 1: Intervals

- $\mu([a,b]) = b - a$
- $\mu((a,b)) = b - a$
- $\mu(\{x\}) = 0$ (single points have measure zero)

### Example 2: Countable Sets

Any countable set has measure zero:

$$\mu(\mathbb{Q}) = \mu(\mathbb{Z}) = \mu(\{1/n : n \in \mathbb{N}\}) = 0$$

> [!tip] Why This Matters
> 
> The rationals are dense in $\mathbb{R}$ but have measure zero. This shows that "dense" doesn't mean "large" in the measure-theoretic sense.

### Example 3: The Cantor Set

The **Cantor set** $C$ is uncountable but has measure zero:

- Constructed by removing middle thirds: $[0,1] \to [0,1/3] \cup [2/3,1] \to \cdots$
- Total length removed: $\frac{1}{3} + \frac{2}{9} + \frac{4}{27} + \cdots = 1$
- Therefore: $\mu(C) = 1 - 1 = 0$

Despite having measure zero, $C$ has the same cardinality as $\mathbb{R}$!

## Lebesgue Measure on $\mathbb{R}^n$

The Lebesgue measure generalizes to higher dimensions:

- On $\mathbb{R}^2$: measures area
- On $\mathbb{R}^3$: measures volume
- On $\mathbb{R}^n$: measures $n$-dimensional volume

For a rectangle $R = [a_1, b_1] \times \cdots \times [a_n, b_n]$:

$$\mu(R) = \prod_{i=1}^{n} (b_i - a_i)$$

---

## Non-Measurable Sets

Not all subsets of $\mathbb{R}$ are Lebesgue measurable! The **Vitali set** is a classic example:

### Construction (using Axiom of Choice)

1. Define equivalence relation on $[0,1]$: $x \sim y$ iff $x - y \in \mathbb{Q}$
2. Choose one representative from each equivalence class → Vitali set $V$
3. $V$ cannot be Lebesgue measurable

> [!warning] Paradoxical Result
> 
> If $V$ were measurable, translation invariance and countable additivity would lead to a contradiction: either $\mu(V) = 0$ (implying $\mu([0,1]) = 0$) or $\mu(V) > 0$ (implying $\mu([0,1]) = \infty$).

## Practice Problems

1. Show that $\mu^*(\emptyset) = 0$ and $\mu^*(\mathbb{R}) = \infty$.
2. Prove that any countable set has Lebesgue measure zero.
3. Calculate the Lebesgue measure of the set $\bigcup_{n=1}^{\infty} (n, n + 1/n^2)$.
4. Show that if $E$ is measurable and $\mu(E) = 0$, then every subset of $E$ is measurable with measure zero.
5. Prove that the Cantor set is uncountable.
6. Let $E \subseteq [0,1]$ be measurable with $\mu(E) > 0$. Show that $E - E = \{x - y : x,y \in E\}$ contains an interval around 0.
7. Verify translation invariance: show $\mu(E + x) = \mu(E)$ for measurable $E$ and $x \in \mathbb{R}$.
8. If $E$ and $F$ are measurable, prove $\mu(E \cup F) + \mu(E \cap F) = \mu(E) + \mu(F)$.
9. Show that there exist measurable sets that are not Borel sets.
10. Construct a set with positive measure that contains no intervals.
