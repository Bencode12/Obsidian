## Definition

The **Lebesgue integral** extends integration to a broader class of functions than the Riemann integral. For a non-negative measurable function $f: X \to [0,\infty]$:

$$\int_X f \, d\mu = \sup \left\{\int_X s \, d\mu : 0 \leq s \leq f, \, s \text{ simple}\right\}$$

For general functions, write $f = f^+ - f^-$ where $f^+ = \max(f,0)$ and $f^- = \max(-f,0)$.

---

## Simple Functions

A **simple function** has the form:

$$s(x) = \sum_{i=1}^{n} a_i \chi_{E_i}(x)$$

where $\chi_{E_i}$ is the characteristic function of measurable set $E_i$.

The integral of a simple function is:

$$\int_X s \, d\mu = \sum_{i=1}^{n} a_i \mu(E_i)$$

---

## Construction Steps

### Step 1: Characteristic Functions

For $E$ measurable:
$$\int_X \chi_E \, d\mu = \mu(E)$$

### Step 2: Simple Functions

Linearity gives:
$$\int_X \left(\sum a_i \chi_{E_i}\right) d\mu = \sum a_i \mu(E_i)$$

### Step 3: Non-negative Measurable Functions

Approximate by increasing sequence of simple functions:
$$\int_X f \, d\mu = \lim_{n \to \infty} \int_X s_n \, d\mu$$

### Step 4: General Integrable Functions

$f$ is **integrable** if $\int |f| \, d\mu < \infty$. Then:
$$\int_X f \, d\mu = \int_X f^+ \, d\mu - \int_X f^- \, d\mu$$

## Key Properties

|Property|Statement|
|---|---|
|**Linearity**|$\int (af + bg) = a\int f + b\int g$|
|**Monotonicity**|If $f \leq g$, then $\int f \leq \int g$|
|**Triangle inequality**|$\left|\int f\right| \leq \int |f|$|
|**Zero a.e.**|If $f = 0$ almost everywhere, then $\int f = 0$|
|**Additivity over domains**|$\int_{A \cup B} f = \int_A f + \int_B f$ for disjoint $A,B$|

## Fundamental Theorems

### Monotone Convergence Theorem (MCT)

If $0 \leq f_1 \leq f_2 \leq \cdots$ and $f_n \to f$ pointwise, then:

$$\lim_{n \to \infty} \int f_n \, d\mu = \int f \, d\mu$$

### Dominated Convergence Theorem (DCT)

If $f_n \to f$ pointwise and $|f_n| \leq g$ for some integrable $g$, then:

$$\lim_{n \to \infty} \int f_n \, d\mu = \int f \, d\mu$$

> [!tip] Why DCT is Powerful
> 
> DCT allows interchanging limits and integrals under mild conditions. This is crucial in analysis, probability, and PDEs.

---

## Examples

### Example 1: Dirichlet Function

The function $f(x) = \begin{cases} 1 & x \in \mathbb{Q} \\ 0 & x \notin \mathbb{Q} \end{cases}$ on $[0,1]$:

- **Not Riemann integrable** (upper sum = 1, lower sum = 0)
- **Lebesgue integrable**: $\int_0^1 f \, d\mu = 0$ (since $\mu(\mathbb{Q}) = 0$)

### Example 2: Improper Integral

$$\int_0^{\infty} e^{-x} \, dx = 1$$

This converges in both Riemann and Lebesgue sense, but Lebesgue theory handles it more elegantly via MCT.

## Practice Problems

1. Show that every Riemann integrable function is Lebesgue integrable with the same value.
2. Calculate $\int_0^1 x^2 \, d\mu$ using the definition via simple functions.
3. Prove that if $f \geq 0$ and $\int f = 0$, then $f = 0$ almost everywhere.
4. Use MCT to evaluate $\lim_{n \to \infty} \int_0^1 \frac{nx}{1 + n^2x^2} \, dx$.
5. Apply DCT to show $\lim_{n \to \infty} \int_0^{\infty} \left(1 + \frac{x}{n}\right)^{-n} dx = 1$.
6. If $f$ is integrable, prove that for any $\epsilon > 0$, there exists a simple function $s$ with $\int |f - s| < \epsilon$.
7. Show that $\int |f| = 0$ implies $f = 0$ almost everywhere.
8. Let $f_n(x) = n\chi_{(0,1/n)}(x)$. Does $\lim \int f_n = \int \lim f_n$? Explain.
9. Prove Hölder's inequality: $\int |fg| \leq \|f\|_p \|g\|_q$ where $1/p + 1/q = 1$.
10. If $f$ is measurable and bounded on a set of finite measure, show $f$ is integrable.
