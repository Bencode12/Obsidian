## What is a Linear Equation?

A linear equation is an algebraic statement where two expressions are **set equal**, and the variable is always to the **first power** — no squares or cubes.

|Component|Definition|Example in $2x + 5 = 11$|
|---|---|---|
|**Variable**|Symbol representing an unknown|$x$|
|**Coefficient**|Number multiplying the variable|$2$ in $2x$|
|**Constant**|Fixed number that does not change|$5$|

---

## Solving Linear Equations

The goal is to **isolate the variable** using inverse operations.

> [!tip] The Balance Rule Whatever you do to one side, you must do to the other.

|Operation|Inverse|
|---|---|
|$+\ a$|$-\ a$|
|$-\ a$|$+\ a$|
|$\times\ a$|$\div\ a$|
|$\div\ a$|$\times\ a$|

**Example — Solve $2x + 5 = 11$**

$$2x + 5 = 11$$ $$2x = 6 \quad (-5 \text{ both sides})$$ $$x = 3 \quad (\div 2 \text{ both sides})$$ $$\text{Check: } 2(3) + 5 = 11 \ \checkmark$$

## Linear Equations in Two Variables

When an equation contains two variables it represents a **straight line** on a graph.

$$x + y = 10$$

|Form|Notation|Meaning|
|---|---|---|
|**Standard form**|$ax + by = c$|General linear equation|
|**Function notation**|$f(x) = mx + b$|Each input $x$ gives exactly one output $y$|

See [[3.6_Functions]] for more on $f(x) = mx + b$.
