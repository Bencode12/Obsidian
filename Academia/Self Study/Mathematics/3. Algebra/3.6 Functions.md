## What is a Function?

A function is a relationship where **each input gives exactly one output**. Think of it as a machine: one value in, one value out.

$$y = f(x)$$

|Symbol|Name|Role|
|---|---|---|
|$f$|Function name|Labels the rule|
|$x$|Input|Independent variable|
|$f(x)$|Output|Dependent variable|

> [!tip] One input → one output If a single $x$ value produces two different $y$ values, it is **not a function**.

---

## Domain and Range

|Term|Definition|Example: $f(x) = \sqrt{x}$|
|---|---|---|
|**Domain**|All valid input values ($x$)|$x \geq 0$ — cannot square root a negative|
|**Range**|All possible output values ($y$)|$y \geq 0$ — square roots are non-negative|

---

## Evaluating Functions

Replace $x$ with the given value and simplify.

$$f(x) = 3x - 5$$ $$f(2) = 3(2) - 5 = 6 - 5 = 1$$

---

## The Vertical Line Test

A graph represents a function if **any vertical line crosses it at most once**. If a vertical line hits the graph at two or more points, it is not a function.

|Result|Conclusion|
|---|---|
|Every vertical line crosses once|✅ Function|
|Any vertical line crosses more than once|❌ Not a function|
