### **3.6.1 What is a function?**

A **function** is a relationship where **each input (x) gives exactly one output (y)**.

Notation:

y=f(x)y = f(x)y=f(x)

Example:

- f(x) = 2x + 3
- Input x = 4 → f(4) = 2×4 + 3 = 11

### **3.6.2 Domain and range**

- **Domain** → all possible input values (x)
- **Range** → all possible output values (y)

Example:

- f(x) = √x → domain x ≥ 0, range y ≥ 0

### **3.6.3 Evaluating functions**

- Replace the variable with a given number.

Example:

- f(x) = 3x − 5
- f(2) = 3×2 − 5 = 1

### **3.6.4 Function as a machine**

- Input → Function → Output
- Each input has exactly **one output**
