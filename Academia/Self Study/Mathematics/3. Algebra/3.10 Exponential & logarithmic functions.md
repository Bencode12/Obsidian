### **3.10.1 What is an exponential function?**

An **exponential function** is a function where the **variable appears in the exponent**:

f(x)=axf(x) = a^xf(x)=ax

- a>0,a≠1a > 0, a \neq 1a>0,a=1 → base
- $x$ → exponent / variable

Example:

- f(x) = 2^x → 2¹ = 2, 2² = 4, 2³ = 8

Exponential functions **grow or decay rapidly**.

### **3.10.2 Properties of exponential functions**

1. am×an=am+na^m × a^n = a^{m+n}am×an=am+n
2. am÷an=am−na^m ÷ a^n = a^{m−n}am÷an=am−n
3. (am)n=am×n(a^m)^n = a^{m×n}(am)n=am×n
4. a0=1a^0 = 1a0=1

### **3.10.3 What is a logarithmic function?**

A **logarithm** is the inverse of an exponential function:

y=log⁡a(x)  ⟺  ay=xy = \log_a(x) \iff a^y = xy=loga(x)⟺ay=x

- Base a>0,a≠1a > 0, a \neq 1a>0,a=1
- Logarithm answers: “To what power should the base be raised to get x?”

Example:

- log⁡2(8)=3\log_2(8) = 3log2(8)=3 → 2³ = 8

### **3.10.4 Properties of logarithms**

1. log⁡a(xy)=log⁡ax+log⁡ay\log_a(xy) = \log_a x + \log_a yloga(xy)=logax+logay
2. log⁡a(x/y)=log⁡ax−log⁡ay\log_a(x/y) = \log_a x − \log_a yloga(x/y)=logax−logay
3. log⁡a(xn)=nlog⁡ax\log_a(x^n) = n \log_a xloga(xn)=nlogax
4. log⁡aa=1,log⁡a1=0\log_a a = 1, \log_a 1 = 0logaa=1,loga1=0

### **3.10.5 Converting between forms**

- Exponential → Logarithmic: a^y = x → log_a(x) = y
- Logarithmic → Exponential: log_a(x) = y → a^y = x
