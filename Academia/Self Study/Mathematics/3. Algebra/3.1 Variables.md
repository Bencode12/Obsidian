## What is a Variable?

A variable is a **symbol** (usually a letter) representing an unknown or changeable number.

$$x,\ y,\ n,\ a,\ b$$

|Example|What the variable means|
|---|---|
|$5 + x = 12$|$x$ is an unknown value to find|
|$f(t) = 2t + 3$|$t$ can take any value — it changes|

---

## Why Use Variables?

- Represent **unknown** quantities
- Write **general formulas** that work for any number
- Solve algebraic problems **systematically**

---

## Expressions vs Equations

||Definition|Example|
|---|---|---|
|**Expression**|Combination of numbers, variables, and operations — **no equals sign**|$2x + 5$|
|**Equation**|Two expressions set **equal** to each other|$2x + 5 = 11$|

> [!tip] An expression has no solution — it can only be **evaluated** (given a value of $x$). An equation has a solution — it can be **solved** for $x$.

---

## Substituting Values

Replace the variable with a number to **evaluate** an expression.

$$3x + 2 \quad \text{at} \quad x = 4$$ $$= 3 \times 4 + 2 = 12 + 2 = 14$$

## Practice Problems

1. Evaluate $5x - 3$ at $x = 6$.
2. Evaluate $2a^2 + a$ at $a = 3$.
3. Is $4y + 7$ an expression or an equation?
4. Is $4y + 7 = 19$ an expression or an equation? Solve it.
5. Write an expression for "three more than twice a number $n$".
