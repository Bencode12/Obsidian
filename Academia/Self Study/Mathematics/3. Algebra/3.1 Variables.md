### **3.1.1 What is a variable?**

A **variable** is a **symbol** (usually a letter) that represents an **unknown or changeable number**.

Examples:

- x,y,n,a,bx, y, n, a, bx,y,n,a,b
- 5 + x = 12 → x is unknown
- f(t) = 2t + 3 → t is a variable that can change

### **3.1.2 Why use variables?**

Variables allow us to:

- Represent unknown quantities
- Write general formulas
- Solve algebraic problems systematically

### **3.1.3 Expressions vs equations**

- **Expression**: combination of numbers, variables, and operations  
    Example: 2x + 5
- **Equation**: a statement that two expressions are equal  
    Example: 2x + 5 = 11

### **3.1.4 Substituting values**

You can replace the variable with a number to evaluate an expression.

Example:

- Expression: 3x + 2
- x = 4 → 3×4 + 2 = 14
