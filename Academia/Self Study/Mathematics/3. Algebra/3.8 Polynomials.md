### **3.8.1 What is a polynomial?**

A **polynomial** is an expression with **variables and coefficients**, using only **addition, subtraction, and multiplication** (no division by variable).

General form:

anxn+an−1xn−1+...+a1x+a0a_nx^n + a_{n-1}x^{n-1} + ... + a_1x + a_0anxn+an−1xn−1+...+a1x+a0

- $n$ → degree of the polynomial (highest power of x)
- a0,a1,...,ana_0, a_1, ..., a_na0,a1,...,an → coefficients

Examples:

- 3x² + 5x − 2 → degree 2
- x³ − 4x + 1 → degree 3
- 7 → degree 0 (constant)

### **3.8.2 Types of polynomials**

- **Monomial** → one term: 5x
- **Binomial** → two terms: x + 3
- **Trinomial** → three terms: x² + x + 1

### **3.8.3 Adding and subtracting polynomials**

Combine like terms.

Example:

- (3x² + 2x + 1) + (x² − x + 4) = 4x² + x + 5

### **3.8.4 Multiplying polynomials**

Use distributive property.

Example:

- (x + 2)(x + 3) = x² + 5x + 6
