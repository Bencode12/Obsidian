# Programming Study Guide

## Structure

### Computer Science
- **Algorithms** - Sorting, searching, graph algorithms, dynamic programming
- **Data Structures** - Arrays, linked lists, trees, graphs, hash tables
- **Theory of Computation** - Automata, formal languages, complexity theory
- **Discrete Mathematics** - Logic, sets, relations, combinatorics

### Languages
- **Python** - Scripting, data science, automation
- **JavaScript & TypeScript** - Web development, Node.js
- **C & C++** - Systems programming, performance-critical applications
- **Rust** - Memory-safe systems programming
- **Java** - Enterprise, Android development
- **Go** - Cloud services, distributed systems
- **Haskell** - Functional programming fundamentals

### Software Development
- **Web Development**
  - Frontend - HTML/CSS, React, Vue, Angular
  - Backend - APIs, servers, databases
  - Full Stack - End-to-end applications
- **Mobile Development** - iOS, Android, cross-platform
- **Game Development** - Engines, graphics, game logic
- **Desktop Applications** - GUI applications, native development

### Software Engineering
- **Design Patterns** - Creational, structural, behavioral patterns
- **Architecture** - MVC, microservices, monolithic, event-driven
- **Testing** - Unit, integration, end-to-end testing
- **Version Control** - Git workflows, collaboration
- **DevOps & CI/CD** - Automation, deployment pipelines

### Systems
- **Operating Systems** - Processes, memory, filesystems
- **Computer Networks** - Protocols, routing, security
- **Databases** - SQL, NoSQL, data modeling
- **Compilers & Interpreters** - Lexing, parsing, code generation
- **Security** - Cryptography, vulnerabilities, secure coding

### Paradigms
- **Object-Oriented Programming** - Classes, inheritance, polymorphism
- **Functional Programming** - Pure functions, immutability, higher-order functions
- **Procedural Programming** - Functions, control flow
- **Logic Programming** - Prolog, constraints, inference
