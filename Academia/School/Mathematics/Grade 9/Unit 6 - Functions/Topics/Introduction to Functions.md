## What is a Function?

A function is a **rule** that assigns **exactly one output $y$ to every input $x$**.

$$f(x) = y$$

If one value of $x$ gives two different values of $y$ — it is **not a function**.

---

## Types of Graphs

|Function|Shape|Notes|
|---|---|---|
|$f(x)$|Straight line|Crosses $x$-axis at $x = -2$, crosses $y$-axis at $y = 1$|
|$g(x)$|Parabola|U-shaped curve|
|$h(x)$|Piecewise line|Horizontal at $y = 1$ for $x < 0$, then drops down; crosses $x$-axis at $x = 1$|
|$t(x)$|S-curve|**Not a function** — one $x$ gives more than one $y$ (fails the vertical line test)|

> [!tip] Vertical Line Test Draw a vertical line anywhere on the graph. If it crosses the graph **more than once**, it is not a function. $t(x)$ fails this test.

---

## Key Vocabulary

|Term|Notation|Meaning|
|---|---|---|
|**Domain**|$D_f$|All valid input values ($x$)|
|**Range**|$E_f$|All output values ($y$) the function can produce|
|**Argument**|$x$|The input value|
|**Value**|$f(x)$|The output — the answer ($y$)|
|**Zeros**|on $O_x$ (the $x$-axis)|Values of $x$ where $f(x) = 0$|

---

## Describing a Function

### 1. Positive and Negative

$$f(x) > 0 \quad \text{when } x \in \ldots \qquad \text{(graph is above the } x\text{-axis)}$$

$$f(x) < 0 \quad \text{when } x \in \ldots \qquad \text{(graph is below the } x\text{-axis)}$$

### 2. Increasing and Decreasing

$$f(x) \text{ increases when } x \in \ldots \qquad \text{(graph goes up left to right)}$$

$$f(x) \text{ decreases when } x \in \ldots \qquad \text{(graph goes down left to right)}$$

## Worked Example — Reading a Graph

Points on $f(x)$: $(-5,\ 1),\ (-2,\ -1),\ (-1,\ 0),\ (0,\ 2),\ (2,\ 4),\ (3,\ 0),\ (4,\ -2)$

**1. Domain**

$$D_f = [-5;\ 4]$$

**2. Range**

$$E_f = [-2;\ 4]$$

**3. Zeros** — where $f(x) = 0$

From the points: $f(-1) = 0$ and $f(3) = 0$

$$\text{Zeros at } x = -1 \text{ and } x = 3$$

**4. Positive** — where $f(x) > 0$ (above $x$-axis)

$$f(x) > 0 \quad \text{when } x \in [-5;\ -2) \cup (-1;\ 3)$$

**5. Negative** — where $f(x) < 0$ (below $x$-axis)

$$f(x) < 0 \quad \text{when } x \in (-2;\ -1) \cup (3;\ 4]$$

> [!tip] Open vs closed brackets Use $[\ ]$ when the endpoint **is included** (the function equals zero or the interval starts/ends there). Use $(\ )$ when the endpoint **is not included** — e.g. at zeros, the function is neither positive nor negative, so the zero itself is excluded from both positive and negative intervals.
